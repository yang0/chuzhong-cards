# 第二单元 夏商周时期

## 基本信息

- 科目：历史
- 年级：七年级上册
- 教材：人教版 / 统编版《中国历史》
- 单元：第二单元 夏商周时期
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元主要讲述夏、商、西周王朝的更替，春秋战国时期的社会变化，以及夏商周时期的科技文化成就。

学习主线：

1. 夏、商、西周相继建立和灭亡，早期国家制度逐步形成。
2. 春秋战国时期，王室衰微、诸侯争霸，社会生产力和政治制度发生重大变化。
3. 百家争鸣和青铜器、甲骨文等文化成就，反映出中国早期文明的发展。

核心对比：

- 禅让制：传说中尧舜禹时期推举贤德。
- 世袭制：夏朝以后王位在家族内部继承。
- 分封制：西周为巩固统治而分封诸侯。
- 春秋争霸：周王室衰微，诸侯争做霸主。
- 战国变法：各国通过改革增强实力，封建制度逐步确立。

## Anki 记忆卡

### 卡1

**配图：** ![](第二单元-夏商周时期/01-xia-first-dynasty.png)

**提问：** 夏朝由谁建立？夏朝建立在中国古代国家发展中有什么地位？

**答案：** 夏朝由禹建立；夏朝是中国历史上的第一个王朝。

**易错分析：** 不要把“禹建立夏朝”和“尧舜禹时期禅让制”混在一起。禹以前是传说中的部落联盟时期，夏朝建立后进入王朝国家阶段。

### 卡2

**配图：** ![](第二单元-夏商周时期/02-hereditary-succession.png)

**提问：** 夏朝建立后，王位继承制度发生了什么重要变化？

**答案：** 王位由禅让制转变为世袭制，启继承禹的位置。

**易错分析：** 禅让制强调推举贤德，世袭制强调家族继承。不要把“选贤”写成夏朝以后的主要继承方式。

### 卡3

**配图：** ![](第二单元-夏商周时期/03-jie-tang-dynasty-change.png)

**提问：** 夏朝最后一位君主是谁？商汤灭夏反映了什么历史现象？

**答案：** 夏朝最后一位君主是桀；商汤灭夏反映了暴政会导致王朝灭亡。

**易错分析：** 不要把夏桀和商纣混淆。桀是夏朝末代君主，纣是商朝末代君主。

### 卡4

**配图：** ![](第二单元-夏商周时期/04-shang-pangeng-yin.png)

**提问：** 商朝建立者是谁？商朝后来把都城迁到哪里后相对稳定？

**答案：** 商朝由汤建立；盘庚迁都到殷后，商朝统治相对稳定。

**易错分析：** 不要把“商汤灭夏”和“盘庚迁殷”混成同一事件。汤是建立商朝，盘庚是后来迁都。

### 卡5

**配图：** ![](第二单元-夏商周时期/05-muye-king-wu.png)

**提问：** 商朝末代君主是谁？周武王灭商的关键战役是什么？

**答案：** 商朝末代君主是纣；周武王在牧野之战中打败商军，灭亡商朝。

**易错分析：** 不要把“牧野之战”写成春秋战国时期的战争。它是西周建立前后商周更替的关键战役。

### 卡6

**配图：** ![](第二单元-夏商周时期/06-western-zhou-capital.png)

**提问：** 西周建立者是谁？西周的都城在哪里？

**答案：** 西周由周武王建立，都城在镐京。

**易错分析：** 不要把周武王和周幽王混淆。周武王建立西周，周幽王时期西周灭亡。

### 卡7

**配图：** ![](第二单元-夏商周时期/07-enfeoffment-purpose.png)

**提问：** 西周实行分封制的主要目的是什么？

**答案：** 主要目的是稳定周初政治形势，巩固疆土，扩大统治范围。

**易错分析：** 不要只答“分土地”。分封制背后的核心是巩固西周统治。

### 卡8

**配图：** ![](第二单元-夏商周时期/08-enfeoffment-content.png)

**提问：** 西周分封制中，周天子把哪些内容分封给宗亲和功臣？

**答案：** 周天子把土地和人民分封给宗亲、功臣等，让他们建立诸侯国。

**易错分析：** 分封的不只是“土地”，还包括人民和相应统治权。

### 卡9

**配图：** ![](第二单元-夏商周时期/09-vassal-duties.png)

**提问：** 西周分封制下，诸侯对周天子承担哪些义务？

**答案：** 诸侯要向周王进献贡物，并服从周王调兵。

**易错分析：** 不要把诸侯理解成完全独立的国君。分封制初期，诸侯名义上要服从周天子。

### 卡10

**配图：** ![](第二单元-夏商周时期/10-western-zhou-fall.png)

**提问：** 西周灭亡与哪位国王有关？西周灭亡后，中国历史进入哪个时期？

**答案：** 西周灭亡与周幽王统治腐败有关；西周灭亡后进入东周时期。

**易错分析：** 东周不是一个统一强盛的新王朝阶段，而是王室衰微、诸侯势力增强的时期。

### 卡11

**配图：** ![](第二单元-夏商周时期/11-eastern-zhou-periods.png)

**提问：** 东周在历史上通常分为哪两个时期？

**答案：** 东周分为春秋时期和战国时期。

**易错分析：** 不要把春秋和战国都当作单独王朝。它们是东周内部的两个历史阶段。

### 卡12

**配图：** ![](第二单元-夏商周时期/12-spring-autumn-hegemony.png)

**提问：** 春秋时期诸侯争霸的根本背景是什么？

**答案：** 周王室衰微，诸侯国势力增强，各诸侯争夺霸主地位。

**易错分析：** 不要只背“诸侯打仗”。关键背景是周天子权威下降。

### 卡13

**配图：** ![](第二单元-夏商周时期/13-qi-hegemony-slogan.png)

**提问：** 春秋时期齐桓公能够首先称霸的重要口号是什么？

**答案：** 齐桓公打出“尊王攘夷”的旗号。

**易错分析：** “尊王攘夷”不是尊重秦王，也不是战国变法口号。这里的“王”指周天子。

### 卡14

**配图：** ![](第二单元-夏商周时期/14-iron-tools-ox-plowing.png)

**提问：** 春秋时期社会生产力发展的重要表现有哪些？

**答案：** 铁制农具和牛耕出现并逐渐推广。

**易错分析：** 不要只写“农业发展”。考试常考具体技术：铁制农具、牛耕。

### 卡15

**配图：** ![](第二单元-夏商周时期/15-seven-warring-states.png)

**提问：** 战国时期的“战国七雄”是哪七个诸侯国？

**答案：** 齐、楚、燕、韩、赵、魏、秦。

**易错分析：** 不要把晋、鲁、宋等春秋重要诸侯国写进战国七雄。口诀可记“齐楚燕韩赵魏秦”。

### 卡16

**配图：** ![](第二单元-夏商周时期/16-reform-rich-strong.png)

**提问：** 战国时期各国变法的共同目的是什么？

**答案：** 富国强兵，在兼并战争中取得优势。

**易错分析：** 不要把变法理解成单纯改善民生。战国变法的直接目标是增强国家实力。

### 卡17

**配图：** ![](第二单元-夏商周时期/17-shang-yang-qin-xiaogong.png)

**提问：** 战国时期秦国商鞅变法发生在哪位国君支持下？

**答案：** 商鞅变法是在秦孝公支持下进行的。

**易错分析：** 不要写成秦始皇。商鞅变法早于秦始皇统一六国。

### 卡18

**配图：** ![](第二单元-夏商周时期/18-county-system.png)

**提问：** 商鞅变法在政治上加强中央对地方控制的重要措施是什么？

**答案：** 确立县制，由国君直接派官吏治理。

**易错分析：** 不要把县制完全归到秦始皇首创。商鞅变法已在秦国确立县制，秦统一后进一步推广郡县制。

### 卡19

**配图：** ![](第二单元-夏商周时期/19-encourage-farming-weaving.png)

**提问：** 商鞅变法在经济上鼓励农业生产的重要措施是什么？

**答案：** 鼓励耕织，生产粮食、布帛多的人可以免除徭役。

**易错分析：** 不要只答“奖励军功”。奖励军功主要是军事和社会等级方面的措施。

### 卡20

**配图：** ![](第二单元-夏商周时期/20-qin-strong-foundation.png)

**提问：** 商鞅变法对秦国的发展产生了什么影响？

**答案：** 商鞅变法使秦国国力大为增强，为以后秦统一全国奠定基础。

**易错分析：** 不要直接写“商鞅变法完成了统一”。统一全国的是后来的秦王嬴政。

### 卡21

**配图：** ![](第二单元-夏商周时期/21-dujiangyan-li-bing.png)

**提问：** 战国时期都江堰由谁主持修建？它位于哪一地区？

**答案：** 都江堰由秦国蜀郡郡守李冰主持修建，位于成都平原附近。

**易错分析：** 不要把都江堰写成秦始皇修建。主持修建者是李冰。

### 卡22

**配图：** ![](第二单元-夏商周时期/22-dujiangyan-functions.png)

**提问：** 战国时期都江堰的主要功能是什么？

**答案：** 都江堰具有防洪、灌溉、水运等功能，使成都平原成为“天府之国”。

**易错分析：** 不要只答“灌溉”。都江堰是综合性水利工程，防洪和灌溉都重要。

### 卡23

**配图：** ![](第二单元-夏商周时期/23-hundred-schools-background.png)

**提问：** 春秋战国时期“百家争鸣”局面出现的社会背景是什么？

**答案：** 社会急剧变化，旧制度瓦解，新制度逐步确立，各学派提出不同治国主张。

**易错分析：** 不要把百家争鸣理解成单纯文学繁荣。它与春秋战国社会大变革密切相关。

### 卡24

**配图：** ![](第二单元-夏商周时期/24-confucius-ren-education.png)

**提问：** 春秋时期孔子的核心思想是什么？他在教育方面有什么重要主张？

**答案：** 孔子的核心思想是“仁”；教育方面主张有教无类。

**易错分析：** 不要把孔子的“仁”和法家的“法治”混淆。孔子属于儒家。

### 卡25

**配图：** ![](第二单元-夏商周时期/25-hanfei-legalism-qin.png)

**提问：** 战国时期法家代表韩非主张怎样治理国家？这种思想后来最受哪个国家重视？

**答案：** 韩非主张以法治国，建立中央集权专制统治；这种思想后来受到秦国重视。

**易错分析：** 不要把韩非归入儒家或道家。韩非是法家代表，主张严刑峻法和中央集权。

### 卡26

**配图：** ![](第二单元-夏商周时期/26-oracle-bones-writing.png)

**提问：** 商周时期甲骨文主要刻写在哪里？它有什么历史地位？

**答案：** 甲骨文主要刻写在龟甲和兽骨上；我国有文字可考的历史从商朝开始。

**易错分析：** 不要把甲骨文说成刻在青铜器上。青铜器上的文字通常称为金文或铭文。

### 卡27

**配图：** ![](第二单元-夏商周时期/27-bronze-ritual-power.png)

**提问：** 商周时期青铜器的典型代表是什么？它反映了什么手工业水平？

**答案：** 典型代表有司母戊鼎（又称后母戊鼎）等大型青铜器；反映商周时期青铜铸造工艺高超。

**易错分析：** 不要把青铜器只当作日常工具。商周青铜器很多用于礼器，体现等级和权力。

### 卡28

**配图：** ![](第二单元-夏商周时期/28-laozi-daoism.png)

**提问：** 春秋时期道家学派创始人老子的主要思想有哪些？

**答案：** 老子主张顺应自然，认为事物有对立面，对立双方可以相互转化。

**易错分析：** 不要把老子的思想写成“仁政”或“以法治国”。仁政多与儒家孟子有关，以法治国是法家思想。

### 卡29

**配图：** ![](第二单元-夏商周时期/29-mozi-love-no-war.png)

**提问：** 战国时期墨家代表墨子的主要主张是什么？

**答案：** 墨子主张“兼爱”“非攻”，反对不义战争。

**易错分析：** 不要把墨子的“兼爱非攻”和儒家的“仁”完全等同。二者都重视人与人关系，但学派和表达不同。

### 卡30

**配图：** ![](第二单元-夏商周时期/30-mencius-benevolent-rule.png)

**提问：** 战国时期儒家代表孟子的政治主张是什么？

**答案：** 孟子主张实行“仁政”，提出“民贵君轻”的思想。

**易错分析：** 不要把孟子和孔子混成一个人。孔子是春秋时期儒家创始人，孟子是战国时期儒家代表。

### 卡31

**配图：** ![](第二单元-夏商周时期/31-zhuangzi-freedom-nature.png)

**提问：** 战国时期道家代表庄子的主要思想倾向是什么？

**答案：** 庄子强调顺应自然和民心，追求精神自由。

**易错分析：** 不要把庄子归入法家。庄子是道家代表，法家代表是韩非等人。

### 卡32

**配图：** ![](第二单元-夏商周时期/32-schools-comparison.png)

**提问：** 对比儒家、道家、法家，三者在治国思路上最典型的区别是什么？

**答案：** 儒家重视仁德和礼治，道家强调顺应自然，法家强调法治和中央集权。

**易错分析：** 总结类卡片要抓“治国思路”的差异，不要只背代表人物名字。
