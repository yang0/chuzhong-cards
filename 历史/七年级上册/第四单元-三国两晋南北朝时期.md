# 第四单元 三国两晋南北朝时期

## 基本信息

- 科目：历史
- 年级：七年级上册
- 教材：人教版 / 统编版《中国历史》
- 单元：第四单元 三国两晋南北朝时期
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元主要讲述三国鼎立、西晋短暂统一与民族内迁、东晋南朝和江南开发、北方民族交融，以及三国两晋南北朝时期的科技文化。

学习主线：

1. 东汉末年军阀割据，经过官渡之战、赤壁之战，逐渐形成三国鼎立。
2. 西晋短暂统一后走向分裂，北方民族内迁，政权更替频繁。
3. 东晋南朝时期，江南地区得到开发；北方通过改革和迁徙出现民族交融。
4. 科技、书法、绘画、雕塑等文化成就丰富，体现多元交融的时代特点。

核心对比：

- 官渡之战：曹操打败袁绍，为统一北方奠定基础。
- 赤壁之战：孙刘联军打败曹操，为三国鼎立局面形成奠定基础。
- 三国鼎立：魏、蜀、吴并立。
- 西晋统一：结束三国分裂，但统治短暂。
- 江南开发：北方人口南迁带来劳动力和技术。
- 北魏孝文帝改革：促进北方民族交融。

## Anki 记忆卡

### 卡1

**配图：** ![](第四单元-三国两晋南北朝时期/01-card.png)

**提问：** 东汉末年官渡之战的交战双方是谁？结果怎样？

**答案：** 官渡之战的交战双方是曹操和袁绍；曹操以少胜多打败袁绍。

**易错分析：** 不要把官渡之战和赤壁之战混淆。官渡之战发生在曹操和袁绍之间。

### 卡2

**配图：** ![](第四单元-三国两晋南北朝时期/02-card.png)

**提问：** 官渡之战对曹操统一北方有什么影响？

**答案：** 官渡之战为曹操以后统一北方奠定了基础。

**易错分析：** 官渡之战没有直接形成三国鼎立。三国鼎立的形成与赤壁之战关系更密切。

### 卡3

**配图：** ![](第四单元-三国两晋南北朝时期/03-card.png)

**提问：** 东汉末年赤壁之战的交战双方是谁？结果怎样？

**答案：** 赤壁之战的双方是曹操军队和孙刘联军；孙刘联军打败曹操。

**易错分析：** 不要把赤壁之战写成曹操打败孙刘联军。赤壁之战是曹操南下受挫。

### 卡4

**配图：** ![](第四单元-三国两晋南北朝时期/04-card.png)

**提问：** 赤壁之战对三国局面的形成有什么影响？

**答案：** 赤壁之战为三国鼎立局面的形成奠定了基础。

**易错分析：** 不要把赤壁之战的意义写成统一全国。它恰恰使统一暂时难以完成。

### 卡5

**配图：** ![](第四单元-三国两晋南北朝时期/05-card.png)

**提问：** 三国鼎立局面中，魏、蜀、吴分别由谁建立？

**答案：** 魏由曹丕建立，蜀由刘备建立，吴由孙权建立。

**易错分析：** 不要把魏的建立者写成曹操。曹操奠定曹魏基础，但称帝建魏的是曹丕。

### 卡6

**配图：** ![](第四单元-三国两晋南北朝时期/06-card.png)

**提问：** 三国鼎立局面中，魏、蜀、吴的都城分别在哪里？

**答案：** 魏都洛阳，蜀都成都，吴都建业。

**易错分析：** 不要把吴都写成南京而不写教材名“建业”。建业在今南京。

### 卡7

**配图：** ![](第四单元-三国两晋南北朝时期/07-card.png)

**提问：** 三国时期，哪一政权派卫温到达夷洲？这说明了什么？

**答案：** 吴国派卫温到达夷洲；说明大陆与台湾地区已有联系。

**易错分析：** 不要把夷洲写成西域。夷洲一般指今天台湾一带。

### 卡8

**配图：** ![](第四单元-三国两晋南北朝时期/08-card.png)

**提问：** 西晋由谁建立？西晋在哪一年完成统一？

**答案：** 西晋由司马炎建立；280 年，西晋灭吴，完成统一。

**易错分析：** 不要把西晋统一和秦统一混淆。秦统一是公元前 221 年，西晋统一是 280 年。

### 卡9

**配图：** ![](第四单元-三国两晋南北朝时期/09-card.png)

**提问：** 西晋灭亡的重要内部原因之一是什么？

**答案：** 西晋统治集团腐朽，八王之乱严重削弱了西晋力量。

**易错分析：** 不要只说“少数民族进攻”。西晋内部动乱是其衰亡的重要原因。

### 卡10

**配图：** ![](第四单元-三国两晋南北朝时期/10-card.png)

**提问：** 西晋时期北方少数民族内迁主要包括哪些民族？

**答案：** 主要有匈奴、鲜卑、羯、氐、羌等。

**易错分析：** 不要把这些民族简单等同于后来某一个统一民族。教材强调多个北方民族内迁。

### 卡11

**配图：** ![](第四单元-三国两晋南北朝时期/11-card.png)

**提问：** 东晋由谁建立？都城在哪里？

**答案：** 东晋由司马睿建立，都城在建康。

**易错分析：** 不要把东晋都城写成洛阳。洛阳是东汉、曹魏等政权的重要都城。

### 卡12

**配图：** ![](第四单元-三国两晋南北朝时期/12-card.png)

**提问：** 东晋时期前秦与东晋之间的著名战役是什么？结果怎样？

**答案：** 淝水之战；东晋以少胜多打败前秦。

**易错分析：** 不要把淝水之战和赤壁之战混淆。赤壁之战是三国形成前，淝水之战发生在东晋十六国时期。

### 卡13

**配图：** ![](第四单元-三国两晋南北朝时期/13-card.png)

**提问：** 东晋南朝时期，江南地区得到开发的重要原因是什么？

**答案：** 北方人口大量南迁，带来劳动力、先进生产工具和生产技术。

**易错分析：** 不要只答“江南自然条件好”。自然条件是基础，北民南迁带来的劳动力和技术是重要原因。

### 卡14

**配图：** ![](第四单元-三国两晋南北朝时期/14-card.png)

**提问：** 东晋南朝时期江南开发产生了什么影响？

**答案：** 江南地区经济迅速发展，为以后经济重心南移奠定基础。

**易错分析：** 不要写成“经济重心已经完成南移”。完成南移是在更晚的南宋时期。

### 卡15

**配图：** ![](第四单元-三国两晋南北朝时期/15-card.png)

**提问：** 南朝先后经历了哪四个朝代？

**答案：** 宋、齐、梁、陈。

**易错分析：** 不要把南朝的“宋”和后来的北宋、南宋混淆。这里的宋是南朝第一个朝代。

### 卡16

**配图：** ![](第四单元-三国两晋南北朝时期/16-card.png)

**提问：** 北魏是由哪个民族建立的政权？

**答案：** 北魏是鲜卑族拓跋部建立的政权。

**易错分析：** 不要把北魏写成汉族王朝。北魏是北方少数民族建立并逐渐汉化的重要政权。

### 卡17

**配图：** ![](第四单元-三国两晋南北朝时期/17-card.png)

**提问：** 北魏孝文帝迁都到哪里？迁都的目的是什么？

**答案：** 北魏孝文帝迁都洛阳；目的是加强对中原地区的统治，并学习汉族先进文化。

**易错分析：** 不要把北魏孝文帝迁都写成迁往平城。平城是北魏早期都城，孝文帝迁都洛阳。

### 卡18

**配图：** ![](第四单元-三国两晋南北朝时期/18-card.png)

**提问：** 北魏孝文帝改革中，促进民族交融的汉化措施有哪些？

**答案：** 说汉语、穿汉服、改汉姓、与汉族贵族联姻等。

**易错分析：** 不要只答“迁都洛阳”。迁都是重要措施，但汉化措施还包括语言、服饰、姓氏、婚姻等。

### 卡19

**配图：** ![](第四单元-三国两晋南北朝时期/19-card.png)

**提问：** 北魏孝文帝改革产生了什么历史影响？

**答案：** 促进了民族交融，也增强了北魏实力。

**易错分析：** 不要把孝文帝改革说成“阻碍民族交融”。它是促进北方民族交融的重要改革。

### 卡20

**配图：** ![](第四单元-三国两晋南北朝时期/20-card.png)

**提问：** 南北朝时期民族交融主要表现在哪些方面？

**答案：** 各民族在生产生活、语言、服饰、婚姻、制度和文化等方面相互影响。

**易错分析：** 不要把民族交融理解成某一民族完全消失。交融强调相互学习、相互影响。

### 卡21

**配图：** ![](第四单元-三国两晋南北朝时期/21-card.png)

**提问：** 三国两晋南北朝时期，数学家祖冲之最突出的科学成就是什么？

**答案：** 祖冲之把圆周率精确到小数点以后的第七位数字。

**易错分析：** 不要把祖冲之和张衡混淆。张衡是东汉科学家，发明地动仪；祖冲之生活在南北朝时期。

### 卡22

**配图：** ![](第四单元-三国两晋南北朝时期/22-card.png)

**提问：** 北朝农学家贾思勰的代表著作是什么？这部书属于哪一领域？

**答案：** 贾思勰著有《齐民要术》，这是一部农业科学著作。

**易错分析：** 不要把《齐民要术》写成医学著作。它总结的是农业生产经验。

### 卡23

**配图：** ![](第四单元-三国两晋南北朝时期/23-card.png)

**提问：** 东晋书法家王羲之的代表作是什么？他被后世誉为什么？

**答案：** 王羲之代表作是《兰亭集序》，被后世誉为“书圣”。

**易错分析：** 不要把“书圣”写成“画圣”。王羲之以书法著称。

### 卡24

**配图：** ![](第四单元-三国两晋南北朝时期/24-card.png)

**提问：** 南北朝时期石窟艺术的代表有哪些？

**答案：** 代表有山西大同云冈石窟和河南洛阳龙门石窟。

**易错分析：** 不要把石窟艺术只当成建筑。它综合体现雕塑、绘画、佛教文化和民族交融。

### 卡25

**配图：** ![](第四单元-三国两晋南北朝时期/25-card.png)

**提问：** 北魏地理学家郦道元的代表著作是什么？

**答案：** 郦道元著有《水经注》。

**易错分析：** 不要把《水经注》和《齐民要术》混淆。《水经注》是地理学著作，《齐民要术》是农业科学著作。

### 卡26

**配图：** ![](第四单元-三国两晋南北朝时期/26-card.png)

**提问：** 东晋画家顾恺之的代表作品有哪些？

**答案：** 顾恺之的代表作品有《女史箴图》和《洛神赋图》。

**易错分析：** 不要把顾恺之和王羲之混淆。顾恺之以绘画著称，王羲之以书法著称。

### 卡27

**配图：** ![](第四单元-三国两晋南北朝时期/27-card.png)

**提问：** 对比三国两晋南北朝时期的科技文化人物，祖冲之、贾思勰、王羲之分别对应什么成就？

**答案：** 祖冲之对应圆周率，贾思勰对应《齐民要术》，王羲之对应书法《兰亭集序》。

**易错分析：** 这类题容易把人物和成就错配。记忆时按领域分组：数学祖冲之，农学贾思勰，书法王羲之。
