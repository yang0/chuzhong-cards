# 第二单元 夏商周时期 配图 Prompts

统一风格：4:3 横版，初中生友好的可爱学习手账风。要求不是单纯可爱贴纸，而是“可爱外观 + 明确知识逻辑”。每张图必须有一个清楚的箭头链、对比结构或因果结构。少写难渲染文字，优先用图形说明关系。

## 21-dujiangyan-li-bing.png

都江堰建设者与地点。逻辑：岷江水 -> 李冰主持修建 -> 成都平原水利工程。画可爱的岷江水流、工程分水口、成都平原农田。标签：李冰 / 都江堰 / 成都平原。

## 22-dujiangyan-functions.png

都江堰功能。逻辑：岷江水 -> 分流工程 -> 防洪 + 灌溉 + 水运 -> 天府之国。画三条分支箭头，分别去洪水变小、农田变绿、小船通行。标签：防洪 / 灌溉 / 水运 / 天府之国。

## 23-hundred-schools-background.png

百家争鸣背景。逻辑：旧制度松动 + 新制度形成 + 社会大变革 -> 各家提出治国方案。画破旧旧牌匾、新制度种子、许多思想小灯泡。标签：社会大变革 / 各家主张 / 百家争鸣。

## 24-confucius-ren-education.png

孔子。逻辑：仁 -> 处理人与人关系；有教无类 -> 学生都能入学。画中心“仁”像温暖太阳，光照到不同学生。标签：孔子 / 仁 / 有教无类。

## 25-hanfei-legalism-qin.png

韩非法家。逻辑：法 -> 规则统一 -> 中央集权 -> 秦国采用。画法令卷轴连接到中央印章，再连接到秦国地图。标签：韩非 / 法治 / 中央集权 / 秦。

## 26-oracle-bones-writing.png

甲骨文。逻辑：龟甲兽骨上的文字 -> 商朝记录 -> 有文字可考历史开始。画龟甲、兽骨、刻痕文字、时间线从商朝亮起。标签：甲骨文 / 龟甲兽骨 / 商朝。

## 27-bronze-ritual-power.png

青铜器。逻辑：青铜铸造 -> 大型礼器 -> 等级与权力。画可爱的鼎在中心，后面是工匠火炉，旁边有等级台阶。避免写司母戊鼎。标签：青铜器 / 礼器 / 权力。

## 28-laozi-daoism.png

老子。逻辑：顺应自然 + 对立转化。画河水绕石自然流动，黑白小圆互相转化。标签：老子 / 顺应自然 / 对立转化。

## 29-mozi-love-no-war.png

墨子。逻辑：兼爱 -> 人人互相帮助；非攻 -> 反对不义战争。画两个城邦用桥连接互助，旁边战争云被红叉挡住。标签：墨子 / 兼爱 / 非攻。

## 30-mencius-benevolent-rule.png

孟子。逻辑：仁政 -> 爱护百姓 -> 民贵君轻。画百姓在天平更重，君主冠冕较轻；上方有温暖治理伞。标签：孟子 / 仁政 / 民贵君轻。

## 31-zhuangzi-freedom-nature.png

庄子。逻辑：顺应自然 -> 心灵自由。画小船顺水漂、蝴蝶飞出书页，轻松但不玄幻。标签：庄子 / 顺应自然 / 精神自由。

## 32-schools-comparison.png

儒家、道家、法家对比。逻辑：儒家靠仁德礼治，道家顺应自然，法家靠规则法治。画三条治国路线通向国家小图标：爱心礼书、流水树叶、法令尺子。标签：儒家 / 道家 / 法家。
