# 第三单元 秦汉时期

## 基本信息

- 科目：历史
- 年级：七年级上册
- 教材：人教版 / 统编版《中国历史》
- 单元：第三单元 秦汉时期
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元主要讲述秦统一中国、秦末农民起义、西汉建立与强盛、东汉兴衰、丝绸之路以及秦汉科技文化。

学习主线：

1. 秦灭六国，建立中国历史上第一个统一的多民族封建国家。
2. 秦朝暴政导致秦末农民起义，汉朝建立并在西汉前期逐渐恢复发展。
3. 汉武帝时期大一统局面巩固，丝绸之路开通，中外交流扩大。
4. 秦汉科技文化成就突出，对中国和世界文明产生深远影响。

核心对比：

- 秦始皇：完成统一，建立中央集权制度。
- 秦二世时期：暴政加剧，秦末农民起义爆发。
- 汉高祖刘邦：建立西汉。
- 文景之治：休养生息，社会经济恢复。
- 汉武帝：政治、思想、经济、军事多方面巩固大一统。
- 东汉：刘秀建立，出现光武中兴，后期外戚宦官交替专权。

## Anki 记忆卡

### 卡1

**配图：** ![](第三单元-秦汉时期/01-qin-unification.png)

**提问：** 七年级上册历史中，秦国在哪一年完成统一？建立者是谁？都城在哪里？

**答案：** 公元前 221 年，秦王嬴政完成统一，建立秦朝，定都咸阳。

**易错分析：** 不要把秦统一时间写成公元前 202 年。公元前 202 年是西汉建立。

### 卡2

**配图：** ![](第三单元-秦汉时期/02-first-unified-state.png)

**提问：** 秦统一六国后建立的国家，在中国历史上有什么重要地位？

**答案：** 秦朝是中国历史上第一个统一的多民族封建国家。

**易错分析：** 夏朝是中国历史上的第一个王朝，但不是第一个统一的多民族封建国家。

### 卡3

**配图：** ![](第三单元-秦汉时期/03-emperor-system.png)

**提问：** 秦朝中央集权制度中，最高统治者称什么？

**答案：** 最高统治者称皇帝，总揽全国一切军政大权。

**易错分析：** 不要把秦朝最高统治者称为“王”。秦始皇创立皇帝制度。

### 卡4

**配图：** ![](第三单元-秦汉时期/04-commandery-county.png)

**提问：** 秦朝在地方行政制度上推行了什么制度？

**答案：** 秦朝在地方推行郡县制，由中央直接派官吏治理地方。

**易错分析：** 不要把秦朝地方制度写成分封制。分封制是西周巩固统治的重要制度。

### 卡5

**配图：** ![](第三单元-秦汉时期/05-small-seal-script.png)

**提问：** 秦始皇统一文字时，把哪种字体作为全国规范文字？

**答案：** 秦始皇把小篆作为全国规范文字。

**易错分析：** 不要答成甲骨文。甲骨文主要是商朝文字，小篆是秦统一后的规范文字。

### 卡6

**配图：** ![](第三单元-秦汉时期/06-round-square-coin.png)

**提问：** 秦朝统一货币时，全国通用的货币是什么形制？

**答案：** 全国通用圆形方孔半两钱。

**易错分析：** 不要只答“铜钱”。考试常考具体形制：圆形方孔。

### 卡7

**配图：** ![](第三单元-秦汉时期/07-standardization-measures.png)

**提问：** 秦始皇为巩固统一，在经济和交通方面采取了哪些统一措施？

**答案：** 统一货币、统一度量衡，并统一车辆和道路的宽窄。

**易错分析：** 不要把“统一文字”漏掉或混入经济措施。文字属于文化制度方面，货币和度量衡属于经济方面。

### 卡8

**配图：** ![](第三单元-秦汉时期/08-great-wall-xiongnu.png)

**提问：** 秦朝修筑长城主要是为了抵御北方哪个民族的进攻？

**答案：** 主要是为了抵御匈奴。

**易错分析：** 不要把秦长城的主要对象写成蒙古。蒙古兴起在更晚的历史时期。

### 卡9

**配图：** ![](第三单元-秦汉时期/09-qin-tyranny.png)

**提问：** 秦朝灭亡的重要原因是什么？

**答案：** 秦朝实行暴政，徭役、赋税沉重，刑罚严酷，导致社会矛盾激化。

**易错分析：** 不要只说“农民起义导致秦亡”。农民起义是直接表现，深层原因是秦的暴政。

### 卡10

**配图：** ![](第三单元-秦汉时期/10-dazexiang-uprising.png)

**提问：** 中国历史上第一次大规模农民起义是哪一次？爆发地点与领导者是谁？

**答案：** 陈胜、吴广起义；爆发于大泽乡，领导者是陈胜、吴广。

**易错分析：** 不要把陈胜、吴广起义写成黄巾起义。黄巾起义发生在东汉末年。

### 卡11

**配图：** ![](第三单元-秦汉时期/11-chu-han-struggle.png)

**提问：** 秦末楚汉之争的双方是谁？最终谁取得胜利？

**答案：** 楚汉之争的双方是项羽和刘邦；最终刘邦获胜。

**易错分析：** 陈胜、吴广起义是反秦农民起义；楚汉之争是秦亡后刘邦与项羽争夺帝位的战争。

### 卡12

**配图：** ![](第三单元-秦汉时期/12-western-han-founded.png)

**提问：** 西汉由谁建立？建立时间和都城分别是什么？

**答案：** 刘邦建立西汉，时间是公元前 202 年，都城是长安。

**易错分析：** 不要把西汉都城写成洛阳。洛阳是东汉都城。

### 卡13

**配图：** ![](第三单元-秦汉时期/13-rest-and-recovery.png)

**提问：** 西汉初年统治者实行休养生息政策的主要原因是什么？

**答案：** 秦末战争后社会生产遭到严重破坏，经济凋敝，人口减少，需要恢复生产。

**易错分析：** 不要把休养生息理解成对外扩张政策。它的重点是减轻负担、恢复经济。

### 卡14

**配图：** ![](第三单元-秦汉时期/14-wenjing-rule.png)

**提问：** 西汉前期“文景之治”出现的主要原因是什么？

**答案：** 汉文帝、汉景帝继续实行休养生息政策，减轻赋税和徭役，重视农业生产。

**易错分析：** 不要把“文景之治”归到汉武帝时期。它发生在汉文帝、汉景帝时期。

### 卡15

**配图：** ![](第三单元-秦汉时期/15-tui-en-order.png)

**提问：** 汉武帝为削弱诸侯王势力，采纳了谁的什么建议？

**答案：** 汉武帝采纳主父偃的“推恩令”建议。

**易错分析：** 不要把推恩令理解成分封制。推恩令表面上让诸侯王分封子弟，实际削弱诸侯国势力。

### 卡16

**配图：** ![](第三单元-秦汉时期/16-respect-confucianism.png)

**提问：** 汉武帝在思想上巩固大一统，采纳了董仲舒的什么建议？

**答案：** 采纳董仲舒“罢黜百家，尊崇儒术”的建议。

**易错分析：** 不要写成秦始皇的焚书坑儒。汉武帝是尊崇儒术，使儒学成为正统思想。

### 卡17

**配图：** ![](第三单元-秦汉时期/17-economic-control.png)

**提问：** 汉武帝为加强中央对经济的控制，采取了哪些重要措施？

**答案：** 把铸币权收归中央，实行盐铁官营、专卖，并统一调配物资、平抑物价。

**易错分析：** 不要把这些措施写成商鞅变法内容。商鞅变法发生在战国秦国。

### 卡18

**配图：** ![](第三单元-秦汉时期/18-fight-xiongnu.png)

**提问：** 汉武帝时期，西汉在军事上北击哪个民族并巩固了边疆？

**答案：** 北击匈奴，巩固了北方边疆。

**易错分析：** 不要把汉武帝北击匈奴和秦始皇修长城混为同一事件。二者都与匈奴有关，但时期不同。

### 卡19

**配图：** ![](第三单元-秦汉时期/19-eastern-han-founded.png)

**提问：** 东汉由谁建立？都城在哪里？

**答案：** 刘秀建立东汉，定都洛阳。

**易错分析：** 刘秀即光武帝。不要把东汉建立者写成刘邦，刘邦建立的是西汉。

### 卡20

**配图：** ![](第三单元-秦汉时期/20-guangwu-revival.png)

**提问：** 东汉初年“光武中兴”指的是什么局面？

**答案：** 光武帝统治时期社会比较安定，经济得到恢复和发展，史称“光武中兴”。

**易错分析：** 不要把光武中兴和文景之治混淆。文景之治属于西汉，光武中兴属于东汉。

### 卡21

**配图：** ![](第三单元-秦汉时期/21-relatives-eunuchs.png)

**提问：** 东汉中后期政治黑暗的重要表现是什么？

**答案：** 外戚和宦官交替专权，政治腐朽。

**易错分析：** 不要只说“皇帝昏庸”。东汉后期的典型问题是外戚宦官交替专权。

### 卡22

**配图：** ![](第三单元-秦汉时期/22-yellow-turban.png)

**提问：** 东汉末年黄巾起义爆发的背景是什么？

**答案：** 东汉后期政治腐败、灾荒频繁、社会矛盾尖锐，导致农民起义爆发。

**易错分析：** 不要把黄巾起义和陈胜、吴广起义混淆。陈胜、吴广起义发生在秦末。

### 卡23

**配图：** ![](第三单元-秦汉时期/23-zhang-qian.png)

**提问：** 西汉时期，谁两次出使西域，促进了汉朝与西域各地的联系？

**答案：** 张骞两次出使西域，促进了汉朝与西域各地的联系。

**易错分析：** 不要把张骞写成班超。张骞主要是西汉出使西域，班超主要活动在东汉。

### 卡24

**配图：** ![](第三单元-秦汉时期/24-silk-road-route.png)

**提问：** 陆上丝绸之路从西汉都城出发，大致经过哪些地区通往欧洲？

**答案：** 从长安出发，经河西走廊、西域，通往中亚、西亚，直到欧洲。

**易错分析：** 不要把丝绸之路理解成只在中国境内的道路。它是连接东西方的交通和交流通道。

### 卡25

**配图：** ![](第三单元-秦汉时期/25-silk-road-meaning.png)

**提问：** 丝绸之路开通的历史意义是什么？

**答案：** 丝绸之路促进了中国同其他国家和地区的贸易与文化交流。

**易错分析：** 不要只答“卖丝绸”。丝绸只是代表性商品，核心意义是东西方经济文化交流。

### 卡26

**配图：** ![](第三单元-秦汉时期/26-cai-lun-paper.png)

**提问：** 汉朝哪项造纸技术改进与蔡伦有关？其历史意义是什么？

**答案：** 东汉蔡伦改进造纸术；促进了文化传播，是中国对世界文明的重要贡献。

**易错分析：** 不要写成蔡伦发明文字。蔡伦与改进造纸术有关。

### 卡27

**配图：** ![](第三单元-秦汉时期/27-zhang-zhongjing.png)

**提问：** 东汉医学家张仲景的代表著作是什么？他被后世尊称为什么？

**答案：** 张仲景著有《伤寒杂病论》，被后世尊称为“医圣”。

**易错分析：** 不要把张仲景和华佗混淆。华佗擅长外科手术，发明麻沸散。

### 卡28

**配图：** ![](第三单元-秦汉时期/28-shiji-simaqian.png)

**提问：** 西汉史学家司马迁写成的纪传体通史是什么？

**答案：** 司马迁写成《史记》，这是我国古代第一部纪传体通史。

**易错分析：** 不要把《史记》写成编年体。纪传体以人物传记为中心。

### 卡29

**配图：** ![](第三单元-秦汉时期/29-hua-tuo.png)

**提问：** 东汉名医华佗在医学上的重要贡献有哪些？

**答案：** 华佗擅长外科手术，发明“麻沸散”，还创编“五禽戏”。

**易错分析：** 不要把华佗和张仲景混淆。张仲景著有《伤寒杂病论》，华佗以外科、麻沸散和五禽戏著称。

### 卡30

**配图：** ![](第三单元-秦汉时期/30-buddhism-spread.png)

**提问：** 佛教大约在什么时候通过什么通道传入中国？

**答案：** 佛教大约在西汉末年经丝绸之路传入中国。

**易错分析：** 不要把佛教说成中国本土产生的宗教。佛教起源于古印度，经丝绸之路传入中国。

### 卡31

**配图：** ![](第三单元-秦汉时期/31-daoism-rise.png)

**提问：** 道教在中国历史上大约什么时候兴起？它与佛教来源有什么不同？

**答案：** 道教大约在东汉末年兴起，是中国本土宗教；佛教则是从古印度传入中国。

**易错分析：** 不要把道教和道家完全等同。道家是春秋战国时期的思想学派，道教是后来形成的宗教。
