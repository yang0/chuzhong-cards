# 第一单元 史前时期

## 基本信息

- 科目：历史
- 年级：七年级上册
- 教材：人教版 / 统编版《中国历史》
- 单元：第一单元 史前时期
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元主要讲述中国境内早期人类活动、原始农业的发展，以及中华文明的起源。

学习主线：

1. 远古人类出现，开始制作工具、使用火。
2. 原始农业发展，人类逐渐定居，形成聚落。
3. 社会分化加剧，早期城市和早期国家逐步出现，中华文明开始形成。

核心对比：

- 元谋人：云南元谋，距今约 170 万年。
- 北京人：北京周口店龙骨山，距今约 70 万年至 20 万年，会制作打制石器，会使用火并保存火种。
- 山顶洞人：模样和现代人基本相同，会人工取火，掌握磨光和钻孔技术。
- 河姆渡居民：长江流域，水稻，干栏式建筑。
- 半坡居民：黄河流域，粟、黍，半地穴式房屋。

## Anki 记忆卡

### 卡1

**配图：** ![](第一单元-史前时期/01-yuanmou-earliest-human.png)

**提问：** “远古时期的人类活动”中，我国境内目前已确认的最早古人类代表之一是谁？发现地点和年代分别是什么？

**答案：** 元谋人；发现于云南元谋，距今约 170 万年。

**易错分析：** 不要答成北京人。北京人距今约 70 万年至 20 万年，比元谋人晚。

### 卡2

**配图：** ![](第一单元-史前时期/02-peking-man-location-age.png)

**提问：** 七年级上册历史中，北京人遗址位于哪里？北京人大约生活在什么年代？

**答案：** 北京人遗址位于北京周口店龙骨山；距今约 70 万年至 20 万年。

**易错分析：** 不要把“北京人”和“山顶洞人”混为一谈。二者都与周口店有关，但年代和特征不同。

### 卡3

**配图：** ![](第一单元-史前时期/03-tool-making-human-standard.png)

**提问：** 在北京人的生产生活中，最能体现其已经是“人”的劳动能力的表现是什么？

**答案：** 北京人会制作和使用打制石器。

**易错分析：** 不要只答“会使用工具”。历史课强调“制造工具”是区分人与动物的重要标准。

### 卡4

**配图：** ![](第一单元-史前时期/04-peking-man-fire.png)

**提问：** 北京人在用火方面有什么特点？

**答案：** 北京人已经能够使用火，并且会长时间保存火种。

**易错分析：** 不要写成“北京人会人工取火”。人工取火通常对应山顶洞人的进步表现。

### 卡5

**配图：** ![](第一单元-史前时期/05-shandingdong-progress.png)

**提问：** 山顶洞人与北京人相比，在用火和工具技术方面有哪些进步？

**答案：** 山顶洞人已经懂得人工取火，并掌握磨光和钻孔技术。

**易错分析：** 不要把“人工取火、磨光钻孔”套到北京人身上。北京人主要是打制石器、保存火种。

### 卡6

**配图：** ![](第一单元-史前时期/06-shandingdong-modern-appearance.png)

**提问：** 七年级上册历史中，山顶洞人的外貌与北京人相比有什么明显变化？

**答案：** 山顶洞人的模样和现代人基本相同。

**易错分析：** 不要笼统写“更聪明”。考试通常考具体表述：模样和现代人基本相同。

### 卡7

**配图：** ![](第一单元-史前时期/07-origin-agriculture-signs.png)

**提问：** 史前时期原始农业兴起的主要表现有哪些？

**答案：** 农作物种植、家畜饲养、聚落定居、磨制石器发展。

**易错分析：** 不要只答“种植农作物”。原始农业兴起是一组变化，常和定居、家畜、磨制石器一起考。

### 卡8

**配图：** ![](第一单元-史前时期/08-agriculture-settlement.png)

**提问：** 史前时期原始农业的发展，对人类生活方式产生了什么重要影响？

**答案：** 促进人类从迁徙生活走向定居生活。

**易错分析：** 不要直接写“出现国家”。原始农业为文明形成奠定基础，但不等于国家已经出现。

### 卡9

**配图：** ![](第一单元-史前时期/09-hemudu-rice-yangtze.png)

**提问：** 河姆渡居民生活在哪一流域？他们主要种植哪种农作物？

**答案：** 河姆渡居民生活在长江流域，主要种植水稻。

**易错分析：** 不要和半坡居民混淆。河姆渡对应“长江流域、水稻”。

### 卡10

**配图：** ![](第一单元-史前时期/10-hemudu-stilt-house.png)

**提问：** 河姆渡居民为什么适合居住干栏式建筑？

**答案：** 河姆渡居民生活在南方长江流域，气候潮湿，干栏式建筑有利于通风、防潮。

**易错分析：** 不要只机械背“河姆渡住干栏式房屋”，要知道它和南方潮湿环境有关。

### 卡11

**配图：** ![](第一单元-史前时期/11-banpo-millet-yellow-river.png)

**提问：** 半坡居民生活在哪一流域？他们主要种植哪类农作物？

**答案：** 半坡居民生活在黄河流域，主要种植粟、黍。

**易错分析：** 不要答成水稻。水稻对应长江流域的河姆渡居民。

### 卡12

**配图：** ![](第一单元-史前时期/12-banpo-semi-subterranean-house.png)

**提问：** 半坡居民的房屋类型是什么？这种房屋与什么自然环境有关？

**答案：** 半坡居民主要居住半地穴式房屋；与北方黄河流域相对寒冷、干燥的环境有关。

**易错分析：** 不要和河姆渡的干栏式建筑对调。

### 卡13

**配图：** ![](第一单元-史前时期/13-rice-vs-millet-comparison.png)

**提问：** 对比河姆渡居民和半坡居民，长江流域与黄河流域在农作物上最典型的区别是什么？

**答案：** 长江流域的河姆渡居民主要种植水稻；黄河流域的半坡居民主要种植粟、黍。

**易错分析：** 这组对比非常高频，最容易把“水稻”和“粟黍”互换。

### 卡14

**配图：** ![](第一单元-史前时期/14-house-type-comparison.png)

**提问：** 对比河姆渡居民和半坡居民，二者房屋类型分别是什么？

**答案：** 河姆渡居民住干栏式建筑；半坡居民住半地穴式房屋。

**易错分析：** 记忆钩子：南方潮湿，架起来；北方较冷，挖下去。

### 卡15

**配图：** ![](第一单元-史前时期/15-civilization-origin-changes.png)

**提问：** 中华文明起源阶段，社会从普通聚落走向早期文明，出现了哪些重要变化？

**答案：** 早期城市逐步形成，社会分化加剧，出现掌握权力的人物，早期国家开始形成。

**易错分析：** 不要只答“会种地”。文明起源重点是城市、分化、权力和早期国家。

### 卡16

**配图：** ![](第一单元-史前时期/16-liangzhu-civilization.png)

**提问：** 良渚古城遗址为什么能说明中华文明已经进入较高发展阶段？

**答案：** 良渚古城有大型城址、水利系统、高等级建筑、贵族墓地和精美玉器，反映出社会分化和权力组织。

**易错分析：** 不要把良渚古城当作普通村落遗址。它的重点是“古城 + 分化 + 权力 + 文明”。

### 卡17

**配图：** ![](第一单元-史前时期/17-yao-shun-yu-abdication.png)

**提问：** 传说中尧、舜、禹时期，部落联盟首领通过什么方式更替？这种制度叫什么？

**答案：** 通过推举贤德之人的方式更替，称为禅让制。

**易错分析：** 不要只问“禅让制是什么”。要记住它属于传说中的尧舜禹时期，核心是“选贤”。

### 卡18

**配图：** ![](第一单元-史前时期/18-dayu-flood-control.png)

**提问：** 传说中大禹治水采用的主要方法是什么？这与以前堵塞洪水的方法有什么不同？

**答案：** 大禹采用疏导的方法治水；不是单纯堵水，而是引导洪水流入江河。

**易错分析：** 不要只背“三过家门而不入”。考试更常考治水方法：疏导。

### 卡19

**配图：** ![](第一单元-史前时期/19-prehistory-development-sequence.png)

**提问：** 从远古人类到中华文明起源，史前时期的发展顺序可以怎样概括？

**答案：** 远古人类出现并使用工具、火；原始农业发展并出现定居聚落；社会分化加剧，早期城市和早期国家逐步形成。

**易错分析：** 这不是问“第一单元总线索是什么”，而是考发展顺序。不要把遗址、人名、制度零散记忆，要抓住“人类活动 -> 农业定居 -> 文明起源”。
