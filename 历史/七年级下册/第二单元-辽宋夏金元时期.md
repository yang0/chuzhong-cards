# 第二单元 辽宋夏金元时期：民族关系发展和社会变化

## 基本信息

- 科目：历史
- 年级：七年级下册
- 教材：人教版 / 统编版《中国历史》（2024 版目录）
- 单元：第二单元 辽宋夏金元时期：民族关系发展和社会变化
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元围绕辽、宋、夏、金、元多民族政权并立到元朝统一，以及社会经济、对外交流、科技文化的发展展开。

学习主线：北宋强化中央集权，但与辽、西夏并立；金灭辽和北宋，南宋与金对峙；蒙古兴起并建立元朝，最终完成统一。同期经济重心南移完成，海外贸易和科技文化高度发展。

## Anki 记忆卡

### 卡1

**配图：** ![](第二单元-辽宋夏金元时期/01-card.png)

**提问：** 北宋由谁建立？建立时间和都城分别是什么？

**答案：** 960 年，赵匡胤建立北宋，定都东京，又称汴京。

**易错分析：** 不要把北宋都城写成临安。临安是南宋都城。

### 卡2

**配图：** ![](第二单元-辽宋夏金元时期/02-card.png)

**提问：** 宋太祖为防止武将专权，采取了什么著名措施？

**答案：** 宋太祖通过“杯酒释兵权”等方式解除禁军高级将领兵权，加强中央对军队的控制。

**易错分析：** 不要把杯酒释兵权理解成削弱中央。它恰恰是加强皇权和中央集权。

### 卡3

**配图：** ![](第二单元-辽宋夏金元时期/03-card.png)

**提问：** 北宋重文轻武政策的目的是什么？

**答案：** 目的是防止唐末以来武将专横跋扈的弊端重现，维护中央集权。

**易错分析：** 不要只说宋朝轻视军队。重文轻武有利于政权稳定，但也造成军队战斗力较弱。

### 卡4

**配图：** ![](第二单元-辽宋夏金元时期/04-card.png)

**提问：** 辽、西夏分别是由哪些民族建立的政权？

**答案：** 辽由契丹族建立，西夏由党项族建立。

**易错分析：** 不要把辽写成女真族建立。女真族建立的是金。

### 卡5

**配图：** ![](第二单元-辽宋夏金元时期/05-card.png)

**提问：** 北宋与辽之间达成的盟约叫什么？其影响是什么？

**答案：** 澶渊之盟。此后辽宋之间保持了较长时间和平局面，双方边境贸易发展。

**易错分析：** 不要只记宋给岁币就判断完全消极。教材也强调它带来相对和平和交流。

### 卡6

**提问：** 金政权由哪个民族建立？后来灭掉了哪两个政权？

**答案：** 金由女真族建立，先灭辽，后灭北宋。

**易错分析：** 不要把金和后来的清朝混为一谈。二者都与女真有关，但历史时期不同。

**易错分析：** 不要把南宋建立者写成赵匡胤。赵匡胤建立北宋。

### 卡8

**配图：** ![](第二单元-辽宋夏金元时期/08-card.png)

**答案：** 岳飞率领岳家军在郾城大败金军，取得郾城大捷。

**易错分析：** 不要把岳飞抗金写成抗元。岳飞生活在宋金对峙时期。

### 卡9

**配图：** ![](第二单元-辽宋夏金元时期/09-card.png)

**提问：** 宋金达成和议后，南宋与金形成了怎样的局面？

**答案：** 宋金和议后，南宋向金称臣并给岁币，双方以淮水至大散关一线为界，形成南北对峙局面。

**易错分析：** 不要把宋金和议理解成南宋统一全国。它形成的是对峙，不是统一。

### 卡10

**配图：** ![](第二单元-辽宋夏金元时期/10-card.png)

**提问：** 蒙古族兴起过程中，铁木真在 1206 年完成了什么大事？

**答案：** 1206 年，铁木真统一蒙古草原，建立蒙古政权，被尊称为成吉思汗。

**易错分析：** 不要把成吉思汗写成元朝建立者。元朝建立者是忽必烈。

### 卡11

**提问：** 元朝由谁建立？建立时间和都城分别是什么？

**答案：** 1271 年，忽必烈改国号为元，次年定都大都。

**易错分析：** 不要把元朝建立时间写成 1279 年。1279 年是元灭南宋、完成统一。

### 卡12

**配图：** ![](第二单元-辽宋夏金元时期/12-card.png)

**提问：** 元朝统一全国的时间和历史意义是什么？

**答案：** 1279 年，元朝灭南宋，完成全国统一，结束了较长时期的政权并立局面。

**易错分析：** 不要把 1271 年和 1279 年混淆。1271 年建元，1279 年统一。

### 卡13

**配图：** ![](第二单元-辽宋夏金元时期/13-card.png)

**提问：** 元朝为有效管辖辽阔疆域，在地方实行了什么制度？

**答案：** 元朝在地方实行行省制度。

**易错分析：** 不要把行省制度写成郡县制。郡县制秦朝推行，行省制度是元朝的重要地方行政制度。

### 卡14

**提问：** 元朝中央管辖西藏地区的机构是什么？这说明了什么？

**答案：** 元朝由宣政院直接统辖西藏事务，说明西藏正式成为中央直接管辖下的地方行政区域。

**易错分析：** 不要把西藏正式纳入中央管辖写到清朝。元朝已设宣政院管理西藏。

### 卡17

**易错分析：** 不要把“苏湖熟，天下足”理解为北方农业优势。它体现南方经济发展。

**提问：** 辽宋夏金元时期民族关系发展的总体趋势是什么？

**答案：** 政权并立中有战争也有和平交往，各民族之间交流频繁，民族交融进一步发展。

**易错分析：** 不要只用战争概括这一时期。并立、对峙、交流、交融都是关键词。

### 卡29

**配图：** ![](第二单元-辽宋夏金元时期/29-card.png)

**提问：** 元朝时期形成的一个新的民族是什么？

**答案：** 元朝时期，来自中亚、西亚的人同汉、蒙古、畏兀儿等长期杂居相处，逐渐形成回族。

**易错分析：** 不要把回族形成写到唐朝。唐朝有外来人口交流，回族形成主要在元朝。

### 卡30

**配图：** ![](第二单元-辽宋夏金元时期/30-card.png)

**提问：** 七年级下册第二单元的时代特征可以怎样概括？

**答案：** 辽宋夏金元时期的时代特征是民族关系发展和社会变化，突出表现为多民族政权并立到统一、经济重心南移完成、科技文化繁荣。

**易错分析：** 不要只概括为“宋朝经济”。本单元同时包含民族关系、统一进程、经济和文化变化。
