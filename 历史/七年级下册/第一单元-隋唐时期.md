# 第一单元 隋唐时期：繁荣与开放的时代

## 基本信息

- 科目：历史
- 年级：七年级下册
- 教材：人教版 / 统编版《中国历史》（2024 版目录）
- 单元：第一单元 隋唐时期：繁荣与开放的时代
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元围绕隋唐时期的统一、制度创新、盛世局面、民族交融、中外交流和科技文化展开。

学习主线：隋朝完成统一并开通大运河，唐朝在贞观之治和开元盛世中走向繁荣，安史之乱后由盛转衰；隋唐开放包容，形成频繁的民族交往和中外文化交流。

## Anki 记忆卡

### 卡1

**配图：** ![](第一单元-隋唐时期/01-card.png)

**提问：** 隋朝在哪一年完成全国统一？统一的历史意义是什么？

**答案：** 589 年，隋文帝灭陈，统一全国，结束了长期分裂局面，顺应了统一多民族国家发展的趋势。

**易错分析：** 不要把隋朝统一时间写成 581 年。581 年是隋朝建立，589 年才是统一全国。

### 卡2

**配图：** ![](第一单元-隋唐时期/02-card.png)

**提问：** 隋文帝统治时期出现的社会经济发展局面，历史上称为什么？

**答案：** 隋文帝时期，国家统一安定，经济恢复发展，人口和垦田面积增加，史称“开皇之治”。

**易错分析：** 不要把“开皇之治”和唐太宗时期的“贞观之治”混淆。

### 卡3

**配图：** ![](第一单元-隋唐时期/03-card.png)

**提问：** 隋炀帝开通大运河的主要目的是什么？

**答案：** 主要目的是加强南北交通，巩固隋王朝对全国的统治。

**易错分析：** 不要只答“方便旅游”或“运输粮食”。运输是重要作用，但教材表述的主要目的是加强南北交通、巩固统治。

### 卡4

**配图：** ![](第一单元-隋唐时期/04-card.png)

**提问：** 隋朝大运河以哪里为中心？北抵哪里，南至哪里？

**答案：** 隋朝大运河以洛阳为中心，北抵涿郡，南至余杭。

**易错分析：** 不要把中心写成大都或北京。大都是元朝都城，隋朝大运河中心是洛阳。

### 卡5

**配图：** ![](第一单元-隋唐时期/05-card.png)

**提问：** 隋朝大运河开通后，对中国古代经济交流有什么作用？

**答案：** 大运河加强了南北地区政治、经济和文化交流。

**易错分析：** 不要只看到徭役沉重。大运河开凿确实加重人民负担，但长期作用是促进南北交流。

### 卡6

**配图：** ![](第一单元-隋唐时期/06-card.png)

**提问：** 中国古代科举制正式确立的标志是什么？

**答案：** 隋炀帝时进士科的创立，标志着科举制正式确立。

**易错分析：** 不要把科举制正式确立写成隋文帝创立。隋文帝开始用考试选官，隋炀帝设进士科才是正式确立。

### 卡7

**配图：** ![](第一单元-隋唐时期/07-card.png)

**提问：** 隋朝灭亡的根本原因是什么？

**答案：** 隋炀帝的残暴统治，包括大兴土木、多次巡游、三征辽东和徭役兵役沉重，激化社会矛盾。

**易错分析：** 不要只写“农民起义导致灭亡”。起义是直接推动，根源是隋炀帝暴政。

### 卡8

**配图：** ![](第一单元-隋唐时期/08-card.png)

**提问：** 唐朝由谁建立？建立时间和都城分别是什么？

**答案：** 618 年，李渊建立唐朝，定都长安。

**易错分析：** 不要把唐朝建立者写成李世民。李世民是唐太宗，唐朝建立者是李渊。

### 卡9

**配图：** ![](第一单元-隋唐时期/09-card.png)

**提问：** 唐太宗统治时期出现的治世局面称为什么？

**答案：** 唐太宗统治时期，政治比较清明，经济发展，国力增强，史称“贞观之治”。

**易错分析：** 不要把“贞观之治”放到唐玄宗时期。唐玄宗前期是“开元盛世”。

### 卡10

**配图：** ![](第一单元-隋唐时期/10-card.png)

**提问：** 唐太宗在用人和纳谏方面有什么突出做法？

**答案：** 唐太宗吸取隋亡教训，勤于政事，虚心纳谏，从善如流，广纳贤才，知人善任。

**易错分析：** 不要只背人物名字。考题常要求说明唐太宗善于用人纳谏与贞观之治的关系。

### 卡11

**配图：** ![](第一单元-隋唐时期/11-card.png)

**提问：** 武则天统治对唐朝历史发展有什么作用？

**答案：** 武则天继续推行发展生产、选拔人才的政策，使唐朝社会经济继续发展，为后来开元盛世奠定基础。

**易错分析：** 不要只记武则天是女皇。历史评价要抓住她承前启后的作用。

### 卡12

**配图：** ![](第一单元-隋唐时期/12-card.png)

**提问：** 唐玄宗统治前期出现的鼎盛局面称为什么？

**答案：** 唐玄宗统治前期，政治稳定，经济繁荣，国库充盈，民众生活安定，唐朝进入鼎盛时期，史称“开元盛世”。

**易错分析：** 不要把唐玄宗后期的衰落也算进开元盛世。开元盛世主要指唐玄宗统治前期。

### 卡13

**配图：** ![](第一单元-隋唐时期/13-card.png)

**提问：** 唐朝农业生产工具中，曲辕犁的作用是什么？

**答案：** 曲辕犁便于深耕和灵活操作，提高了耕作效率。

**易错分析：** 不要把曲辕犁和筒车混淆。曲辕犁用于耕地，筒车用于灌溉。

### 卡14

**配图：** ![](第一单元-隋唐时期/14-card.png)

**提问：** 唐朝农业生产工具中，筒车的作用是什么？

**答案：** 筒车利用水力进行灌溉，提高了农业灌溉效率。

**易错分析：** 不要把筒车说成交通工具。它是灌溉工具。

### 卡15

**配图：** ![](第一单元-隋唐时期/15-card.png)

**提问：** 唐朝陶瓷业中闻名中外的彩色陶器是什么？

**答案：** 唐三彩是唐朝闻名中外的彩色陶器。

**易错分析：** 不要把唐三彩理解为只有三种颜色。“三彩”泛指多彩釉色。

### 卡16

**配图：** ![](第一单元-隋唐时期/16-card.png)

**提问：** 唐朝长安城在当时世界城市史上有什么地位？

**答案：** 长安城规模宏伟，布局严整，是当时中国政治、经济和文化交往中心，也是一座国际性大都会。

**易错分析：** 不要把长安只看成普通都城。唐长安突出特点是开放和国际性。

### 卡17

**配图：** ![](第一单元-隋唐时期/17-card.png)

**提问：** 唐太宗时期，唐朝与吐蕃友好交往的代表事件是什么？

**答案：** 文成公主入藏，嫁给吐蕃赞普松赞干布，促进了唐蕃经济文化交流。

**易错分析：** 不要把文成公主入藏说成战争事件。它是民族交往与友好交流的代表。

### 卡18

**配图：** ![](第一单元-隋唐时期/18-card.png)

**提问：** 唐朝实行开明民族政策带来了什么影响？

**答案：** 促进了边疆各族与中原地区的交往、交流、交融，增强了统一多民族国家的发展。

**易错分析：** 不要把民族交融理解成单向同化。它包含经济、文化、生活方式等多方面相互影响。

### 卡19

**配图：** ![](第一单元-隋唐时期/19-card.png)

**提问：** 唐朝时期，日本派往中国学习制度和文化的使团叫什么？

**答案：** 日本派遣唐使来华，学习唐朝先进制度、天文历法、文字、建筑等文化。

**易错分析：** 不要把遣唐使写成鉴真。遣唐使是日本来华使团，鉴真是唐朝僧人东渡日本。

### 卡20

**配图：** ![](第一单元-隋唐时期/20-card.png)

**提问：** 唐朝高僧鉴真东渡日本的历史贡献是什么？

**答案：** 鉴真东渡日本，传播佛教、医药、文学、书法、建筑等唐朝文化。

**易错分析：** 不要把鉴真和玄奘混淆。鉴真是东渡日本，玄奘是西行天竺。

### 卡21

**配图：** ![](第一单元-隋唐时期/21-card.png)

**提问：** 唐朝高僧玄奘西行天竺的主要贡献是什么？

**答案：** 玄奘西行天竺取经，回国后主持翻译佛经，并写成《大唐西域记》。

**易错分析：** 不要把玄奘西行说成去日本。玄奘去的是天竺，即古印度地区。

### 卡22

**配图：** ![](第一单元-隋唐时期/22-card.png)

**提问：** 安史之乱对唐朝国势产生了什么影响？

**答案：** 安史之乱使唐朝由盛转衰，中央权力削弱，藩镇割据局面逐渐形成。

**易错分析：** 不要把安史之乱说成唐朝灭亡的时间点。它是由盛转衰的转折点，不是唐亡本身。

### 卡23

**配图：** ![](第一单元-隋唐时期/23-card.png)

**提问：** 唐朝后期黄巢起义产生了什么影响？

**答案：** 黄巢起义给唐朝统治以致命打击，加速了唐朝灭亡。

**易错分析：** 不要把黄巢起义和东汉黄巾起义混淆。黄巢起义发生在唐朝末年。

### 卡24

**配图：** ![](第一单元-隋唐时期/24-card.png)

**提问：** 唐朝灭亡后，中国历史进入了哪一分裂时期？

**答案：** 907 年唐朝灭亡后，中国进入五代十国时期。

**易错分析：** 不要把五代十国放在隋唐之前。它发生在唐朝灭亡之后。

### 卡25

**配图：** ![](第一单元-隋唐时期/25-card.png)

**提问：** 隋唐时期文化繁荣的主要原因可以从哪些方面理解？

**答案：** 国家统一强盛、经济繁荣、民族交融、中外交流频繁，共同推动隋唐文化繁荣。

**易错分析：** 不要只背诗人作品。文化繁荣背后有政治、经济、民族和对外交往基础。

### 卡26

**配图：** ![](第一单元-隋唐时期/26-card.png)

**提问：** 唐朝诗歌繁荣，李白、杜甫、白居易分别有什么称号或特点？

**答案：** 李白被称为“诗仙”，诗风飘逸洒脱；杜甫被称为“诗圣”，诗反映历史现实；白居易诗歌通俗易懂。

**易错分析：** 不要把李白和杜甫称号写反。李白是诗仙，杜甫是诗圣。

### 卡27

**配图：** ![](第一单元-隋唐时期/27-card.png)

**提问：** 隋唐书法名家颜真卿和柳公权的书法合称什么？

**答案：** 颜真卿、柳公权的楷书端正劲美，后人称为“颜筋柳骨”。

**易错分析：** 不要把颜真卿、柳公权归入宋元书法。二人是唐朝书法代表。

### 卡28

**配图：** ![](第一单元-隋唐时期/28-card.png)

**提问：** 隋唐时期最突出的时代特征是什么？

**答案：** 隋唐时期的突出时代特征是繁荣与开放。

**易错分析：** 不要只概括为“统一”。统一是基础，单元标题强调的是繁荣与开放。
