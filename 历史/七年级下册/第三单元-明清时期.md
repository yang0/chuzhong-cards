# 第三单元 明清时期（至鸦片战争前）：统一多民族封建国家的巩固与发展

## 基本信息

- 科目：历史
- 年级：七年级下册
- 教材：人教版 / 统编版《中国历史》（2024 版目录）
- 单元：第三单元 明清时期（至鸦片战争前）：统一多民族封建国家的巩固与发展
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析

## 单元纲要

本单元围绕明清时期至鸦片战争前统一多民族封建国家的巩固与发展展开。

学习主线：明朝强化皇权并开展郑和下西洋，后期因政治腐败和农民起义灭亡；清朝建立后加强边疆治理和君主专制，社会经济继续发展，但闭关锁国逐渐限制了对外联系。

## Anki 记忆卡

### 卡1

**配图：** ![](第三单元-明清时期/01-card.png)

**提问：** 明朝由谁建立？建立时间和都城最初在哪里？

**答案：** 1368 年，朱元璋建立明朝，定都应天府。

**易错分析：** 不要把明朝建立者写成朱棣。朱棣是明成祖，后来迁都北京。

### 卡2

**配图：** ![](第三单元-明清时期/02-card.png)

**提问：** 明太祖朱元璋为强化皇权，在中央废除了什么制度？

**答案：** 明太祖废除丞相制度和中书省，提升六部职权，使皇权进一步加强。

**易错分析：** 不要把废丞相写成秦始皇措施。秦朝设丞相，明太祖废丞相。

### 卡3

**配图：** ![](第三单元-明清时期/03-card.png)

**提问：** 明朝为监视官民，先后设立了哪些特务机构？

**答案：** 明朝设立锦衣卫，后来又设东厂等厂卫特务机构。

**易错分析：** 不要把锦衣卫理解成普通军队。它的重要职能是监视官民、侦缉。

### 卡4

**配图：** ![](第三单元-明清时期/04-card.png)

**提问：** 明朝科举考试的命题范围主要是什么？

**答案：** 明朝科举考试题目必须来自“四书”“五经”，考生答卷格式逐渐形成八股文。

**易错分析：** 不要把八股取士说成促进思想自由。它禁锢思想，服务于皇权统治。

### 卡5

**配图：** ![](第三单元-明清时期/05-card.png)

**提问：** 明成祖时期，郑和下西洋的主要目的是什么？

**答案：** 主要目的是提高明朝在国外的地位和威望，同时开展海外交往。

**易错分析：** 不要把郑和下西洋理解成殖民扩张。它主要是和平交往和宣扬国威。

### 卡6

**配图：** ![](第三单元-明清时期/06-card.png)

**提问：** 郑和下西洋最远到达哪里？

**答案：** 郑和船队最远到达非洲东海岸和红海沿岸。

**易错分析：** 不要把郑和航行终点写成欧洲或美洲。

### 卡7

**配图：** ![](第三单元-明清时期/07-card.png)

**提问：** 郑和下西洋的历史意义是什么？

**答案：** 增进了中国与亚非国家和地区的相互了解和友好往来，开创了西太平洋与印度洋之间的亚非海上交通线。

**易错分析：** 不要只答“规模大”。考试更常考和平交往和海上交通意义。

### 卡8

**配图：** ![](第三单元-明清时期/08-card.png)

**提问：** 明朝中期抗倭名将是谁？他领导的军队叫什么？

**答案：** 戚继光是明朝抗倭名将，他训练的军队称为戚家军。

**易错分析：** 不要把戚继光抗倭写成抗击沙俄。抗击沙俄是清朝康熙时期相关史事。

### 卡9

**配图：** ![](第三单元-明清时期/09-card.png)

**提问：** 明长城修筑的主要目的是什么？

**答案：** 明朝修筑长城主要是为了防御北方蒙古贵族南扰。

**易错分析：** 不要把明长城防御对象写成匈奴。秦汉长城常与匈奴相关，明长城主要防御蒙古贵族。

### 卡10

**配图：** ![](第三单元-明清时期/10-card.png)

**提问：** 明朝北京城的核心建筑群是什么？

**答案：** 明朝北京城的核心是宫城，即紫禁城。

**易错分析：** 不要把紫禁城说成普通商业区。它是皇帝居住和处理政务的宫城。

### 卡11

**配图：** ![](第三单元-明清时期/11-card.png)

**提问：** 明朝灭亡的主要原因有哪些？

**答案：** 明朝后期政治腐败、财政危机严重、土地兼并突出、灾荒频繁，农民起义不断爆发。

**易错分析：** 不要只答李自成起义。起义是直接推动，深层原因是明朝统治危机。

### 卡12

**配图：** ![](第三单元-明清时期/12-card.png)

**提问：** 推翻明朝统治的农民起义领袖是谁？他建立的政权叫什么？

**答案：** 李自成领导农民起义，建立大顺政权，1644 年攻入北京，明朝灭亡。

**易错分析：** 不要把李自成写成建立清朝。清朝由满洲贵族建立。

### 卡13

**配图：** ![](第三单元-明清时期/13-card.png)

**提问：** 清朝前身后金是谁建立的？后来谁改国号为清？

**答案：** 努尔哈赤建立后金；皇太极改族名为满洲，并于 1636 年改国号为清。

**易错分析：** 不要把努尔哈赤和皇太极作用混淆。努尔哈赤建后金，皇太极改国号为清。

### 卡14

**配图：** ![](第三单元-明清时期/14-card.png)

**提问：** 郑成功收复台湾的时间和意义是什么？

**答案：** 1662 年，郑成功打败荷兰殖民者，收复台湾，维护了国家主权。

**易错分析：** 不要把郑成功收复台湾写成清政府直接完成。郑成功先收复台湾，清朝后来统一台湾。

### 卡15

**配图：** ![](第三单元-明清时期/15-card.png)

**提问：** 清朝设置台湾府的时间和意义是什么？

**答案：** 1684 年，清朝设置台湾府，隶属福建省，加强了中央政府对台湾的管辖，促进台湾开发。

**易错分析：** 不要把台湾府设置时间写成 1662 年。1662 年是郑成功收复台湾。

### 卡16

**配图：** ![](第三单元-明清时期/16-card.png)

**提问：** 清朝对西藏地方管理中，册封制度和驻藏大臣分别有什么作用？

**答案：** 清朝册封达赖、班禅，并设置驻藏大臣，与达赖、班禅共同管理西藏事务，加强对西藏的管辖。

**易错分析：** 不要只写宗教关系。清朝对西藏管理体现中央政府管辖。

### 卡17

**配图：** ![](第三单元-明清时期/17-card.png)

**提问：** 清朝乾隆时期平定大小和卓叛乱后，设置了什么机构管理新疆？

**答案：** 清朝设置伊犁将军，管辖包括巴勒喀什池在内的整个新疆地区。

**易错分析：** 不要把伊犁将军写成管理西藏。西藏相关机构是驻藏大臣。

### 卡18

**配图：** ![](第三单元-明清时期/18-card.png)

**提问：** 清朝为反击沙俄侵略，康熙帝组织了哪两次战役？

**答案：** 康熙帝组织两次雅克萨之战，打败沙俄侵略军。

**易错分析：** 不要把雅克萨之战写成抗倭战争。抗倭主要发生在明朝中期。

### 卡19

**配图：** ![](第三单元-明清时期/19-card.png)

**提问：** 1689 年中俄签订的边界条约是什么？

**答案：** 1689 年，中俄签订《尼布楚条约》，从法律上肯定了黑龙江和乌苏里江流域包括库页岛在内的广大地区都是中国领土。

**易错分析：** 不要把《尼布楚条约》写成不平等条约。教材中它是平等协商签订的边界条约。

### 卡20

**配图：** ![](第三单元-明清时期/20-card.png)

**提问：** 清朝前期统一多民族国家巩固的表现有哪些？

**答案：** 清朝加强对台湾、西藏、新疆等边疆地区的管辖，并反击沙俄侵略，巩固了统一多民族国家。

**易错分析：** 不要只背单个事件。要能把台湾、西藏、新疆、东北边疆串成边疆治理主线。

### 卡21

**配图：** ![](第三单元-明清时期/21-card.png)

**提问：** 清朝雍正时期设立军机处的影响是什么？

**答案：** 军机处的设立使君主专制进一步强化。

**易错分析：** 不要把军机处理解为分散皇权的机构。它提高办事效率，但权力集中于皇帝。

### 卡22

**配图：** ![](第三单元-明清时期/22-card.png)

**提问：** 明清时期文字狱造成了什么影响？

**答案：** 文字狱禁锢了人们的思想言论，严重阻碍思想、学术的发展和进步。

**易错分析：** 不要把文字狱看成普通文化管理。它是君主专制强化在思想文化领域的表现。

### 卡23

**配图：** ![](第三单元-明清时期/23-card.png)

**提问：** 清朝实行闭关锁国政策的主要表现是什么？

**答案：** 清朝严格限制对外贸易，后来只开放广州一处作为对外通商口岸，并由广州十三行统一经营对外贸易。

**易错分析：** 不要把闭关锁国理解成完全没有对外贸易。它是严格限制，不是绝对断绝。

### 卡24

**配图：** ![](第三单元-明清时期/24-card.png)

**提问：** 闭关锁国政策对中国社会发展产生了什么消极影响？

**答案：** 它使中国逐渐落伍于世界历史发展进程，限制了中外经济文化交流。

**易错分析：** 不要只答“防御侵略”。闭关锁国有自卫心理，但消极影响更突出。

### 卡25

**配图：** ![](第三单元-明清时期/25-card.png)

**提问：** 明清时期农业和商品经济发展的表现有哪些？

**答案：** 高产作物推广，粮食产量提高；手工业和商业发展，出现较大商帮，如晋商、徽商。

**易错分析：** 不要把高产作物玉米、甘薯的推广写到唐宋。大规模推广主要在明清。

### 卡26

**配图：** ![](第三单元-明清时期/26-card.png)

**提问：** 明朝科技名著《本草纲目》的作者是谁？内容特点是什么？

**答案：** 李时珍著《本草纲目》，是一部规模空前的药物学著作。

**易错分析：** 不要把《本草纲目》写成农业著作。农业著作代表是《农政全书》。

### 卡27

**配图：** ![](第三单元-明清时期/27-card.png)

**提问：** 明朝科技名著《天工开物》的作者是谁？历史地位是什么？

**答案：** 宋应星著《天工开物》，被誉为“中国 17 世纪的工艺百科全书”。

**易错分析：** 不要把《天工开物》写成医学书。它总结农业和手工业生产技术。

### 卡28

**配图：** ![](第三单元-明清时期/28-card.png)

**提问：** 明清小说中，《红楼梦》的作者和文学地位是什么？

**答案：** 《红楼梦》前 80 回作者是曹雪芹，是清代小说艺术成就最高、影响最深远的作品之一。

**易错分析：** 不要把《红楼梦》写成明代小说。它属于清代小说代表。

### 卡29

**配图：** ![](第三单元-明清时期/29-card.png)

**提问：** 明清戏曲中，昆曲和京剧的发展有什么代表性意义？

**答案：** 昆曲在明清时期发展成熟，京剧在清代中后期逐渐形成，体现传统戏曲艺术的发展。

**易错分析：** 不要把京剧形成时间写得过早。京剧形成于清代中后期。

### 卡30

**配图：** ![](第三单元-明清时期/30-card.png)

**提问：** 七年级下册第三单元的时代特征可以怎样概括？

**答案：** 明清时期至鸦片战争前的时代特征是统一多民族封建国家的巩固与发展，同时君主专制强化、对外逐渐保守。

**易错分析：** 不要只说“封建社会衰落”。本单元既有国家巩固发展，也有专制强化和闭关带来的隐患。
