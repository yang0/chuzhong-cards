# 第五单元 新形势下坚持和发展中国特色社会主义

- 科目：社会
- 年级：八年级下册
- 教材：统编版2026春
- 卡片数：3



## 1. H8B-05-001

- 章节：科学发展与社会建设
- 提问：科学发展观强调怎样的发展？
- 答案：强调以人为本，全面、协调、可持续发展，统筹经济社会、城乡区域、人与自然等关系。
- 易错：发展不能只看GDP增长，还要看人的发展、公平和生态。
- 记忆钩子：以人为本，全面协调可持续。
- 考点：理念与材料匹配题
- 提示词：prompts/H8B-05-001.md



## 2. H8B-05-002

- 章节：民族团结与祖国统一
- 提问：“一国两制”的基本含义和实践是什么？
- 答案：在一个中国前提下，国家主体坚持社会主义制度，香港、澳门保持原有资本主义制度和生活方式长期不变，推动祖国和平统一。
- 易错：“两制”的前提和基础是“一国”。
- 记忆钩子：一国是前提，两制是安排。
- 考点：制度辨析题
- 提示词：prompts/H8B-05-002.md



## 3. H8B-05-003

- 章节：国防外交与科技
- 提问：新中国综合国力提升体现在哪些方面？
- 答案：国防和军队现代化推进，独立自主和平外交拓展，航天、信息、高铁等科技成就显著，国际地位和影响力提高。
- 易错：综合国力不是单一经济指标，而是多领域能力的综合。
- 记忆钩子：经济、国防、科技、外交共同支撑国力。
- 考点：成就归类题
- 提示词：prompts/H8B-05-003.md
