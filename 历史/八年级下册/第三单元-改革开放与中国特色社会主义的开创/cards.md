# 第三单元 改革开放与中国特色社会主义的开创

- 科目：社会
- 年级：八年级下册
- 教材：统编版2026春
- 卡片数：3



## 1. H8B-03-001

- 章节：中共十一届三中全会
- 提问：中共十一届三中全会为什么是伟大历史转折？
- 答案：会议作出把党和国家工作中心转移到经济建设上、实行改革开放的历史性决策，开启社会主义现代化建设新时期。
- 易错：会议召开于1978年，不是改革开放全部完成的标志。
- 记忆钩子：工作中心转向经济建设，改革开放由此开启。
- 考点：意义必考题
- 提示词：prompts/H8B-03-001.md



## 2. H8B-03-002

- 章节：农村改革
- 提问：家庭联产承包责任制为什么能调动农民积极性？
- 答案：在坚持土地集体所有的前提下，把经营自主权落实到农户，实行包干到户、自负盈亏，使劳动成果与投入直接联系。
- 易错：家庭联产承包责任制没有改变土地集体所有制性质。
- 记忆钩子：所有权归集体，经营权到农户。
- 考点：制度辨析题
- 提示词：prompts/H8B-03-002.md



## 3. H8B-03-003

- 章节：对外开放
- 提问：我国对外开放格局怎样形成？
- 答案：从设立经济特区起步，逐步开放沿海城市、沿海经济开放区，再向内地推进，形成全方位、多层次、宽领域的开放格局。
- 易错：对外开放不是只开放沿海，也不等于照搬外国制度。
- 记忆钩子：特区—沿海城市—开放区—内地。
- 考点：地图与时序题
- 提示词：prompts/H8B-03-003.md
