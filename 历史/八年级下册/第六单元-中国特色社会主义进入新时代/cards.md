# 第六单元 中国特色社会主义进入新时代

- 科目：社会
- 年级：八年级下册
- 教材：统编版2026春
- 卡片数：3



## 1. H8B-06-001

- 章节：新时代的历史方位
- 提问：中国特色社会主义进入新时代意味着什么？
- 答案：我国发展站在新的历史方位，社会主要矛盾发生转化，党和国家事业取得历史性成就、发生历史性变革。
- 易错：进入新时代不意味着我国发展中国家的基本国情已经根本改变。
- 记忆钩子：新方位、新矛盾、新任务。
- 考点：概念与国情辨析题
- 提示词：prompts/H8B-06-001.md



## 2. H8B-06-002

- 章节：新发展理念
- 提问：新发展理念包括哪些内容？
- 答案：创新、协调、绿色、开放、共享，五者相互贯通，共同回答新时代实现什么样的发展、怎样发展。
- 易错：五大理念是整体，不能把绿色发展等同于全部发展理念。
- 记忆钩子：创新协调绿色开放共享。
- 考点：材料对应题
- 提示词：prompts/H8B-06-002.md



## 3. H8B-06-003

- 章节：脱贫攻坚与全面小康
- 提问：脱贫攻坚胜利有什么历史意义？
- 答案：历史性解决绝对贫困问题，为全面建成小康社会作出关键贡献，体现社会主义制度优势并为全球减贫贡献中国经验。
- 易错：消除绝对贫困不等于不存在发展差距，相对贫困和共同富裕仍需持续推进。
- 记忆钩子：解决绝对贫困，迈向共同富裕。
- 考点：成就与意义题
- 提示词：prompts/H8B-06-003.md
