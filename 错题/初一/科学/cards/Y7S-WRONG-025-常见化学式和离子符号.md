# Y7S-WRONG-025 常见化学式和离子符号
原图：`Y7S-WRONG-018.jpg`

## 题目

写出常见物质的名称或化学式，并补全常见离子符号和所带电荷数。

## 整理

常见写法：

| 名称 | 化学式 |
|---|---|
| 氧气 | $O_2$ |
| 氖气 | $Ne$ |
| 铜 | $Cu$ |
| 五氧化二氮 | $N_2O_5$ |
| 石墨 | $C$ |
| 铁 | $Fe$ |
| 氦气 | $He$ |
| 氯化氢 | $HCl$ |
| 氯化铝 | $AlCl_3$ |
| 四氧化三铁 | $Fe_3O_4$ |

常见离子：

| 离子名称 | 离子符号 | 电荷 |
|---|---|---|
| 氢氧根离子 | $OH^-$ | -1 |
| 硫酸根离子 | $SO_4^{2-}$ | -2 |
| 硝酸根离子 | $NO_3^-$ | -1 |
| 碳酸根离子 | $CO_3^{2-}$ | -2 |
| 碳酸氢根离子 | $HCO_3^-$ | -1 |
| 铵根离子 | $NH_4^+$ | +1 |

易错点：氦气、氖气等稀有气体通常直接写元素符号，不写成 $He_2$、$Ne_2$。
