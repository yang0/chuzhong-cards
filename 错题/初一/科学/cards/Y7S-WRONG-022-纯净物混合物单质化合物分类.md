# Y7S-WRONG-022 纯净物混合物单质化合物分类
原图：`Y7S-WRONG-016.jpg`

## 题目

判断下列物质类别：

①冰水混合物；②液氮；③二氧化碳；④铁丝；⑤氮气；⑥氧化铜；⑦洁净的空气。

分别填出混合物、化合物、单质、金属单质、非金属单质、稀有气体单质。

## 整理

| 类别 | 序号 |
|---|---|
| 混合物 | ⑦ |
| 化合物 | ①③⑥ |
| 单质 | ②④⑤ |
| 金属单质 | ④ |
| 非金属单质 | ②⑤ |
| 稀有气体单质 | 无 |

易错点：

- 冰水混合物虽然叫“混合物”，但冰和水都是 $H_2O$，属于纯净物、化合物。
- 洁净空气仍含多种气体，属于混合物。
- 液氮和氮气都是 $N_2$，属于非金属单质，不是稀有气体。
