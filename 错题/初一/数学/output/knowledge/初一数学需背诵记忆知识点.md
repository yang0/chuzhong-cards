# 初一数学需背诵和记忆知识点整理

依据：人教版七年级下册数学主线，结合当前错题中暴露出的薄弱点整理。

说明：本文件先解决“知识点整理”，暂不考虑 Anki 导入格式。后续可把每条“记忆点”拆成正反面卡片。

## 一、相交线与平行线

### 1. 对顶角

记忆点：两条直线相交形成的对顶角相等。

常用语言：

```text
因为 ∠1 与 ∠3 是对顶角，
所以 ∠1 = ∠3。
```

易错提醒：对顶角必须是两条直线相交形成的“相对的角”，不是看起来位置相对就一定是对顶角。

### 2. 邻补角

记忆点：有一条公共边，另一边互为反向延长线的两个角叫邻补角；邻补角和为 $180^\circ$。

常用语言：

```text
因为 ∠1 与 ∠2 互为邻补角，
所以 ∠1 + ∠2 = 180^\circ。
```

易错提醒：邻补角一定互补，但互补的两个角不一定是邻补角。

### 3. 垂直

记忆点：两条直线相交成直角，就说这两条直线互相垂直。

符号：

```text
a ⟂ b
```

常用结论：

```text
同一平面内，垂直于同一条直线的两条直线互相平行。
```

错题触发：`Y7M-WRONG-035` 中由 $EF\perp AB$，$CD\perp AB$ 推出 $EF\parallel CD$。

### 4. 点到直线的距离

记忆点：从直线外一点到这条直线的垂线段的长度，叫做点到直线的距离。

易错提醒：距离是“垂线段长度”，不是随便连一条线段的长度。

### 5. 三线八角

记忆点：两条直线被第三条直线所截，会形成同位角、内错角、同旁内角。

需要会认：

- 同位角：位置相同，像“同一方向的角”。
- 内错角：在两条直线内部，分居截线两侧。
- 同旁内角：在两条直线内部，位于截线同旁。

易错提醒：先确定“被截的两条线”和“截线”，再判断角的位置关系。不要只凭图形方向猜。

错题触发：`Y7M-WRONG-002` 反射光线中识别同位角、同旁内角、内错角。

### 6. 平行线判定

记忆点：证明两条直线平行，常用三句话。

```text
同位角相等，两直线平行。
内错角相等，两直线平行。
同旁内角互补，两直线平行。
```

常用语言：

```text
因为 ∠1 = ∠2，
且 ∠1 与 ∠2 是同位角，
所以 a ∥ b。
```

易错提醒：判定平行线时，结论是“两直线平行”；不要把性质和判定倒着写。

错题触发：`Y7M-WRONG-003`、`Y7M-WRONG-005`、`Y7M-WRONG-033`。

### 7. 平行线性质

记忆点：已知两条直线平行，常用三句话。

```text
两直线平行，同位角相等。
两直线平行，内错角相等。
两直线平行，同旁内角互补。
```

易错提醒：性质的前提是“已经知道平行”，判定的目标是“证明平行”。

错题触发：`Y7M-WRONG-031`、`Y7M-WRONG-034`、`Y7M-WRONG-035`。

### 8. 平行公理及推论

记忆点：

```text
经过直线外一点，有且只有一条直线与这条直线平行。
```

推论：

```text
如果两条直线都与第三条直线平行，那么这两条直线也互相平行。
```

错题触发：`Y7M-WRONG-032` 中 $JP\parallel AH$，$JK\parallel AH$，且都过点 $J$，所以 $P,J,K$ 共线。

### 9. 平移

记忆点：平移只改变图形的位置，不改变图形的形状和大小。

平移性质：

- 对应线段平行且相等。
- 对应角相等。
- 对应点所连线段平行且相等。

易错提醒：平移没有旋转，方向不变；不要把翻折、旋转和平移混淆。

## 二、实数

### 10. 平方根

记忆点：如果 $x^2=a$，那么 $x$ 叫做 $a$ 的平方根。

结论：

```text
正数有两个平方根，它们互为相反数。
0 的平方根是 0。
负数没有平方根。
```

易错提醒：$\sqrt a$ 表示 $a$ 的算术平方根，结果非负。

错题关联：`Y7M-WRONG-038` 中由 $(x^n)^2=25$ 得 $x^n=\pm5$，不能只写 $5$。

### 11. 算术平方根

记忆点：正数 $a$ 的正的平方根叫做 $a$ 的算术平方根，记作 $\sqrt a$。

常用：

```text
\sqrt{25}=5，而不是 ±5。
```

易错提醒：解方程 $x^2=25$ 时答案是 $x=\pm5$；计算 $\sqrt{25}$ 时答案是 $5$。

### 12. 立方根

记忆点：如果 $x^3=a$，那么 $x$ 叫做 $a$ 的立方根，记作 $\sqrt[3]{a}$。

结论：

```text
正数有正的立方根。
0 的立方根是 0。
负数有负的立方根。
```

易错提醒：负数没有平方根，但有立方根。

### 13. 无理数

记忆点：无限不循环小数叫无理数。

常见无理数：

```text
\sqrt2,\ \sqrt3,\ \pi
```

易错提醒：无限小数不一定是无理数；无限循环小数是有理数。

### 14. 实数分类

记忆点：

```text
实数 = 有理数 + 无理数
```

有理数包括整数和分数，也包括有限小数和无限循环小数。

## 三、平面直角坐标系

### 15. 有序数对

记忆点：有顺序的两个数 $a,b$ 组成的数对叫有序数对，记作 $(a,b)$。

易错提醒：$(a,b)$ 和 $(b,a)$ 通常表示不同位置。

### 16. 点的坐标

记忆点：点 $P(x,y)$ 中，$x$ 是横坐标，$y$ 是纵坐标。

读图顺序：

```text
先横后纵，先 x 后 y。
```

易错提醒：不要把横纵坐标写反。

### 17. 象限符号

记忆点：

| 象限 | 横坐标 | 纵坐标 |
|---|---|---|
| 第一象限 | $+$ | $+$ |
| 第二象限 | $-$ | $+$ |
| 第三象限 | $-$ | $-$ |
| 第四象限 | $+$ | $-$ |

坐标轴上的点不属于任何象限。

### 18. 坐标轴上的点

记忆点：

```text
x 轴上的点：纵坐标 y = 0。
y 轴上的点：横坐标 x = 0。
```

易错提醒：在 $x$ 轴上不是 $x=0$，而是 $y=0$。

### 19. 点的平移与坐标变化

记忆点：

```text
向右平移 a 个单位：x 加 a。
向左平移 a 个单位：x 减 a。
向上平移 b 个单位：y 加 b。
向下平移 b 个单位：y 减 b。
```

易错提醒：左右改横坐标，上下改纵坐标。

## 四、二元一次方程组

### 20. 二元一次方程

记忆点：含有两个未知数，并且含未知数的项的次数都是 $1$ 的方程，叫二元一次方程。

例：

```text
2x+3y=5
```

非例：

```text
xy=5，x^2+y=3
```

### 21. 二元一次方程组的解

记忆点：使方程组中每一个方程都成立的一组未知数的值，叫做这个方程组的解。

易错提醒：必须同时满足所有方程，只满足一个方程不够。

错题触发：`Y7M-WRONG-065` 中正确解要代回原方程组，看错哪个系数就只能影响对应方程。

### 22. 代入消元法

记忆点：从一个方程中用一个未知数表示另一个未知数，再代入另一个方程，消去一个未知数。

适用场景：某个未知数的系数是 $1$ 或 $-1$ 时常用。

### 23. 加减消元法

记忆点：通过两方程相加或相减，消去一个未知数。

关键步骤：

```text
先让某个未知数的系数相同或互为相反数。
```

易错提醒：方程两边同时乘同一个数，不能只乘一边或只乘某几项。

### 24. 实际问题列方程组

记忆点：

```text
设未知数 -> 找两个等量关系 -> 列方程组 -> 解方程组 -> 检验并作答
```

常见等量关系：

- 总价 = 单价 × 数量
- 工作量 = 工作效率 × 工作时间
- 路程 = 速度 × 时间

错题触发：`Y7M-WRONG-066` 装修费用与工作效率问题。

## 五、不等式与不等式组

### 25. 不等式的解

记忆点：使不等式成立的未知数的值，叫做不等式的解。

### 26. 不等式的解集

记忆点：一个含未知数的不等式的所有解组成这个不等式的解集。

易错提醒：解集是一类数，不是单个数。

### 27. 不等式基本性质 1

记忆点：

```text
不等式两边同时加上或减去同一个数，不等号方向不变。
```

### 28. 不等式基本性质 2

记忆点：

```text
不等式两边同时乘以或除以同一个正数，不等号方向不变。
```

### 29. 不等式基本性质 3

记忆点：

```text
不等式两边同时乘以或除以同一个负数，不等号方向改变。
```

易错提醒：乘除负数必须变号，这是不等式最容易错的点。

### 30. 一元一次不等式解法

记忆点：

```text
去分母 -> 去括号 -> 移项 -> 合并同类项 -> 系数化为 1
```

易错提醒：去分母时每一项都要乘；系数化为 $1$ 时如果除以负数，不等号要变向。

### 31. 一元一次不等式组

记忆点：几个一元一次不等式合在一起，组成一元一次不等式组。

解法：分别求每个不等式的解集，再求公共部分。

口诀：

```text
同大取大，同小取小；
大小小大中间找；
大大小小找不到。
```

## 六、数据的收集、整理与描述

### 32. 全面调查

记忆点：对全体对象进行调查，叫全面调查。

适用：总体数量较小，或调查结果要求非常准确。

### 33. 抽样调查

记忆点：从总体中抽取一部分对象进行调查，再用样本情况估计总体情况。

适用：总体数量大，或全面调查不方便、不经济。

错题触发：`Y7M-WRONG-060` 本市居民电费不能只调查某校同学家庭。

### 34. 总体、个体、样本、样本容量

记忆点：

- 总体：所要考察对象的全体。
- 个体：总体中的每一个考察对象。
- 样本：从总体中抽取的一部分个体。
- 样本容量：样本中个体的数目。

易错提醒：样本容量是数量，不带单位。

### 35. 样本代表性

记忆点：样本要能代表总体，抽取时应尽量随机、覆盖不同类型。

易错提醒：样本不是越多越好，来源单一的大样本也可能不合理。

错题触发：只调查“某校所有同学家庭”样本来源单一，不代表全市居民。

### 36. 频数与频率

记忆点：

```text
频数：某类数据出现的次数。
频率：频数 ÷ 数据总数。
```

### 37. 统计图选择

记忆点：

- 条形统计图：比较数量多少。
- 折线统计图：表示变化趋势。
- 扇形统计图：表示部分占总体的百分比。
- 频数分布直方图：描述一组数据的分布情况。

## 七、错题中出现的补充记忆点

以下内容不完全属于人教版七下主线，但在当前错题中频繁出现，应作为“错题补充记忆”。

### 38. 同底数幂乘法

记忆点：

```text
a^m\cdot a^n=a^{m+n}
```

条件：底数相同，指数相加。

### 39. 同底数幂除法

记忆点：

```text
a^m\div a^n=a^{m-n}\quad(a\ne0)
```

易错提醒：底数不变，指数相减；注意负号底数。

错题触发：`Y7M-WRONG-044`。

### 40. 幂的乘方

记忆点：

```text
(a^m)^n=a^{mn}
```

易错提醒：指数相乘，不是相加。

### 41. 积的乘方

记忆点：

```text
(ab)^n=a^n b^n
```

错题触发：`Y7M-WRONG-038` 中 $(xy)^n=x^n y^n$。

### 42. 负数幂的奇偶性

记忆点：

```text
(-a)^{偶数}=a^{偶数}
(-a)^{奇数}=-a^{奇数}
```

错题触发：`Y7M-WRONG-044` 中 $2n-1$ 是奇数，$2n$ 是偶数。

### 43. 平方差公式

记忆点：

```text
(a+b)(a-b)=a^2-b^2
```

常见变形：

```text
(a-1)(a+1)=a^2-1
```

错题触发：`Y7M-WRONG-041`、`Y7M-WRONG-042`。

### 44. 完全平方公式

记忆点：

```text
(a+b)^2=a^2+2ab+b^2
(a-b)^2=a^2-2ab+b^2
```

错题触发：`Y7M-WRONG-071`。

### 45. 多项式乘多项式

记忆点：

```text
(a+b)(c+d)=ac+ad+bc+bd
```

易错提醒：每一项都要乘到，减号要带着走。

错题触发：`Y7M-WRONG-039`。

### 46. 整体代入

记忆点：题目给了某个整体的值，先把目标式化成这个整体，再代入。

例：

```text
已知 x^2-6x=20，
就把目标式化成 x^2-6x。
```

错题触发：`Y7M-WRONG-039-第5题`、`Y7M-WRONG-070`。

### 47. 因式分解：提公因式

记忆点：多项式各项有公共因式时，先提公因式。

```text
ma+mb=m(a+b)
```

错题触发：`Y7M-WRONG-050`、`Y7M-WRONG-051`。

### 48. 因式分解中的相反式

记忆点：

```text
y-x=-(x-y)
```

平方时：

```text
(y-x)^2=(x-y)^2
```

立方时：

```text
(y-x)^3=-(x-y)^3
```

错题触发：`Y7M-WRONG-050`。

### 49. 分式定义

记忆点：分母中含有字母的式子叫分式。

易错提醒：只看分母，分子含字母不一定是分式。

错题触发：`Y7M-WRONG-052`。

### 50. 分式有意义

记忆点：

```text
分母 ≠ 0
```

错题触发：`Y7M-WRONG-052`、`Y7M-WRONG-078`。

### 51. 分式值为 0

记忆点：

```text
分子 = 0 且 分母 ≠ 0
```

易错提醒：不能只令分子为 $0$，还要检查分母。

错题触发：`Y7M-WRONG-052`。

### 52. 分式除法的条件

记忆点：除以一个分式时，除式不能为 $0$。

例如：

```text
\frac{x-1}{x-2}\div\frac{x-3}{x-4}
```

需要：

```text
x-2≠0，x-4≠0，且 x-3≠0。
```

错题触发：`Y7M-WRONG-078`。

### 53. 分式基本性质

记忆点：分式的分子、分母同时乘以或除以同一个不为 $0$ 的整式，分式的值不变。

```text
\frac ab=\frac{am}{bm}\quad(b\ne0,m\ne0)
```

### 54. 分式通分

记忆点：异分母分式相加减，先通分，再加减分子。

易错提醒：通分后分母不变，只加减分子。

错题触发：`Y7M-WRONG-024`。

### 55. 分式中的相反式

记忆点：

```text
2-x=-(x-2)
```

所以：

```text
\frac{1}{2-x}=-\frac{1}{x-2}
```

错题触发：`Y7M-WRONG-080`。

### 56. 平均速度

记忆点：

```text
平均速度 = 总路程 ÷ 总时间
```

易错提醒：两段路程相同、速度分别为 $a,b$ 时，平均速度不是 $\dfrac{a+b}{2}$，而是

```text
\frac{2ab}{a+b}
```

错题触发：`Y7M-WRONG-081`。

### 57. 百分数增长

记忆点：增长 $m\%$ 后变为原来的

```text
1+\frac{m}{100}
```

连续增长两个月：

```text
a\left(1+\frac{m}{100}\right)^2
```

错题触发：`Y7M-WRONG-043`。

### 58. 数位表示

记忆点：三位数 $\overline{abc}$ 表示

```text
100a+10b+c
```

错题触发：`Y7M-WRONG-046`。

### 59. 被 9 整除的余数规律

记忆点：一个整数除以 $9$ 的余数，等于它各位数字和除以 $9$ 的余数。

理由：

```text
100a+10b+c=99a+9b+(a+b+c)
```

错题触发：`Y7M-WRONG-046`。

### 60. 等比求和雏形

记忆点：

```text
(x^n-1)\div(x-1)=x^{n-1}+x^{n-2}+\cdots+x+1
```

常用于：

```text
1+2+2^2+\cdots+2^n
```

错题触发：`Y7M-WRONG-019`。

## 八、优先背诵顺序

第一优先级：错题反复触发，必须先背熟。

1. 平行线判定与性质。
2. 三线八角识别。
3. 垂直于同一条直线的两直线平行。
4. 平行公理：过直线外一点有且只有一条平行线。
5. 分式有意义、分式值为 $0$、分式除法条件。
6. 平方差公式、完全平方公式。
7. 同底数幂乘除、幂的乘方、积的乘方。
8. 整体代入。

第二优先级：人教版七下考试基础。

1. 平方根、算术平方根、立方根。
2. 实数分类。
3. 坐标象限与点的平移。
4. 二元一次方程组解法。
5. 不等式性质，尤其乘除负数变号。
6. 抽样调查中的代表性。

第三优先级：应用题常用模型。

1. 工作量 = 工作效率 × 工作时间。
2. 总价 = 单价 × 数量。
3. 路程 = 速度 × 时间。
4. 平均速度 = 总路程 ÷ 总时间。
5. 百分数增长模型。
