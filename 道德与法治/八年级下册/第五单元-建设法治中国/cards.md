# 第五单元 建设法治中国

- 科目：社会
- 年级：八年级下册
- 教材：统编版2026春
- 卡片数：3



## 1. MOR8B-05-001

- 章节：全面依法治国
- 提问：全面依法治国为什么是国家治理的一场深刻革命？
- 答案：法治为国家治理提供稳定规则和权利保障，规范公共权力、维护公平正义、促进发展，是中国式现代化的重要保障。
- 易错：依法治国不是只惩罚违法者，还包括立法、执法、司法、守法全过程。
- 记忆钩子：法治固根本、稳预期、利长远。
- 考点：意义材料题
- 提示词：prompts/MOR8B-05-001.md



## 2. MOR8B-05-002

- 章节：法治与德治
- 提问：法治和德治是什么关系？
- 答案：国家和社会治理需要法律与道德共同发挥作用，以法治承载道德理念、以道德滋养法治精神，二者相辅相成。
- 易错：法治和德治不能相互替代，也不是非此即彼。
- 记忆钩子：法安天下，德润人心。
- 考点：观点辨析题
- 提示词：prompts/MOR8B-05-002.md



## 3. MOR8B-05-003

- 章节：青少年与法治
- 提问：青少年怎样为法治中国建设作贡献？
- 答案：增强尊法学法守法用法意识，依法行使权利、履行义务，敢于并善于同违法行为斗争，参与法治实践。
- 易错：见义勇为也要讲策略，未成年人应优先保护自己并及时求助。
- 记忆钩子：尊法于心，守法于行，维权有方。
- 考点：实践行动题
- 提示词：prompts/MOR8B-05-003.md
