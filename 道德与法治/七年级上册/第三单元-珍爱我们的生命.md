# 第三单元 珍爱我们的生命

## 基本信息

- 科目：道德与法治
- 年级：七年级上册
- 教材：统编版 / 人教版《道德与法治》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元包括生命可贵、守护生命安全、保持身心健康。学习主线是：认识生命的特点和价值，增强安全意识，提高自我保护能力，养成健康生活方式。

## Anki 记忆卡

### 卡1



**配图：** ![](第三单元-珍爱我们的生命/01-card.png)
**提问：** 生命有哪些特点？

**答案：** 生命是来之不易的、独特的、不可逆的、短暂的，也是有接续的。

**易错分析：** 不要只答“生命很重要”，要能说出生命的具体特点。

**记忆钩子：** 来之不易、独特、不可逆、短暂、接续。

### 卡2



**配图：** ![](第三单元-珍爱我们的生命/02-card.png)
**提问：** 为什么要敬畏生命？

**答案：** 生命至上，每个人的生命都宝贵；敬畏生命要求我们珍爱自己的生命，也关怀他人的生命。

**易错分析：** 不要只强调自己的生命，敬畏生命也包括尊重和关怀他人生命。

**记忆钩子：** 珍爱自己，也关怀别人。

### 卡3



**配图：** ![](第三单元-珍爱我们的生命/03-card.png)
**提问：** 怎样守护生命？

**答案：** 要关注身体健康，养护精神，增强安全意识，掌握自救自护方法，养成健康生活方式。

**易错分析：** 不要把守护生命只理解成不生病，还包括安全、心理和精神生活。

**记忆钩子：** 身体、精神、安全三手抓。

### 卡4



**配图：** ![](第三单元-珍爱我们的生命/04-card.png)
**提问：** 为什么要增强安全意识？

**答案：** 安全是健康成长的重要保障，增强安全意识有助于预防危险、减少伤害、保护生命。

**易错分析：** 不要等危险发生后才重视安全，预防更重要。

**记忆钩子：** 安全意识是生命保护伞。

### 卡5



**配图：** ![](第三单元-珍爱我们的生命/05-card.png)
**提问：** 遇到突发危险时应怎样做？

**答案：** 要保持冷静，迅速判断情况，采取正确自救自护方法，必要时及时求助或报警。

**易错分析：** 不要慌乱冒险，也不要盲目围观。

**记忆钩子：** 冷静、判断、自护、求助。

### 卡6



**配图：** ![](第三单元-珍爱我们的生命/06-card.png)
**提问：** 怎样爱护身体？

**答案：** 要养成健康生活方式，合理饮食，规律作息，适量运动，远离不良诱惑。

**易错分析：** 不要把爱护身体只理解成生病看医生，平时习惯更关键。

**记忆钩子：** 吃好、睡好、动起来。

### 卡7



**配图：** ![](第三单元-珍爱我们的生命/07-card.png)
**提问：** 为什么要滋养心灵？

**答案：** 健康不仅包括身体健康，也包括心理和精神健康；滋养心灵有助于形成积极生活态度。

**易错分析：** 不要只重视身体，不重视心理和精神需求。

**记忆钩子：** 心灵也需要营养。

### 卡8



**配图：** ![](第三单元-珍爱我们的生命/08-card.png)
**提问：** 怎样保持心理健康？

**答案：** 要正确认识情绪，学会调节压力，主动沟通求助，培养兴趣爱好，保持积极心态。

**易错分析：** 不要认为有烦恼就是软弱，主动求助是积极表现。

**记忆钩子：** 有压力，能调节，会求助。

### 卡9



**配图：** ![](第三单元-珍爱我们的生命/09-card.png)
**提问：** 珍爱生命和追求成长之间有什么关系？

**答案：** 珍爱生命不是逃避困难，而是在保护生命的基础上积极生活、勇敢面对挑战、实现生命价值。

**易错分析：** 不要把珍爱生命理解成害怕一切风险，关键是理性面对风险。

**记忆钩子：** 珍爱生命，也要活出精彩。
