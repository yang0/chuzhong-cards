# 第二单元 成长的时空

## 基本信息

- 科目：道德与法治
- 年级：七年级上册
- 教材：统编版 / 人教版《道德与法治》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元包括幸福和睦的家庭、和谐的师生关系、友谊之树常青、在集体中成长。学习主线是：在家庭、学校、朋友和集体关系中学习做人、学会相处、共同成长。

## Anki 记忆卡

### 卡1



**配图：** ![](第二单元-成长的时空/01-card.png)
**提问：** 家庭的含义是什么？

**答案：** 家庭是由婚姻关系、血缘关系或收养关系结合成的亲属生活组织。

**易错分析：** 不要把家只理解成一所房子，家更是亲属生活共同体。

**记忆钩子：** 家不只是房子，还有亲情和关系。

### 卡2



**配图：** ![](第二单元-成长的时空/02-card.png)
**提问：** 家对我们的成长有什么意义？

**答案：** 家是生命的居所、心灵的港湾，是我们的第一所学校和修身之所，也是社会的基本细胞。

**易错分析：** 不要只答“家给我们生活条件”，道法强调情感、教育和社会意义。

**记忆钩子：** 家是港湾，也是第一所学校。

### 卡3



**配图：** ![](第二单元-成长的时空/03-card.png)
**提问：** 怎样建设美好家庭？

**答案：** 家庭成员要相互关爱、彼此尊重、主动沟通、分担家务，共同传承优良家风。

**易错分析：** 不要认为建设家庭只是父母的事，子女也有责任。

**记忆钩子：** 爱、尊重、沟通、分担。

### 卡4



**配图：** ![](第二单元-成长的时空/04-card.png)
**提问：** 为什么要尊重老师？

**答案：** 老师承担教书育人的使命，是我们学习的指导者和成长的引路人。

**易错分析：** 不要只从“老师有权威”角度理解尊师，关键在于老师的教育职责和付出。

**记忆钩子：** 老师是成长引路人。

### 卡5



**配图：** ![](第二单元-成长的时空/05-card.png)
**提问：** 怎样建立和谐的师生关系？

**答案：** 要尊重老师、主动交流、正确对待老师的表扬和批评，理解老师的良苦用心。

**易错分析：** 不要把老师批评都看成针对自己，批评也可能是帮助成长。

**记忆钩子：** 尊重 + 沟通 = 师生和谐。

### 卡6



**配图：** ![](第二单元-成长的时空/06-card.png)
**提问：** 友谊有什么重要作用？

**答案：** 友谊让我们体悟生命的美好，获得理解、支持和力量，促进我们共同成长。

**易错分析：** 不要把朋友只理解成一起玩的伙伴，真正的友谊能促进成长。

**记忆钩子：** 好朋友是成长加油站。

### 卡7



**配图：** ![](第二单元-成长的时空/07-card.png)
**提问：** 怎样呵护友谊？

**答案：** 要用心关怀对方，学会尊重、正确处理冲突，承受并正确对待交友中受到的伤害。

**易错分析：** 不要认为好朋友之间永远不会有矛盾，关键是处理矛盾的方法。

**记忆钩子：** 友谊要浇水，也要修枝。

### 卡8



**配图：** ![](第二单元-成长的时空/08-card.png)
**提问：** 交友要坚持哪些原则？

**答案：** 交友要真诚友善、讲原则、明辨是非，不能不加分辨地为朋友做任何事。

**易错分析：** 不要把“讲义气”理解成帮朋友隐瞒错误或做错事。

**记忆钩子：** 友谊有温度，也有原则。

### 卡9



**配图：** ![](第二单元-成长的时空/09-card.png)
**提问：** 集体生活对个人成长有什么作用？

**答案：** 集体生活能涵养品格、发展个性，帮助我们学会合作、承担责任、共同进步。

**易错分析：** 不要把集体只看成约束个人的地方，集体也能成就个人。

**记忆钩子：** 集体是成长练习场。

### 卡10



**配图：** ![](第二单元-成长的时空/10-card.png)
**提问：** 怎样共建美好集体？

**答案：** 要有共同愿景，营造良好氛围，分工协作，各尽其能，主动承担集体责任。

**易错分析：** 不要认为美好集体只靠班干部或老师建设，每个成员都有责任。

**记忆钩子：** 美好集体靠大家共建。

### 卡11



**配图：** ![](第二单元-成长的时空/11-card.png)
**提问：** 个人利益和集体利益发生冲突时怎么办？

**答案：** 应坚持集体主义，把集体利益放在个人利益之上，同时保护个人正当利益。

**易错分析：** 不要理解为完全不要个人利益。道法强调个人正当利益也应受到尊重。

**记忆钩子：** 集体优先，正当个人利益也要尊重。

### 卡12



**配图：** ![](第二单元-成长的时空/12-card.png)
**提问：** 在成长关系中最重要的共同主题是什么？

**答案：** 在家庭、师生、友谊和集体关系中，我们都要学会尊重、沟通、责任和共同成长。

**易错分析：** 不要把各课割裂背诵，它们共同指向“在关系中成长”。

**记忆钩子：** 尊重、沟通、责任、成长。
