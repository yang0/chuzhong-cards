# 第一单元 少年有梦

## 基本信息

- 科目：道德与法治
- 年级：七年级上册
- 教材：统编版 / 人教版《道德与法治》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元包括第一课“开启初中生活”、第二课“正确认识自我”、第三课“梦想始于当下”。学习主线是：适应初中生活，认识自己、完善自己，把少年梦想落实到学习和行动中。

## Anki 记忆卡

### 卡1



**配图：** ![](第一单元-少年有梦/01-card.png)
**提问：** 初中生活“新”在哪里？

**答案：** 我们有了新的身份，进入新的环境，遇到新的老师和同学；初中阶段为发展提供新机会，也对成长提出新要求。

**易错分析：** 不要只答“换了学校”。“新”还包括机会、要求和人生阶段价值。

**记忆钩子：** 新身份、新环境、新机会、新要求。

### 卡2



**配图：** ![](第一单元-少年有梦/02-card.png)
**提问：** 初中阶段为我们发展提供了哪些新机会？

**答案：** 富有挑战的课程、丰富的社团活动、各种社会实践和新的集体生活，都为我们发展提供机会。

**易错分析：** 不要只写学习课程，社团、实践和集体生活也是机会。

**记忆钩子：** 课程、社团、实践、集体。

### 卡3



**配图：** ![](第一单元-少年有梦/03-card.png)
**提问：** 初中阶段对我们的成长提出了哪些新要求？

**答案：** 要越来越上进、踏实、包容、坚强，努力成为有理想、敢担当、能吃苦、肯奋斗的中学生。

**易错分析：** 不要只说“好好学习”，道法常要求从品格和行动多角度回答。

**记忆钩子：** 上进、踏实、包容、坚强。

### 卡4



**配图：** ![](第一单元-少年有梦/04-card.png)
**提问：** 为什么要规划初中生活？

**答案：** 合理规划能帮助我们明确目标、安排行动、减少盲目和无序，更好地度过初中阶段。

**易错分析：** 不要把规划理解成空想。规划要服务实际行动。

**记忆钩子：** 规划是成长路线图。

### 卡5



**配图：** ![](第一单元-少年有梦/05-card.png)
**提问：** 怎样做好初中生活规划？

**答案：** 要确定个人发展目标，制定行动方案，并持之以恒，把努力落实到每天的具体行动中。

**易错分析：** 不要只写目标，没有行动方案和坚持，规划难以实现。

**记忆钩子：** 目标、方案、坚持。

### 卡6



**配图：** ![](第一单元-少年有梦/06-card.png)
**提问：** 为什么要正确认识自己？

**答案：** 正确认识自己有助于增强自信、促进自我发展，促进与他人的交往，正确处理个人与社会的关系。

**易错分析：** 不要把认识自己只理解成知道优点，还要全面认识优点和不足。

**记忆钩子：** 认识自己，发展自己。

### 卡7



**配图：** ![](第一单元-少年有梦/07-card.png)
**提问：** 我们可以通过哪些途径认识自己？

**答案：** 可以通过自我观察和分析、他人评价、集体生活和社会实践等途径认识自己。

**易错分析：** 不要只听别人评价，也不能完全忽视别人评价。

**记忆钩子：** 自己看、别人看、实践看。

### 卡8



**配图：** ![](第一单元-少年有梦/08-card.png)
**提问：** 怎样做更好的自己？

**答案：** 要接纳和欣赏自己，扬长避短，主动改正缺点，不断激发潜能，在为他人和社会服务中成长。

**易错分析：** 不要把“接纳自己”理解成不改正缺点。

**记忆钩子：** 接纳、扬长、改缺、激潜。

### 卡9



**配图：** ![](第一单元-少年有梦/09-card.png)
**提问：** 少年为什么要有梦想？

**答案：** 梦想能激发生命热情和前进动力，帮助我们明确人生方向，把个人成长同祖国和时代联系起来。

**易错分析：** 不要把梦想等同于一时愿望，梦想需要方向和行动。

**记忆钩子：** 梦想是心里的发动机。

### 卡10



**配图：** ![](第一单元-少年有梦/10-card.png)
**提问：** 少年梦想应当具有什么特点？

**答案：** 少年梦想要与个人目标紧密相连，也要与时代脉搏和中国梦联系起来。

**易错分析：** 不要只写个人愿望，道法强调个人梦与中国梦相连。

**记忆钩子：** 小我梦想连着中国梦。

### 卡11



**配图：** ![](第一单元-少年有梦/11-card.png)
**提问：** 学习对实现梦想有什么作用？

**答案：** 学习能点亮生命、增长知识和能力，为实现梦想奠定基础。

**易错分析：** 不要把学习只理解成课堂听课，生活和实践中也能学习。

**记忆钩子：** 学习给梦想充电。

### 卡12



**配图：** ![](第一单元-少年有梦/12-card.png)
**提问：** 怎样把梦想变成现实？

**答案：** 要把梦想落实到每一天的学习和实践中，坚持努力，脚踏实地，勇于面对困难和挑战。

**易错分析：** 不要只停留在“有梦想”，关键是持续行动。

**记忆钩子：** 梦想 + 努力 = 更接近现实。
