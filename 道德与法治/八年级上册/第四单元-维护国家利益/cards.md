# 第四单元 维护国家利益

- 科目：社会
- 年级：八年级上册
- 教材：统编版/科学补充
- 卡片数：3



## 1. MOR8A-04-001

- 章节：国家利益
- 提问：国家利益与人民利益是什么关系？
- 答案：国家利益反映广大人民的共同需求，国家利益至上并不排斥人民利益；国家繁荣发展是人民利益实现的重要保障。
- 易错：不能把国家利益与个人正当利益简单对立。
- 记忆钩子：国家好，人民生活才更有保障。
- 考点：教材核心考点
- 提示词：prompts/MOR8A-04-001.md



## 2. MOR8A-04-002

- 章节：坚持国家利益至上
- 提问：怎样坚持国家利益至上？
- 答案：心怀爱国之情，树立和增强危机意识与防范意识；以国家利益为重，必要时牺牲个人利益，坚决维护国家利益。
- 易错：爱国不仅是情感表达，还要落实在守法和行动中。
- 记忆钩子：心系国家，行动守护。
- 考点：教材核心考点
- 提示词：prompts/MOR8A-04-002.md



## 3. MOR8A-04-003

- 章节：总体国家安全观
- 提问：为什么要维护国家安全？
- 答案：国家安全是国家生存与发展的重要保障，也是人民幸福安康的前提；人人都是国家安全的维护者。
- 易错：国家安全不只是军事安全，还包括网络、生态、粮食等领域。
- 记忆钩子：安全是发展的底座。
- 考点：教材核心考点
- 提示词：prompts/MOR8A-04-003.md
