# 第一单元 珍惜青春时光

## 基本信息

- 科目：道德与法治
- 年级：七年级下册
- 教材：统编版 / 人教版《道德与法治》（2025春版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元包括青春正当时、做情绪情感的主人等内容。学习主线是：认识青春期身心变化，学会自我保护和调节情绪情感，珍惜青春时光，用行动书写成长。

## Anki 记忆卡

### 卡1



**配图：** ![](第一单元-珍惜青春时光/01-card.png)
**提问：** 青春期身体变化主要表现在哪些方面？

**答案：** 青春期身体变化主要表现为身体外形变化、内部器官完善、性机能成熟等。

**易错分析：** 不要只把青春期变化理解成外貌变化，身体内部和心理也会变化。

**记忆钩子：** 外形、器官、性机能。

### 卡2



**配图：** ![](第一单元-珍惜青春时光/02-card.png)
**提问：** 怎样正确对待青春期的生理变化？

**答案：** 要正视并欣然接受青春期变化，尊重自己和同伴的生理变化，追求内在美与外在美统一。

**易错分析：** 不要嘲笑他人变化，也不要过度焦虑自己的变化。

**记忆钩子：** 接受自己，尊重他人。

### 卡3



**配图：** ![](第一单元-珍惜青春时光/03-card.png)
**提问：** 青春期心理矛盾应怎样处理？

**答案：** 可以参加集体活动，求助他人，培养兴趣爱好，学习自我调节，积极面对青春期心理矛盾。

**易错分析：** 不要把心理矛盾看成异常，也不要长期压在心里不处理。

**记忆钩子：** 活动、求助、兴趣、调节。

### 卡4



**配图：** ![](第一单元-珍惜青春时光/04-card.png)
**提问：** 青春期为什么要学会自我保护？

**答案：** 青春期身心快速发展，社会经验不足，学会自我保护有助于远离危险、维护身心健康。

**易错分析：** 不要认为危险离自己很远，自我保护是日常成长需要。

**记忆钩子：** 成长快，保护要跟上。

### 卡5



**配图：** ![](第一单元-珍惜青春时光/05-card.png)
**提问：** 怎样学会自我保护？

**答案：** 要增强安全意识和法治意识，谨慎交往，保护隐私，遇到侵害及时求助，学会依法维权。

**易错分析：** 不要把求助当成软弱，及时求助是保护自己的重要方式。

**记忆钩子：** 有边界，会求助，懂法律。

### 卡6



**配图：** ![](第一单元-珍惜青春时光/06-card.png)
**提问：** 情绪有什么作用？

**答案：** 情绪影响我们的观念和行动，可能激励我们克服困难，也可能让我们因冲动而犯错。

**易错分析：** 不要把情绪都看成坏东西，情绪有积极和消极影响。

**记忆钩子：** 情绪像方向盘，要会掌控。

### 卡7



**配图：** ![](第一单元-珍惜青春时光/07-card.png)
**提问：** 为什么要学会调节情绪？

**答案：** 学会调节情绪有助于保持积极心态，改善人际关系，帮助我们更好适应生活和学习。

**易错分析：** 调节情绪不是压抑情绪，而是合理表达和管理。

**记忆钩子：** 不是憋住，是管好。

### 卡8



**配图：** ![](第一单元-珍惜青春时光/08-card.png)
**提问：** 调节情绪常用的方法有哪些？

**答案：** 改变认知评价、转移注意、合理宣泄、放松训练等都是常用方法。

**易错分析：** 不要用伤害自己或他人的方式宣泄情绪。

**记忆钩子：** 改看法、换注意、会宣泄、能放松。

### 卡9



**配图：** ![](第一单元-珍惜青春时光/09-card.png)
**提问：** 情感和情绪有什么联系与区别？

**答案：** 情感与情绪紧密相关，情绪逐渐积累和发展可能形成比较稳定的情感；情绪短暂多变，情感相对稳定深刻。

**易错分析：** 不要把情绪和情感完全等同。

**记忆钩子：** 情绪短，情感深。

### 卡10



**配图：** ![](第一单元-珍惜青春时光/10-card.png)
**提问：** 怎样获得和传递美好情感？

**答案：** 可以通过阅读、交往、参与有意义活动等获得美好情感，并用行动关心他人、服务社会、传递正能量。

**易错分析：** 美好情感不是只靠等待别人给予，也需要主动创造和传递。

**记忆钩子：** 美好情感越传越亮。
