# 第二单元 焕发青春活力

## 基本信息

- 科目：道德与法治
- 年级：七年级下册
- 教材：统编版 / 人教版《道德与法治》（2025春版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元包括人贵自尊、自信给人力量、人生当自强等内容。学习主线是：在人格成长中培养自尊、自信、自强，让青春焕发积极向上的力量。

## Anki 记忆卡

### 卡1



**配图：** ![](第二单元-焕发青春活力/01-card.png)
**提问：** 什么是自尊？

**答案：** 自尊是尊重自己、爱护自己，并期望得到他人、集体和社会尊重的心理需要和品质。

**易错分析：** 不要把自尊理解成虚荣或面子，自尊强调人格尊严和自我尊重。

**记忆钩子：** 自尊不是爱面子，是守住人格。

### 卡2



**配图：** ![](第二单元-焕发青春活力/02-card.png)
**提问：** 为什么说人贵自尊？

**答案：** 自尊能使人珍惜自己的品格和名誉，激励人奋发向上，促进我们获得他人尊重。

**易错分析：** 不要认为自尊只是一种情绪，它会影响行为选择。

**记忆钩子：** 自尊让人站得直。

### 卡3



**配图：** ![](第二单元-焕发青春活力/03-card.png)
**提问：** 怎样做到自尊？

**答案：** 要知荣辱、讲自爱，尊重自己的人格，遵守规则，也尊重他人的尊严。

**易错分析：** 自尊不能变成自傲，尊重自己也要尊重他人。

**记忆钩子：** 尊己也尊人。

### 卡4



**配图：** ![](第二单元-焕发青春活力/04-card.png)
**提问：** 自信有什么作用？

**答案：** 自信能让我们充满激情和希望，激发潜能，坚定信念，增强克服困难的勇气和力量。

**易错分析：** 不要把自信理解成盲目自大，自信要建立在努力和实力基础上。

**记忆钩子：** 自信给行动加电。

### 卡5



**配图：** ![](第二单元-焕发青春活力/05-card.png)
**提问：** 怎样培养自信？

**答案：** 要看到自己的进步和长处，增强实力，积累成功体验，勇于尝试和坚持努力。

**易错分析：** 不要只靠口号获得自信，真正自信来自行动和积累。

**记忆钩子：** 实力是自信的底座。

### 卡6



**配图：** ![](第二单元-焕发青春活力/06-card.png)
**提问：** 自强的含义是什么？

**答案：** 自强是不断克服自己的弱点、战胜自己、超越自己，努力向上的精神品质。

**易错分析：** 不要把自强只理解成身体强壮，它更强调精神和行动。

**记忆钩子：** 自强先战胜自己。

### 卡7



**配图：** ![](第二单元-焕发青春活力/07-card.png)
**提问：** 为什么人生当自强？

**答案：** 自强能让我们在困难面前不低头，磨砺意志，增长才干，走向成熟和成功。

**易错分析：** 不要认为自强就是不需要别人帮助。自强的人也可以合理求助。

**记忆钩子：** 困难是自强的磨刀石。

### 卡8



**配图：** ![](第二单元-焕发青春活力/08-card.png)
**提问：** 怎样培养自强精神？

**答案：** 要树立远大理想，勇于面对困难，磨砺坚强意志，脚踏实地、不懈奋斗。

**易错分析：** 不要只写“有理想”，自强还要靠持续行动。

**记忆钩子：** 理想 + 意志 + 行动。

### 卡9



**配图：** ![](第二单元-焕发青春活力/09-card.png)
**提问：** 自尊、自信、自强三者有什么关系？

**答案：** 自尊让我们守住人格，自信给我们力量，自强推动我们不断进步，三者共同促进青春成长。

**易错分析：** 不要把三者割裂背诵，它们共同指向积极人格。

**记忆钩子：** 自尊站稳，自信发力，自强向前。
