# 第四单元 法治护航健康成长

## 基本信息

- 科目：道德与法治
- 年级：七年级下册
- 教材：统编版 / 人教版《道德与法治》（2025春版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元包括法律为我们护航、走近民法典、远离违法犯罪等内容。学习主线是：认识法律对未成年人成长的保护，了解民法典与生活的关系，增强法治观念，远离违法犯罪。

## Anki 记忆卡

### 卡1



**配图：** ![](第四单元-法治护航健康成长/01-card.png)
**提问：** 为什么未成年人需要特殊保护？

**答案：** 未成年人身心发育尚不成熟，自我保护能力较弱，辨别是非和自我控制能力不强，容易受到不良因素影响和侵害。

**易错分析：** 不要理解成未成年人可以不承担责任。特殊保护不等于放纵。

**记忆钩子：** 未成年需要保护，也要守规则。

### 卡2



**配图：** ![](第四单元-法治护航健康成长/02-card.png)
**提问：** 保护未成年人的防线有哪些？

**答案：** 家庭保护、学校保护、社会保护、网络保护、政府保护和司法保护共同构成保护未成年人的防线。

**易错分析：** 不要只写家庭和学校，新教材强调多方面保护，网络保护也很重要。

**记忆钩子：** 家、校、社、网、政、司。

### 卡3



**配图：** ![](第四单元-法治护航健康成长/03-card.png)
**提问：** 法律对未成年人有什么作用？

**答案：** 法律规范未成年人行为，保护未成年人合法权益，为未成年人健康成长提供法治保障。

**易错分析：** 不要只看到法律约束，法律也保护权利。

**记忆钩子：** 法律既是红线，也是护栏。

### 卡4



**配图：** ![](第四单元-法治护航健康成长/04-card.png)
**提问：** 民法典为什么被称为“社会生活的百科全书”？

**答案：** 民法典调整平等主体之间的人身关系和财产关系，与每个人从出生到死亡的生活密切相关。

**易错分析：** 不要把民法典只理解成处理犯罪的法律。犯罪主要涉及刑法。

**记忆钩子：** 民法典管生活中的人身和财产关系。

### 卡5



**配图：** ![](第四单元-法治护航健康成长/05-card.png)
**提问：** 民事权利主要包括哪些内容？

**答案：** 民事权利包括人身权利和财产权利等，如生命权、身体权、健康权、姓名权、肖像权、财产权等。

**易错分析：** 不要只想到财产权，人身权同样重要。

**记忆钩子：** 人身 + 财产。

### 卡6



**配图：** ![](第四单元-法治护航健康成长/06-card.png)
**提问：** 公民行使权利时应注意什么？

**答案：** 行使权利不能超越法律界限，不能损害国家、社会、集体利益和他人的合法权益。

**易错分析：** 不要把自由理解成想做什么就做什么。

**记忆钩子：** 权利有边界。

### 卡7



**配图：** ![](第四单元-法治护航健康成长/07-card.png)
**提问：** 什么是违法行为？

**答案：** 凡是不履行法律规定的义务，或者作出法律所禁止的行为，都是违法行为。

**易错分析：** 不要认为只有犯罪才违法。违法包括一般违法和犯罪。

**记忆钩子：** 违法不一定犯罪，犯罪一定违法。

### 卡8



**配图：** ![](第四单元-法治护航健康成长/08-card.png)
**提问：** 一般违法行为和犯罪有什么区别？

**答案：** 犯罪具有严重社会危害性、刑事违法性和应受刑罚处罚性；一般违法行为危害程度相对较轻。

**易错分析：** 不要把小错都说成犯罪，也不要轻视一般违法。

**记忆钩子：** 犯罪三特征：严重危害、刑事违法、刑罚处罚。

### 卡9



**配图：** ![](第四单元-法治护航健康成长/09-card.png)
**提问：** 青少年怎样远离违法犯罪？

**答案：** 要增强法治观念，依法自律，明辨是非，谨慎交友，杜绝不良行为，遇事依法求助和维权。

**易错分析：** 不要认为违法犯罪离自己很远，不良行为如果不改可能走向违法犯罪。

**记忆钩子：** 小错早改，远离大错。

### 卡10



**配图：** ![](第四单元-法治护航健康成长/10-card.png)
**提问：** 遇到合法权益受到侵害时怎么办？

**答案：** 要及时寻求父母、老师、学校、有关部门或法律机构帮助，必要时依法维权。

**易错分析：** 不要用报复或暴力方式解决问题，也不要沉默忍受。

**记忆钩子：** 不沉默，不报复，依法求助。

### 卡11



**配图：** ![](第四单元-法治护航健康成长/11-card.png)
**提问：** 法治护航健康成长的核心要求是什么？

**答案：** 既要学会用法律保护自己，也要自觉尊法学法守法用法，做法治中国建设的参与者和推动者。

**易错分析：** 不要只强调被保护，青少年也要主动守法、用法。

**记忆钩子：** 法律保护我，我也守法律。
