# 第三单元 传承中华优秀传统文化

## 基本信息

- 科目：道德与法治
- 年级：七年级下册
- 教材：统编版 / 人教版《道德与法治》（2025春版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元围绕中华优秀传统文化展开，重点理解核心思想理念、中华人文精神和传统美德。学习主线是：认识中华文化的价值，在继承中发展，在日常行动中践行传统美德。

## Anki 记忆卡

### 卡1



**配图：** ![](第三单元-传承中华优秀传统文化/01-card.png)
**提问：** 为什么要传承中华优秀传统文化？

**答案：** 中华优秀传统文化积淀着中华民族最深沉的精神追求，是中华民族独特的精神标识，为我们成长提供思想滋养。

**易错分析：** 不要把传统文化只理解成古代艺术或节日，它还包括思想理念、人文精神和传统美德。

**记忆钩子：** 传统文化是民族精神根脉。

### 卡2



**配图：** ![](第三单元-传承中华优秀传统文化/02-card.png)
**提问：** 中华优秀传统文化中的核心思想理念有哪些价值？

**答案：** 它们蕴含修身齐家、治国平天下、天人合一、讲信修睦等思想，为做人做事提供智慧。

**易错分析：** 不要死背词语，要理解它们与个人修养、社会治理和人与自然关系有关。

**记忆钩子：** 修身、家国、自然、诚信。

### 卡3



**配图：** ![](第三单元-传承中华优秀传统文化/03-card.png)
**提问：** 为什么要弘扬中华人文精神？

**答案：** 中华人文精神强调以人为本、家国情怀、崇德向善，有助于增强民族认同和文化自信。

**易错分析：** 不要把人文精神理解成空洞口号，它体现在尊重人、关心社会、热爱国家的行动中。

**记忆钩子：** 人、家国、德。

### 卡4



**配图：** ![](第三单元-传承中华优秀传统文化/04-card.png)
**提问：** 中华传统美德有哪些重要内容？

**答案：** 中华传统美德包括爱国、敬业、诚信、友善、孝亲敬长、勤劳勇敢、自强不息等。

**易错分析：** 不要只列一种美德，传统美德是一个丰富体系。

**记忆钩子：** 爱国诚信，孝亲自强。

### 卡5



**配图：** ![](第三单元-传承中华优秀传统文化/05-card.png)
**提问：** 怎样正确对待中华传统文化？

**答案：** 要坚持创造性转化、创新性发展，取其精华、去其糟粕，让传统文化适应现代生活。

**易错分析：** 不要认为传统文化要原封不动照搬，也不要全盘否定传统文化。

**记忆钩子：** 传承不是复制，是活化。

### 卡6



**配图：** ![](第三单元-传承中华优秀传统文化/06-card.png)
**提问：** 青少年怎样传承中华优秀传统文化？

**答案：** 要学习传统文化知识，参与文化活动，践行传统美德，增强文化自信，并向他人传播优秀文化。

**易错分析：** 不要把传承只理解成背古诗，还要落实到道德实践和生活行动中。

**记忆钩子：** 学、做、传。

### 卡7



**配图：** ![](第三单元-传承中华优秀传统文化/07-card.png)
**提问：** 文化自信从哪里来？

**答案：** 文化自信来自对中华优秀传统文化、革命文化和社会主义先进文化的认同与传承发展。

**易错分析：** 不要把文化自信理解成排斥其他文化，文化自信也可以开放包容。

**记忆钩子：** 自信不排外，根深更开放。

### 卡8



**配图：** ![](第三单元-传承中华优秀传统文化/08-card.png)
**提问：** 怎样对待外来文化？

**答案：** 要以开放包容的心态学习借鉴外来优秀文化，同时坚守中华文化立场。

**易错分析：** 不要全盘吸收，也不要盲目排斥。

**记忆钩子：** 开放借鉴，守住根脉。

### 卡9



**配图：** ![](第三单元-传承中华优秀传统文化/09-card.png)
**提问：** 传承传统美德为什么要从日常小事做起？

**答案：** 美德不是停留在口头上的知识，只有在孝亲敬长、诚实守信、勤劳节俭等日常行动中才能真正传承。

**易错分析：** 不要把美德当成考试答案，生活实践才是关键。

**记忆钩子：** 美德在行动里发光。
