# 第十章 西半球的国家

## 基本信息

- 科目：地理
- 年级：七年级下册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级下册“十 西半球的国家”，重点包括美国和巴西。学习主线是：用自然条件解释农业带和资源开发，用人口城市分布理解经济活动和环境问题。

## Anki 记忆卡

### 卡1

**配图：** ![](第十章-西半球的国家/01-card.png)


**提问：** 美国农业生产有什么突出特点？

**答案：** 美国农业生产机械化和专业化程度高，形成不同农业带。

**易错分析：** 不要把美国农业理解为小农分散经营为主。

**记忆钩子：** 机械化 + 专业化。

### 卡2

**配图：** ![](第十章-西半球的国家/02-card.png)


**提问：** 美国乳畜带为什么分布在东北部？

**答案：** 东北部气候冷湿，适合牧草生长，城市和人口密集，乳畜产品市场广阔。

**易错分析：** 不要只答气候，市场也很关键。

**记忆钩子：** 湿冷长草，城市喝奶。

### 卡3

**配图：** ![](第十章-西半球的国家/03-card.png)


**提问：** 美国东北部工业区发展早的条件有哪些？

**答案：** 煤铁资源丰富，交通便利，开发历史早，市场广阔。

**易错分析：** 不要只说技术先进，传统工业区发展早有资源和交通基础。

**记忆钩子：** 煤、铁、交、市。

### 卡4

**配图：** ![](第十章-西半球的国家/04-card.png)


**提问：** 美国高新技术产业中心硅谷在哪里？

**答案：** 硅谷位于美国旧金山东南部，是世界著名高新技术产业中心。

**易错分析：** 不要把硅谷写成纽约附近。

**记忆钩子：** 旧金山东南。

### 卡5

**配图：** ![](第十章-西半球的国家/05-card.png)


**提问：** 巴西最显著的人口文化特征之一是什么？

**答案：** 巴西混血种人数量多，文化具有多元融合特点。

**易错分析：** 不要把巴西人口都说成单一人种。

**记忆钩子：** 多元混合。

### 卡6

**配图：** ![](第十章-西半球的国家/06-card.png)


**提问：** 巴西热带雨林主要分布在哪里？

**答案：** 巴西热带雨林主要分布在亚马孙河流域。

**易错分析：** 不要把热带雨林说成主要在巴西高原。

**记忆钩子：** 雨林亚马孙。

### 卡7

**配图：** ![](第十章-西半球的国家/07-card.png)


**提问：** 亚马孙热带雨林有什么生态作用？

**答案：** 亚马孙热带雨林具有调节气候、涵养水源、保护生物多样性等重要生态作用。

**易错分析：** 不要只说提供木材，生态作用更重要。

**记忆钩子：** 绿色水库，生物宝库。

### 卡8

**配图：** ![](第十章-西半球的国家/08-card.png)


**提问：** 巴西人口和城市为什么多分布在东南沿海？

**答案：** 东南沿海气候较适宜、交通便利、开发历史早、工农业发达，所以人口和城市集中。

**易错分析：** 不要认为亚马孙平原面积大人口就多。那里湿热，开发困难。

**记忆钩子：** 东南集中。

### 卡9

**配图：** ![](第十章-西半球的国家/09-card.png)


**提问：** 巴西的首都是哪里？迁都有何作用？

**答案：** 巴西首都是巴西利亚，迁都有利于加强对内陆地区的开发和管理。

**易错分析：** 不要把巴西首都写成里约热内卢或圣保罗。

**记忆钩子：** 巴西利亚在内陆。

### 卡10

**配图：** ![](第十章-西半球的国家/10-card.png)


**提问：** 美国和巴西农业分布都体现了什么地理思想？

**答案：** 农业布局要因地制宜，依据气候、地形、市场和技术等条件安排生产。

**易错分析：** 不要只背农业带名称，要理解因地制宜。

**记忆钩子：** 适合什么，就发展什么。
