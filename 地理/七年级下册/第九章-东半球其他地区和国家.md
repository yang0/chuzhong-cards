# 第九章 东半球其他地区和国家

## 基本信息

- 科目：地理
- 年级：七年级下册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级下册“九 东半球其他地区和国家”，重点包括中东、欧洲西部、撒哈拉以南非洲、澳大利亚。学习主线是：位置和资源决定区域特色，气候和历史影响农业、工业、旅游与人口分布。

## Anki 记忆卡

### 卡1

**配图：** ![](第九章-东半球其他地区和国家/01-card.png)


**提问：** 中东地理位置为什么重要？

**答案：** 中东联系亚、欧、非三洲，沟通大西洋和印度洋，地理位置十分重要。

**易错分析：** 不要只说中东有石油，位置也是重要考点。

**记忆钩子：** 三洲五海之地。

### 卡2

**配图：** ![](第九章-东半球其他地区和国家/02-card.png)


**提问：** 中东石油主要分布在哪里？

**答案：** 中东石油主要分布在波斯湾及其沿岸地区。

**易错分析：** 不要把中东石油说成均匀分布。

**记忆钩子：** 油在波湾。

### 卡3

**配图：** ![](第九章-东半球其他地区和国家/03-card.png)


**提问：** 中东水资源为什么紧张？

**答案：** 中东以热带沙漠气候为主，降水稀少，河流少，水资源短缺。

**易错分析：** 不要认为靠海就一定淡水资源丰富。

**记忆钩子：** 干旱缺水。

### 卡4

**配图：** ![](第九章-东半球其他地区和国家/04-card.png)


**提问：** 欧洲西部畜牧业发达的自然原因是什么？

**答案：** 欧洲西部温带海洋性气候广布，全年温和湿润，适合多汁牧草生长。

**易错分析：** 题目问自然原因时，不要只答经济发达。

**记忆钩子：** 湿润长草，牧草养牛羊。

### 卡5

**配图：** ![](第九章-东半球其他地区和国家/05-card.png)


**提问：** 欧洲西部旅游业发达有哪些条件？

**答案：** 欧洲西部自然和人文旅游资源丰富，交通便利，服务业发达。

**易错分析：** 不要只列一个景点，要概括资源和条件。

**记忆钩子：** 景、文、交、服。

### 卡6

**配图：** ![](第九章-东半球其他地区和国家/06-card.png)


**提问：** 撒哈拉以南非洲为什么被称为黑种人的故乡？

**答案：** 撒哈拉以南非洲是黑色人种的主要分布区。

**易错分析：** 不要把整个非洲都简单等同于黑种人地区，北非人种和文化差异明显。

**记忆钩子：** 沙南黑乡。

### 卡7

**配图：** ![](第九章-东半球其他地区和国家/07-card.png)


**提问：** 撒哈拉以南非洲经济有什么突出特点？

**答案：** 许多国家出口初级农矿产品，进口工业制成品，在国际贸易中处于不利地位。

**易错分析：** 不要只说资源丰富就经济发达。

**记忆钩子：** 卖原料，买成品。

### 卡8

**配图：** ![](第九章-东半球其他地区和国家/08-card.png)


**提问：** 澳大利亚为什么被称为世界活化石博物馆？

**答案：** 澳大利亚长期孤立于南半球海洋中，保存了许多古老特有生物。

**易错分析：** 不要只说动物可爱，原因是长期孤立和进化环境独特。

**记忆钩子：** 孤岛留古。

### 卡9

**配图：** ![](第九章-东半球其他地区和国家/09-card.png)


**提问：** 澳大利亚为什么被称为骑在羊背上的国家？

**答案：** 澳大利亚畜牧业发达，绵羊数量和羊毛出口曾居世界前列。

**易错分析：** 不要把它和“坐在矿车上的国家”混淆。

**记忆钩子：** 羊背牧业。

### 卡10

**配图：** ![](第九章-东半球其他地区和国家/10-card.png)


**提问：** 澳大利亚人口和城市为什么多分布在东南沿海？

**答案：** 东南沿海气候较温暖湿润，交通便利，开发历史较早，人口和城市集中。

**易错分析：** 不要认为内陆面积大人口就多。澳大利亚内陆干旱。

**记忆钩子：** 东南宜居。
