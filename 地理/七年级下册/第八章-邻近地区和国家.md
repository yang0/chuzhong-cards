# 第八章 邻近地区和国家

## 基本信息

- 科目：地理
- 年级：七年级下册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级下册“八 邻近地区和国家”，重点包括日本、东南亚、印度、俄罗斯。学习主线是：用位置和自然条件解释产业、交通、人口和农业差异。

## Anki 记忆卡

### 卡1

**配图：** ![](第八章-邻近地区和国家/01-card.png)


**提问：** 日本多火山地震的原因是什么？

**答案：** 日本位于板块交界地带，地壳活跃，所以多火山地震。

**易错分析：** 不要只说日本是岛国就多地震，关键是板块交界。

**记忆钩子：** 板边多震。

### 卡2

**配图：** ![](第八章-邻近地区和国家/02-card.png)


**提问：** 日本工业为什么多分布在太平洋沿岸和濑户内海沿岸？

**答案：** 这里港口条件好，便于进口原料和燃料、出口工业产品。

**易错分析：** 不要说日本资源丰富所以工业沿海。日本矿产资源贫乏。

**记忆钩子：** 原料进港，产品出港。

### 卡3

**配图：** ![](第八章-邻近地区和国家/03-card.png)


**提问：** 东南亚的位置有什么重要性？

**答案：** 东南亚位于亚洲和大洋洲、太平洋和印度洋之间的十字路口，交通位置重要。

**易错分析：** 不要只说东南亚在中国南边，关键是两洲两洋的十字路口。

**记忆钩子：** 两洲两洋。

### 卡4

**配图：** ![](第八章-邻近地区和国家/04-card.png)


**提问：** 马六甲海峡为什么重要？

**答案：** 马六甲海峡是连接太平洋和印度洋的重要海上通道。

**易错分析：** 不要把马六甲海峡说成连接大西洋和印度洋。

**记忆钩子：** 海上咽喉。

### 卡5

**配图：** ![](第八章-邻近地区和国家/05-card.png)


**提问：** 东南亚主要粮食作物为什么是水稻？

**答案：** 东南亚高温多雨，人口稠密、劳动力丰富，适合水稻种植。

**易错分析：** 不要只答气候，劳动力和人口需求也重要。

**记忆钩子：** 热、水、人。

### 卡6

**配图：** ![](第八章-邻近地区和国家/06-card.png)


**提问：** 印度水旱灾害频繁与什么有关？

**答案：** 印度水旱灾害频繁与西南季风不稳定有关。

**易错分析：** 不要把印度水旱灾害只归因于地形。

**记忆钩子：** 季风不稳，旱涝跟着来。

### 卡7

**配图：** ![](第八章-邻近地区和国家/07-card.png)


**提问：** 印度服务外包产业有什么特点？

**答案：** 印度软件外包产业发展快，被称为“世界办公室”。

**易错分析：** 不要把印度只看成农业国家。

**记忆钩子：** 世界办公室。

### 卡8

**配图：** ![](第八章-邻近地区和国家/08-card.png)


**提问：** 俄罗斯自然资源有什么特点？

**答案：** 俄罗斯自然资源丰富，种类齐全，是世界重要资源大国。

**易错分析：** 不要把资源丰富等同于人口稠密。

**记忆钩子：** 资源大仓库。

### 卡9

**配图：** ![](第八章-邻近地区和国家/09-card.png)


**提问：** 俄罗斯交通运输有什么特点？

**答案：** 俄罗斯交通部门齐全，铁路运输发达，欧洲部分铁路网密集，亚洲部分铁路较稀疏。

**易错分析：** 不要认为俄罗斯河运最发达。其纬度高，河流结冰期长。

**记忆钩子：** 铁路骨架，欧洲更密。

### 卡10

**配图：** ![](第八章-邻近地区和国家/10-card.png)


**提问：** 俄罗斯工业分布为什么常靠近资源产地？

**答案：** 俄罗斯矿产资源丰富，重工业多接近资源产地布局。

**易错分析：** 不要把俄罗斯工业全部说成沿海分布。

**记忆钩子：** 近矿建厂。
