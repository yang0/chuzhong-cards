# 第十一章 极地地区

## 基本信息

- 科目：地理
- 年级：七年级下册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级下册“十一 极地地区”，包括南极地区、北极地区和极地保护。学习主线是：南极和北极都寒冷但中心不同、环境不同、动物不同，极地科研价值大但生态脆弱，必须重视保护。

## Anki 记忆卡

### 卡1

**配图：** ![](第十一章-极地地区/01-card.png)


**提问：** 南极地区的位置有什么特点？

**答案：** 南极地区主要位于南极圈以内，以南极洲为中心，被太平洋、大西洋、印度洋包围。

**易错分析：** 不要把南极理解成只有一个点，它是以南极洲为中心的广大地区。

**记忆钩子：** 南极：陆中洋外。

### 卡2

**配图：** ![](第十一章-极地地区/02-card.png)


**提问：** 北极地区的位置有什么特点？

**答案：** 北极地区主要位于北极圈以内，以北冰洋为中心，被亚洲、欧洲、北美洲环绕。

**易错分析：** 不要把北极地区说成以大陆为中心。

**记忆钩子：** 北极：洋中陆外。

### 卡3

**配图：** ![](第十一章-极地地区/03-card.png)


**提问：** 南极地区自然环境有什么突出特点？

**答案：** 南极地区酷寒、干燥、烈风，被称为冰雪高原。

**易错分析：** 不要只说冷，干燥和烈风也重要。

**记忆钩子：** 冷、干、风。

### 卡4

**配图：** ![](第十一章-极地地区/04-card.png)


**提问：** 为什么南极比北极更寒冷？

**答案：** 南极以高大陆为主，海拔高，冰雪反射强；北极以海洋为主，海洋有调节作用。

**易错分析：** 不要只用纬度解释南北极温差。

**记忆钩子：** 高大陆更冷。

### 卡5

**配图：** ![](第十一章-极地地区/05-card.png)


**提问：** 极地地区科学考察有什么意义？

**答案：** 极地保存着大量自然信息，有助于研究气候变化、冰川、海洋和生物等问题。

**易错分析：** 不要把科考理解成单纯探险。

**记忆钩子：** 冰层像地球气候档案。

### 卡6

**配图：** ![](第十一章-极地地区/06-card.png)


**提问：** 我国在南极建立科考站通常选择什么季节？

**答案：** 通常选择南极暖季，也就是每年11月至次年3月前后进行建设和考察。

**易错分析：** 不要按中国冬夏直接判断南极季节，南北半球季节相反。

**记忆钩子：** 中国冬天，南极较暖。

### 卡7

**配图：** ![](第十一章-极地地区/07-card.png)


**提问：** 南极和北极的代表性动物分别是什么？

**答案：** 南极代表性动物是企鹅，北极代表性动物是北极熊。

**易错分析：** 不要把北极熊画到南极，也不要把企鹅说成北极代表动物。

**记忆钩子：** 南企鹅，北极熊。

### 卡8

**配图：** ![](第十一章-极地地区/08-card.png)


**提问：** 极地地区环境保护为什么重要？

**答案：** 极地生态环境脆弱，一旦破坏恢复困难，并会影响全球气候和海洋环境。

**易错分析：** 不要认为极地无人居住就可以随意开发。

**记忆钩子：** 极地像冰玻璃，破了难修。

### 卡9

**配图：** ![](第十一章-极地地区/09-card.png)


**提问：** 南极地区和北极地区最核心的空间差异是什么？

**答案：** 南极地区以陆地为中心，北极地区以海洋为中心。

**易错分析：** 不要把南极和北极的中心记反。

**记忆钩子：** 南陆北洋。

### 卡10

**配图：** ![](第十一章-极地地区/10-card.png)


**提问：** 学习极地地区时要抓住哪组对比？

**答案：** 南极以陆地为中心、更加寒冷、代表动物是企鹅；北极以海洋为中心、代表动物是北极熊，并有常住居民。

**易错分析：** 不要把南极和北极的中心、动物、居民情况混淆。

**记忆钩子：** 南陆企鹅，北洋北熊。
