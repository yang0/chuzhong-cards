# 第二章 地图

## 基本信息

- 科目：地理
- 年级：七年级上册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级上册“二 地图”，包括地图的阅读、地形图的判读、地图的选择和应用。学习主线是：先掌握方向、比例尺、图例，再会读等高线和分层设色地形图，最后能按目的选择合适地图。

## Anki 记忆卡

### 卡1

**配图：** ![](第二章-地图/01-card.png)


**提问：** 地图的三要素是什么？

**答案：** 地图三要素是方向、比例尺和图例。

**易错分析：** 不要把标题、颜色当作地图三要素。

**记忆钩子：** 方、比、图。

### 卡2

**配图：** ![](第二章-地图/02-card.png)


**提问：** 地图上怎样判断方向？

**答案：** 一般地图按“上北下南、左西右东”判断；有指向标的地图按指向标判断；经纬网地图按经线纬线判断。

**易错分析：** 不要所有地图都机械套用“上北下南”，有指向标时要看指向标。

**记忆钩子：** 先找指向标，没有再用上北下南。

### 卡3

**配图：** ![](第二章-地图/03-card.png)


**提问：** 比例尺大小怎样影响地图表示范围和内容详略？

**答案：** 图幅相同时，比例尺越大，表示范围越小，内容越详细；比例尺越小，表示范围越大，内容越简略。

**易错分析：** 不要把比例尺大误解成实际范围大。

**记忆钩子：** 大详小广。

### 卡4

**配图：** ![](第二章-地图/04-card.png)


**提问：** 图例和注记有什么作用？

**答案：** 图例说明地图符号代表的地理事物，注记是地图上的文字和数字说明。

**易错分析：** 不要把图例和注记混为一谈：图例偏符号，注记偏文字数字。

**记忆钩子：** 图例看符号，注记看文字。

### 卡5

**配图：** ![](第二章-地图/05-card.png)


**提问：** 等高线地形图上怎样判断坡陡坡缓？

**答案：** 等高线密集处坡陡，等高线稀疏处坡缓。

**易错分析：** 不要只看海拔高低判断坡度，坡度看等高线疏密。

**记忆钩子：** 密陡疏缓。

### 卡6

**配图：** ![](第二章-地图/06-card.png)


**提问：** 山峰、山脊、山谷在等高线图上怎样区分？

**答案：** 山峰是等高线闭合且数值向中心增大；山脊等高线向低处凸出；山谷等高线向高处凸出。

**易错分析：** 不要把山脊和山谷凸向记反。

**记忆钩子：** 谷凸向高，水往低处流。

### 卡7

**配图：** ![](第二章-地图/07-card.png)


**提问：** 分层设色地形图中颜色通常表示什么？

**答案：** 分层设色地形图用不同颜色表示地势高低，绿色多表示平原，褐色多表示较高山地高原。

**易错分析：** 不要把颜色简单理解成植被颜色。

**记忆钩子：** 绿低褐高。

### 卡8

**配图：** ![](第二章-地图/08-card.png)


**提问：** 选择地图时要考虑什么？

**答案：** 要根据使用目的选择合适类型和合适比例尺的地图。

**易错分析：** 不要认为一张地图能解决所有问题。查路线、看地形、找行政区需要不同地图。

**记忆钩子：** 用途决定地图。
