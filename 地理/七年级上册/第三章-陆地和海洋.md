# 第三章 陆地和海洋

## 基本信息

- 科目：地理
- 年级：七年级上册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级上册“三 陆地和海洋”，包括大洲和大洋、世界的地形、海陆的变迁。学习主线是：先认识海陆格局，再读世界地形，最后用大陆漂移和板块运动解释海陆变化。

## Anki 记忆卡

### 卡1

**配图：** ![](第三章-陆地和海洋/01-card.png)


**提问：** 地球上海洋和陆地面积比例大致是多少？

**答案：** 地球表面约七分是海洋、三分是陆地，常概括为七分海洋、三分陆地。

**易错分析：** 不要把陆地面积说成大于海洋。

**记忆钩子：** 七海三陆。

### 卡2

**配图：** ![](第三章-陆地和海洋/02-card.png)


**提问：** 世界海陆分布有什么总体特点？

**答案：** 陆地主要集中在北半球和东半球，海洋大多分布在南半球和西半球。

**易错分析：** 不要理解为某个半球全是陆地或全是海洋。

**记忆钩子：** 北东陆多，南西海广。

### 卡3

**配图：** ![](第三章-陆地和海洋/03-card.png)


**提问：** 七大洲按面积从大到小怎样排列？

**答案：** 亚洲、非洲、北美洲、南美洲、南极洲、欧洲、大洋洲。

**易错分析：** 不要把欧洲排在大洋洲后面。

**记忆钩子：** 亚非北南美，南欧大。

### 卡4

**配图：** ![](第三章-陆地和海洋/04-card.png)


**提问：** 四大洋按面积从大到小怎样排列？

**答案：** 太平洋、大西洋、印度洋、北冰洋。

**易错分析：** 不要把印度洋排在大西洋前面。

**记忆钩子：** 太大印北。

### 卡5

**配图：** ![](第三章-陆地和海洋/05-card.png)


**提问：** 亚洲和欧洲的分界线主要有哪些？

**答案：** 乌拉尔山脉、乌拉尔河、里海、大高加索山脉、黑海、土耳其海峡。

**易错分析：** 不要只写乌拉尔山脉，亚欧分界是一串地理事物。

**记忆钩子：** 乌乌里大黑土。

### 卡6

**配图：** ![](第三章-陆地和海洋/06-card.png)


**提问：** 亚洲和非洲、南北美洲的分界线分别是什么？

**答案：** 亚洲和非洲的分界线是苏伊士运河；南、北美洲的分界线是巴拿马运河。

**易错分析：** 不要把苏伊士运河和巴拿马运河写反。

**记忆钩子：** 苏分亚非，巴分美洲。

### 卡7

**配图：** ![](第三章-陆地和海洋/07-card.png)


**提问：** 世界陆地基本地形类型有哪些？

**答案：** 陆地基本地形类型有平原、高原、山地、丘陵和盆地。

**易错分析：** 不要把沙漠、森林当作基本地形类型。

**记忆钩子：** 平高山丘盆。

### 卡8

**配图：** ![](第三章-陆地和海洋/08-card.png)


**提问：** 海陆变迁的主要原因有哪些？

**答案：** 海陆变迁主要由地壳变动、海平面升降和人类活动等引起。

**易错分析：** 不要只归因于人类填海，地壳变动和海平面变化更重要。

**记忆钩子：** 壳、海、人。

### 卡9

**配图：** ![](第三章-陆地和海洋/09-card.png)


**提问：** 大陆漂移学说是谁提出的？核心观点是什么？

**答案：** 魏格纳提出大陆漂移学说，认为大陆曾经连在一起，后来逐渐漂移到现在的位置。

**易错分析：** 不要把大陆漂移理解成短时间快速移动。

**记忆钩子：** 大陆像拼图，慢慢漂开。

### 卡10

**配图：** ![](第三章-陆地和海洋/10-card.png)


**提问：** 为什么板块交界地带多火山地震？

**答案：** 板块交界处地壳比较活跃，板块碰撞或张裂容易形成火山和地震。

**易错分析：** 不要认为板块内部一定没有地震，只是交界处更集中。

**记忆钩子：** 边界最活跃。
