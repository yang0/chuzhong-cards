# 第一章 地球

## 基本信息

- 科目：地理
- 年级：七年级上册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级上册“一 地球”，包括地球的宇宙环境、地球与地球仪、地球的运动。学习主线是：先知道地球在宇宙中的位置，再用地球仪和经纬网定位，最后用自转、公转解释昼夜、时间、季节和五带。

## Anki 记忆卡

### 卡1

**配图：** ![](第一章-地球/01-card.png)


**提问：** 地球在宇宙中属于什么天体系统？

**答案：** 地球是太阳系中的一颗行星，围绕太阳运动，同时又有月球作为天然卫星。

**易错分析：** 不要把地球说成恒星。恒星能自己发光发热，地球是行星。

**记忆钩子：** 太阳是灯，地球是绕灯转的行星。

### 卡2

**配图：** ![](第一章-地球/02-card.png)


**提问：** 人类认识地球形状的过程说明了什么？

**答案：** 地球是一个两极稍扁、赤道略鼓的不规则球体，人类认识地球形状经历了由猜想到实践证明的过程。

**易错分析：** 不要只说地球是正圆球。教材强调它是不规则球体。

**记忆钩子：** 证据升级：想象、航行、卫星照片。

### 卡3

**配图：** ![](第一章-地球/03-card.png)


**提问：** 地球仪上的纬线有什么特点？

**答案：** 纬线指示东西方向，形状是圆圈，长度从赤道向两极逐渐缩短。

**易错分析：** 不要把纬线说成连接南北两极的半圆。那是经线。

**记忆钩子：** 横纬东西。

### 卡4

**配图：** ![](第一章-地球/04-card.png)


**提问：** 地球仪上的经线有什么特点？

**答案：** 经线指示南北方向，形状是半圆，所有经线长度大致相等，并相交于南北两极。

**易错分析：** 不要说经线指示东西方向。

**记忆钩子：** 竖经南北。

### 卡5

**配图：** ![](第一章-地球/05-card.png)


**提问：** 怎样区分东西经和南北纬？

**答案：** 0°经线以东是东经，以西是西经；赤道以北是北纬，以南是南纬。

**易错分析：** 不要把赤道当成东西经分界线。赤道分南北纬。

**记忆钩子：** 经线分东西，赤道分南北。

### 卡6

**配图：** ![](第一章-地球/06-card.png)


**提问：** 经纬网的主要作用是什么？

**答案：** 经纬网可以确定地球表面任意地点的位置，定位时要同时看经度和纬度。

**易错分析：** 不要只看经度或只看纬度，位置由经纬线交点确定。

**记忆钩子：** 横竖一交，位置锁定。

### 卡7

**配图：** ![](第一章-地球/07-card.png)


**提问：** 地球自转产生了哪些地理现象？

**答案：** 地球自转产生昼夜更替和时间差异，自转方向是自西向东。

**易错分析：** 不要把四季变化归因于自转。四季主要与公转有关。

**记忆钩子：** 自转换班，昼夜更替。

### 卡8

**配图：** ![](第一章-地球/08-card.png)


**提问：** 地球公转产生了哪些地理现象？

**答案：** 地球公转和地轴倾斜共同造成四季变化、昼夜长短变化和五带划分。

**易错分析：** 不要只说绕太阳转就有四季，地轴倾斜也很关键。

**记忆钩子：** 斜着绕太阳，四季才明显。

### 卡9

**配图：** ![](第一章-地球/09-card.png)


**提问：** 五带划分主要依据是什么？

**答案：** 五带主要依据不同地区获得太阳光热的多少来划分，界线与回归线、极圈有关。

**易错分析：** 不要把五带划分理解成按国家或大洲划分。

**记忆钩子：** 热量分带，看回归线和极圈。
