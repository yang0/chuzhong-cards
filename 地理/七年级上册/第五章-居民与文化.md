# 第五章 居民与文化

## 基本信息

- 科目：地理
- 年级：七年级上册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级上册“五 居民与文化”，包括人口、人种、语言、宗教、聚落和文化景观。学习主线是：人口在哪里多、为什么多，居民文化有什么差异，聚落和文化景观怎样体现自然与人文环境。

## Anki 记忆卡

### 卡1

**配图：** ![](第五章-居民与文化/01-card.png)


**提问：** 人口自然增长率怎样计算？

**答案：** 人口自然增长率等于人口出生率减去人口死亡率。

**易错分析：** 不要把人口迁入、迁出算进自然增长率。

**记忆钩子：** 自然增长 = 生 - 死。

### 卡2

**配图：** ![](第五章-居民与文化/02-card.png)


**提问：** 世界人口稠密区主要分布在哪里？

**答案：** 世界人口稠密区主要分布在亚洲东部和南部、欧洲西部、北美洲东部等中低纬度近海平原地区。

**易错分析：** 不要认为所有沿海地区人口都稠密，还要看气候、地形、开发历史等条件。

**记忆钩子：** 中低纬、近海、平原。

### 卡3

**配图：** ![](第五章-居民与文化/03-card.png)


**提问：** 世界人口稀疏区主要有哪些？

**答案：** 极端干旱的沙漠、过于湿热的雨林、终年严寒的高纬地区和地势高峻的高原山区人口稀疏。

**易错分析：** 不要只说自然条件差，要能说出干、湿热、寒、高这些限制。

**记忆钩子：** 干、湿、寒、高，人少。

### 卡4

**配图：** ![](第五章-居民与文化/04-card.png)


**提问：** 人口增长过快可能带来哪些问题？

**答案：** 人口增长过快可能造成就业困难、住房紧张、交通拥挤、资源短缺和环境压力加大。

**易错分析：** 不要只看到人口多能提供劳动力，也要看到资源和环境压力。

**记忆钩子：** 人太多，城市和资源都变挤。

### 卡5

**配图：** ![](第五章-居民与文化/05-card.png)


**提问：** 人口增长过慢或负增长可能带来哪些问题？

**答案：** 可能造成劳动力短缺、人口老龄化、社会养老负担加重等问题。

**易错分析：** 不要认为人口越少一定越好。

**记忆钩子：** 人太少，劳动力不够。

### 卡6

**配图：** ![](第五章-居民与文化/06-card.png)


**提问：** 世界三大人种通常指什么？

**答案：** 世界三大人种通常指黄色人种、白色人种和黑色人种。

**易错分析：** 不要用人种差异判断高低优劣，各人种一律平等。

**记忆钩子：** 黄、白、黑，平等无高低。

### 卡7

**配图：** ![](第五章-居民与文化/07-card.png)


**提问：** 世界使用人数最多和使用范围最广的语言分别是什么？

**答案：** 使用人数最多的是汉语，使用范围最广的是英语。

**易错分析：** 不要把“人数最多”和“范围最广”混为一谈。

**记忆钩子：** 人数汉语，范围英语。

### 卡8

**配图：** ![](第五章-居民与文化/08-card.png)


**提问：** 世界三大宗教是什么？

**答案：** 世界三大宗教是基督教、伊斯兰教和佛教。

**易错分析：** 不要把道教列入世界三大宗教。

**记忆钩子：** 基、伊、佛。

### 卡9

**配图：** ![](第五章-居民与文化/09-card.png)


**提问：** 聚落形成和发展的有利条件有哪些？

**答案：** 地形平坦、土壤肥沃、水源充足、交通便利、自然资源丰富等有利于聚落形成和发展。

**易错分析：** 不要只看一个条件，聚落发展常是多种条件共同作用。

**记忆钩子：** 平、水、土、交、资。

### 卡10

**配图：** ![](第五章-居民与文化/10-card.png)


**提问：** 乡村聚落和城市聚落有什么主要差异？

**答案：** 乡村聚落居民主要从事农业生产活动，城市聚落居民主要从事工业、服务业等非农业活动。

**易错分析：** 不要只用房屋高低区分乡村和城市，关键看生产生活方式。

**记忆钩子：** 看产业，不只看楼高。

### 卡11

**配图：** ![](第五章-居民与文化/11-card.png)


**提问：** 传统民居为什么具有地域差异？

**答案：** 传统民居受当地气候、地形、建筑材料和生活方式影响，体现人地关系。

**易错分析：** 不要把民居差异只当成审美差异，它常常适应当地自然环境。

**记忆钩子：** 因地造屋。

### 卡12

**配图：** ![](第五章-居民与文化/12-card.png)


**提问：** 保护传统聚落和文化景观有什么意义？

**答案：** 它们记录历史文化和地域特色，保护它们有利于传承文化，也能合理发展特色旅游。

**易错分析：** 不要把保护理解成完全不许发展，关键是合理保护与利用。

**记忆钩子：** 老聚落是地方记忆相册。
