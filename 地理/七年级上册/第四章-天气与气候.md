# 第四章 天气与气候

## 基本信息

- 科目：地理
- 年级：七年级上册
- 教材：人教版《地理》（2024版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本章对应人教版七年级上册“四 天气与气候”，包括多变的天气、气温和降水、世界的气候。学习主线是：先分清天气和气候，再读气温、降水资料，最后理解世界气候分布和影响因素。

## Anki 记忆卡

### 卡1

**配图：** ![](第四章-天气与气候/01-card.png)


**提问：** 天气和气候有什么区别？

**答案：** 天气是一个地区短时间的大气状况，气候是一个地区多年平均的天气状况。

**易错分析：** 不要把“今天下雨”说成气候，它是天气。

**记忆钩子：** 短天长候。

### 卡2

**配图：** ![](第四章-天气与气候/02-card.png)


**提问：** 卫星云图上绿色、蓝色、白色通常表示什么？

**答案：** 绿色表示陆地，蓝色表示海洋，白色表示云区，白色越浓云层越厚。

**易错分析：** 不要把白色云区理解成陆地。

**记忆钩子：** 蓝海、绿陆、白云。

### 卡3

**配图：** ![](第四章-天气与气候/03-card.png)


**提问：** 气温日变化通常怎样表现？

**答案：** 一天中最高气温通常出现在午后2时左右，最低气温通常出现在日出前后。

**易错分析：** 不要把最高气温写成正午12时。

**记忆钩子：** 最高午后，最低日出。

### 卡4

**配图：** ![](第四章-天气与气候/04-card.png)


**提问：** 北半球陆地气温年变化通常有什么规律？

**答案：** 北半球陆地通常7月气温最高、1月气温最低。

**易错分析：** 不要把南北半球季节混在一起。

**记忆钩子：** 北陆七一。

### 卡5

**配图：** ![](第四章-天气与气候/05-card.png)


**提问：** 世界气温分布的基本规律是什么？

**答案：** 世界气温大体从低纬向高纬递减。

**易错分析：** 不要忽视地形和海陆也会影响局部气温。

**记忆钩子：** 赤热极冷。

### 卡6

**配图：** ![](第四章-天气与气候/06-card.png)


**提问：** 同纬度地区海洋和陆地气温有什么差异？

**答案：** 同纬度地区，夏季陆地气温高于海洋，冬季陆地气温低于海洋。

**易错分析：** 不要说海洋永远比陆地冷或热。

**记忆钩子：** 陆快海慢。

### 卡7

**配图：** ![](第四章-天气与气候/07-card.png)


**提问：** 降水形成需要哪些条件？

**答案：** 空气中有足够水汽，气温下降，水汽凝结并形成能降落到地面的水滴或冰晶。

**易错分析：** 不要把有云等同于一定降水。

**记忆钩子：** 水汽遇冷，云滴长大。

### 卡8

**配图：** ![](第四章-天气与气候/08-card.png)


**提问：** 世界降水分布有什么基本规律？

**答案：** 赤道附近降水多，两极地区降水少；中纬度沿海多、内陆少；回归线附近大陆西岸少、东岸多。

**易错分析：** 不要只背一条规律，降水受纬度、海陆、地形等多因素影响。

**记忆钩子：** 赤湿极干，沿海多雨。

### 卡9

**配图：** ![](第四章-天气与气候/09-card.png)


**提问：** 气候资料图怎样读？

**答案：** 看气温曲线判断冷热，看降水柱状图判断干湿，并结合月份判断季节。

**易错分析：** 不要只看气温或只看降水。

**记忆钩子：** 线温柱水。

### 卡10

**配图：** ![](第四章-天气与气候/10-card.png)


**提问：** 影响气候的主要因素有哪些？

**答案：** 影响气候的主要因素有纬度位置、海陆位置和地形等。

**易错分析：** 不要把某地气候差异只归因于一个因素。

**记忆钩子：** 纬、海、地。

### 卡11

**配图：** ![](第四章-天气与气候/11-card.png)


**提问：** 山地迎风坡和背风坡降水有什么差异？

**答案：** 迎风坡空气被迫抬升，降水较多；背风坡空气下沉，降水较少。

**易错分析：** 不要把迎风坡和背风坡记反。

**记忆钩子：** 迎风多雨。

### 卡12

**配图：** ![](第四章-天气与气候/12-card.png)


**提问：** 人类活动会怎样影响气候？

**答案：** 人类排放温室气体、改变下垫面等会影响局地或全球气候，如城市热岛和全球变暖。

**易错分析：** 不要把所有天气异常都直接归因于人类活动。

**记忆钩子：** 温室气体像给地球盖被子。
