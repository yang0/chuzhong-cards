# 物质科学 测量与物态变化

## 基本信息

- 科目：科学
- 年级：七年级下册
- 教材：浙教版《科学》
- 章节：测量与物态变化
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析
- 来源：错题 Y7S-WRONG-001 至 017

## 单元纲要

本单元涵盖长度的测量、体积的测量、温度的测量、质量的测量、时间的测量，以及物质三态变化（熔化与凝固、汽化与液化、升华与凝华）。

## Anki 记忆卡

### 卡1

**提问：** 使用托盘天平的步骤是什么？读数时砝码和游码各如何计入总质量？

**答案：** 步骤：①放平（天平放水平桌面）；②归零（游码移到零刻度）；③调平（调节平衡螺母）；④称量（左物右码）；⑤读数。读数方法：物体质量 = 砝码总质量 + 游码左侧在标尺上对应的刻度值。

**易错分析：** 游码读的是左侧对应的刻度值，不是右侧。平衡螺母只能在称量前调节，称量中不能动。砝码用镊子夹取。

![配图](figures/物质科学-测量与物态变化/01-card.webp)

### 卡2

**提问：** 如何用托盘天平称取指定质量的食盐（如5g）？

**答案：** ①调平后，在右盘放5g砝码，游码归零（或配合游码凑出5g）。②左盘放称量纸，用药匙逐渐添加食盐，直到天平恢复平衡。③注意：砝码是"目标值"，先放好砝码再添加药品。

**易错分析：** 称量指定质量时，先放砝码再添加药品（与一般称量操作相反）。不能用手直接接触砝码，腐蚀性药品（如NaOH）要放在烧杯中称量。

![配图](figures/物质科学-测量与物态变化/02-card.webp)

### 卡3

**提问：** 使用天平时，哪些操作会影响测量精度？如何判断测量数据是否异常？

**答案：** 影响精度的操作：①游码未归零就调平；②砝码磨损或生锈；③物体和砝码放反；④天平台面不平。判断异常数据：多次测量取平均值，偏离平均值过大的数据为异常数据，应剔除后重新计算平均值。

**易错分析：** 磨损的砝码质量变小，按原标值读数会使测量结果偏大。生锈的砝码质量变大，测量结果偏小。

![配图](figures/物质科学-测量与物态变化/03-card.webp)

### 卡4

**提问：** 使用天平时游码未归零（如游码初始读数为0.2g）就调节平衡螺母使天平平衡，称量时物体在左盘某情况下读数总和为5.2g。物体的实际质量是多少？

**答案：** 实际质量 = 读数 - 初始游码读数 = 5.2g - 0.2g = 5.0g。因为初始游码已在0.2g位置却被"调平"了，相当于是先加了个"虚的质量"，所以最终读数偏大，需减去初始读数。

**易错分析：** 游码未归零就调平 → 测量值偏大 → 实际质量 = 测量值 - 初始游码读数。修正前提是知道初始读数是多少。

![配图](figures/物质科学-测量与物态变化/04-card.webp)

### 卡5

**提问：** 某同学在测量液体质量时，先测出空烧杯质量m₁，再将液体倒入烧杯测总质量m₂，则液体质量 = m₂ - m₁。这种操作有什么问题？

**答案：** 液体倒入烧杯时可能有残留（少量液体留在原容器中倒不干净），导致测出的液体质量偏小。改进方法：先测烧杯+液体的总质量，倒出部分液体后测剩余质量，用差减法计算倒出液体的质量。

**易错分析：** 残留误差是量取液体时的常见误差来源。差量法（倒出前后质量之差）可以避免残留误差。

![配图](figures/物质科学-测量与物态变化/05-card.webp)

### 卡6

**提问：** 天平使用中"物体和砝码位置放反（右物左码）"和"砝码磨损"分别对测量结果有什么影响？

**答案：** 放反时：正确关系为左=右+游码。放反后实际：右=左+游码，所以实际质量=砝码-游码（读数=砝码+游码，读数偏大）。砝码磨损时：砝码实际质量变小但仍按原标值读数，需要加更多砝码，读数偏大。

**易错分析：** 左物右码时：物体=砝码+游码。放反后：物体=砝码-游码。两种情况计算结果不同。

![配图](figures/物质科学-测量与物态变化/06-card.webp)

### 卡7

**提问：** 影响蒸发快慢的因素有哪些？分别举例说明。

**答案：** ①液体温度：温度越高蒸发越快（湿衣服夏天比冬天干得快）。②液体表面积：表面积越大蒸发越快（晾衣服要展开）。③表面空气流速：空气流动越快蒸发越快（有风比无风干得快）。④液体种类：不同液体蒸发快慢不同（酒精比水蒸发快）。

**易错分析：** 三个因素都是"越快"——温度越高、表面积越大、空气流速越大→蒸发越快。蒸发在任何温度下都能发生（沸腾需要沸点）。

![配图](figures/物质科学-测量与物态变化/07-card.webp)

### 卡8

**提问：** 用酒精棉球擦拭温度计玻璃泡后，温度计的示数会怎样变化？为什么？

**答案：** 温度计示数先下降后上升，最终回到室温。原因：酒精蒸发时需要吸热（汽化吸热），导致温度计玻璃泡周围温度降低，示数下降。酒精蒸发完后，温度计从周围环境吸热，示数逐渐回升到室温。

**易错分析：** 酒精蒸发吸热有制冷作用，所以温度计示数会先降后升。这是液体蒸发吸热导致温度降低的典型实验。

![配图](figures/物质科学-测量与物态变化/08-card.webp)

### 卡9

**提问：** 升华和凝华的定义是什么？各举两个生活中的例子。

**答案：** 升华：固态直接变成气态（吸热）。如：干冰（固态CO₂）升华、樟脑丸变小、冰冻衣服变干。凝华：气态直接变成固态（放热）。如：窗玻璃上的冰花、霜的形成、灯泡内壁变黑（钨丝升华后又凝华）。

**易错分析：** 升华和凝华都"不经过液态"，这是关键区别。吸热的物态变化：熔化、汽化、升华（都带"化"）。放热的物态变化：凝固、液化、凝华。

![配图](figures/物质科学-测量与物态变化/09-card.webp)

### 卡10

**提问：** 冬天窗玻璃上的冰花通常出现在室内一侧还是室外一侧？在冰上撒盐会发生什么现象？

**答案：** 冰花出现在室内一侧。因为室内温暖的水蒸气遇到冰冷的窗玻璃，直接凝华成小冰晶。撒盐在冰上会使冰加速熔化，因为盐水的凝固点（约-20℃）比纯水（0℃）低，使冰在低于0℃时也能熔化。

**易错分析：** 冰花在室内一侧（室内水蒸气→冷凝华）。撒盐降低凝固点/熔点，这是冬天路面撒盐融雪的原理。

![配图](figures/物质科学-测量与物态变化/10-card.webp)

### 卡11

**提问：** 在冰块上撒盐会看到什么现象？杯子外壁出现的白霜是从哪里来的？

**答案：** 冰块上撒盐后冰加速熔化（盐降低熔点）。杯子外壁的白霜是空气中温暖的水蒸气遇到冰冷的杯壁直接凝华成的小冰晶（气态→固态，凝华），不是杯子里的水渗出来的，也不是凝固形成的。

**易错分析：** 白霜的形成是凝华（气→固），不是凝固（液→固）也不是液化（气→液）。空气中的水蒸气是看不见的，白霜不是"白气"。

![配图](figures/物质科学-测量与物态变化/11-card.webp)

### 卡12

**提问：** 用量筒测量液体体积时，俯视读数和仰视读数各有什么影响？

**答案：** 俯视读数偏大（读出的数值比实际液体体积大），仰视读数偏小（读出的数值比实际液体体积小）。正确方法：视线与凹液面最低处相平。

**易错分析：** 读取已有液体：俯视→偏大，仰视→偏小。但量取指定体积液体时情况相反：俯视→实际取少了，仰视→实际取多了。要区分"读取"和"量取"两种场景。

![配图](figures/物质科学-测量与物态变化/12-card.webp)

### 卡13

**提问：** 使用实验室温度计测量水的温度时，哪些操作是错误的？为什么？

**答案：** 错误操作：①温度计的玻璃泡碰到容器底或容器壁（测的是容器温度不是水温）；②读数时把温度计从水中取出来读（离开液体后示数会变）；③没等示数稳定就读数（温度变化还没有完成）。正确操作：泡全浸、不碰壁、示数稳、不离液。

**易错分析：** 体温计有缩口可以拿出来读，普通温度计不可以。体温计用前要甩几下让水银回到玻璃泡中。

![配图](figures/物质科学-测量与物态变化/13-card.webp)

### 卡14

**提问：** 晶体和非晶体在熔化图像上有什么区别？

**答案：** 晶体熔化图像有水平线段（熔化过程中温度保持不变），如海波熔化时温度保持在48℃不变、冰在0℃熔化。非晶体熔化图像没有水平线段，温度持续上升，如蜡熔化时温度不断升高。晶体熔化图像分三段：固态升温→固液共存（平台）→液态升温。

**易错分析：** 晶体熔化时虽温度不变但需要持续吸热。水平线段代表"固液共存状态"。区分晶体和非晶体的关键：看图像中是否有水平线段。

![配图](figures/物质科学-测量与物态变化/14-card.webp)

### 卡15

**提问：** 水沸腾时，如果继续加热，水的温度还会升高吗？为什么？

**答案：** 不会。水沸腾后继续加热，温度保持不变（在标准大气压下保持在100℃）。因为沸腾时水吸收的热量全部用于使水从液态变成气态（汽化热），不用于升高温度。沸点受气压影响：气压越高沸点越高（高压锅），气压越低沸点越低（高原）。

**易错分析：** 水沸腾后无论用多大火加热，温度都不会超过沸点。高压锅是"提高沸点"让水在更高温度下沸腾，从而更快煮熟食物。

![配图](figures/物质科学-测量与物态变化/15-card.webp)

### 卡16

**提问：** 蒸发和沸腾有哪些相同点和不同点？

**答案：** 相同点：都是汽化现象，都需要吸热。不同点：①蒸发在任何温度下发生，沸腾需要达到沸点；②蒸发只在液体表面进行，沸腾在表面和内部同时进行；③蒸发缓慢，沸腾剧烈；④蒸发有制冷作用（吸热降温），沸腾时温度不变。

**易错分析：** 蒸发和沸腾的本质都是液态→气态（汽化）。蒸发在液体表面进行，可以在任何温度下发生。沸腾必须在沸点且需要持续加热。

![配图](figures/物质科学-测量与物态变化/16-card.webp)

### 卡17

**提问：** 冬天呼出的"白气"和打开冰箱看到的"白气"是什么？是怎样形成的？

**答案：** 这些"白气"不是水蒸气（水蒸气是无色透明的气体，肉眼看不见），而是水蒸气遇冷液化形成的小水珠（液态）。冬天呼出的水蒸气遇冷空气液化成小水珠。冰箱门冒出的"白气"是冰箱外的水蒸气遇到冷气液化成小水珠。

**易错分析：** "白气"不是水蒸气！水蒸气看不见，"白气"是小水珠——这是最常考的概念混淆。液化放热、汽化吸热。

![配图](figures/物质科学-测量与物态变化/17-card.webp)
