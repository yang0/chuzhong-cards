# 生命科学 生殖与发育

## 基本信息

- 科目：科学
- 年级：七年级下册
- 教材：浙教版《科学》
- 章节：代代相传的生命（生殖、发育、微生物）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析
- 来源：错题 Y7S-WRONG-001 至 017

## 单元纲要

本单元涵盖动物和植物的生殖与发育，以及微生物（真菌、细菌、病毒）的基本特征。核心易混淆点：器官 vs 组织分类、嫁接后果实性状归属、顶端优势原理、病毒与细菌/真菌的区别。

## Anki 记忆卡

### 卡1

**提问：** 青蛙从受精卵发育成成体，经历了哪些基本变化？

**答案：** 经历了细胞分裂（数量增加）、细胞生长（体积增大）和细胞分化（形成不同组织和器官）。

**易错分析：** 青蛙发育过程中①②③全有，不能只选分裂和生长而漏掉细胞分化。

![配图](figures/生命科学-生殖与发育/01-card.webp)

### 卡2

**提问：** 心脏、心肌、肺、血液、骨骼肌、小肠、肾脏、熊猫，分别属于哪个生命结构层次？

**答案：** 器官：心脏、肺、小肠、肾脏；组织：心肌、血液、骨骼肌；个体：熊猫。

**易错分析：** 肌肉名称中带"肌"不一定就是器官；血液在初中生物中归为结缔组织。

![配图](figures/生命科学-生殖与发育/02-card.webp)

### 卡3

**提问：** 受精过程发生在哪里？受精卵分裂后附着在什么部位？胎儿通过什么结构从母体获得氧气和营养物质？

**答案：** 受精发生在输卵管。受精卵分裂形成胚胎后附着在子宫内膜上。胎儿通过胎盘和脐带从母体获得氧气和营养物质。

**易错分析：** 胎盘是母体和胎儿物质交换的重要结构，脐带连接胎儿和胎盘，两者常一起考。

![配图](figures/生命科学-生殖与发育/03-card.webp)

### 卡4

**提问：** 胎儿从母体获得营养物质的途径是怎样的？

**答案：** 母体 → 胎盘 → 脐带 → 胎儿。营养物质来自母体，先通过胎盘完成物质交换，再经脐带运送给胎儿。

**易错分析：** 顺序不能颠倒，不是"脐带 → 胎盘"。

![配图](figures/生命科学-生殖与发育/04-card.webp)

### 卡5

**提问：** 龙凤胎（一男一女双胞胎）是如何形成的？

**答案：** 由两个卵细胞分别与两个精子结合，形成两个受精卵发育而来（异卵双胞胎）。

**易错分析：** 一个受精卵分裂形成的是同卵双胞胎，遗传物质相同，性别一般相同。

![配图](figures/生命科学-生殖与发育/05-card.webp)

### 卡6

**提问：** 两栖动物幼体和成体的呼吸方式各是什么？生殖和发育离不开什么？

**答案：** 幼体用鳃呼吸，成体主要用肺呼吸（皮肤辅助）。体外受精，生殖和幼体发育离不开水环境。

**易错分析：** 两栖动物不是体内受精。成体不是只用肺呼吸，皮肤也有辅助呼吸作用。

![配图](figures/生命科学-生殖与发育/06-card.webp)

### 卡7

**提问：** 种子中胚的组成部分有哪些？胚乳属于胚吗？

**答案：** 胚由胚芽、胚轴、胚根和子叶组成。胚乳是储存营养物质的结构，不属于胚。

**易错分析：** 胚乳不属于胚，它是提供营养的结构。胚是种子的核心部分。

![配图](figures/生命科学-生殖与发育/07-card.webp)

### 卡8

**提问：** 甜玉米的胚与老玉米的胚乳结合形成的种子播种后，植株性状由什么决定？

**答案：** 胚发育成新植物体，主要决定植株本身的遗传性状。胚乳主要储存营养，不发育成植株。

**易错分析：** 植株性状由胚的遗传来源决定，不是由胚乳决定。胚乳的作用是储存营养供胚发育。

![配图](figures/生命科学-生殖与发育/08-card.webp)

### 卡9

**提问：** 将水蜜桃枝条嫁接到毛桃砧木上，接穗结出的果实是什么？为什么？

**答案：** 接穗上只结水蜜桃。嫁接属于无性繁殖，果实性状由接穗决定，砧木主要提供根系和水分、无机盐。

**易错分析：** 嫁接后接穗上的果实性状不变，不会变成砧木的品种。接穗和砧木都是同一物种的不同品种才能嫁接成功。

![配图](figures/生命科学-生殖与发育/09-card.webp)

### 卡10

**提问：** 果农为什么要在果树生长旺季剪去过密的枝条和顶芽？

**答案：** 剪去过密枝条可以改善通风透光，减少营养消耗。剪去顶芽利用了顶端优势原理——去除顶芽后，侧芽受到的生长素抑制减弱，侧枝增多，有利于果树多结果实。

**易错分析：** 修剪顶芽的目的是促进侧枝生长让果树多结果，不是为了让树长得更高。顶芽产生的生长素向下运输会抑制侧芽发育。

![配图](figures/生命科学-生殖与发育/10-card.webp)

### 卡11

**提问：** 摘心（去顶芽）为什么能促进侧芽发育？原理是什么？

**答案：** 摘心即去除植株的顶芽，打破了顶端优势。原理是：顶芽产生的生长素向下运输使侧芽部位生长素浓度过高而抑制其生长；去除顶芽后生长素来源减少，侧芽的抑制作用被解除，侧芽开始发育成侧枝。

**易错分析：** 摘心是去顶芽，不是去侧芽或叶子。生长素浓度过高反而抑制生长（两重性）。

![配图](figures/生命科学-生殖与发育/11-card.webp)

### 卡12

**提问：** 扦插时为什么要把插条下端的叶片去掉，并把下端剪成斜口？

**答案：** ①去掉叶片是为了减少水分蒸腾（刚插下去的枝条还没有根，吸水能力弱）。②斜口可以增加切口面积，有利于吸收水分和促进生根。

**易错分析：** 去掉叶片主要是为了减少蒸腾作用（水分散失），不是因为有病菌。斜口是为了增加吸水面积和生根面积。

![配图](figures/生命科学-生殖与发育/12-card.webp)

### 卡13

**提问：** 蘑菇属于哪类生物？它的各部分结构名称分别是什么？

**答案：** 蘑菇属于真菌（真核生物）。地上部分是子实体（菌盖+菌柄），地下部分是营养菌丝。菌盖下方有菌褶，菌褶上会产生孢子用于繁殖。

**易错分析：** 蘑菇是真菌，不是植物（不能光合作用）。日常吃的部分是子实体，真正的植物体是地下的菌丝体。蘑菇靠孢子繁殖。

![配图](figures/生命科学-生殖与发育/13-card.webp)

### 卡14

**提问：** 病毒在结构上有哪些特点？为什么病毒必须寄生在活细胞内？

**答案：** 病毒没有细胞结构，由蛋白质外壳和内部的遗传物质（DNA或RNA）组成。病毒自身没有代谢系统和核糖体，不能独立合成蛋白质，必须寄生在活细胞内，利用宿主细胞的结构和物质来繁殖。

**易错分析：** 病毒没有细胞结构，不是原核生物也不是真核生物。病毒在生物体外以结晶状态存在，没有生命活动。

![配图](figures/生命科学-生殖与发育/14-card.webp)

### 卡15

**提问：** 酵母菌和细菌在细胞结构上有什么根本区别？

**答案：** 酵母菌有完整的细胞结构（细胞壁、细胞膜、细胞质、细胞核），是真核生物。细菌有细胞壁、细胞膜、细胞质，但无成形的细胞核，是原核生物。酵母菌以出芽繁殖为主，细菌以分裂繁殖为主。

**易错分析：** 酵母菌是真菌（真核生物），不是细菌。有无成形细胞核是两者的根本区别。

![配图](figures/生命科学-生殖与发育/15-card.webp)

### 卡16

**提问：** 埃博拉病毒在结构上有什么特点？属于哪类病毒？

**答案：** 埃博拉病毒没有细胞结构，呈长丝状（丝状病毒），由蛋白质外壳和内部的RNA（遗传物质）组成。它属于RNA病毒，必须寄生在活细胞内才能繁殖。

**易错分析：** 埃博拉病毒是丝状病毒（外形为长丝状），遗传物质是RNA不是DNA。病毒必须寄生在活细胞中，离开宿主会以结晶状态存在。

![配图](figures/生命科学-生殖与发育/16-card.webp)

### 卡17

**提问：** 在"酵母菌发酵"实验中，变量是什么？如何控制其他变量使实验公平？

**答案：** 实验变量是温度（一组放温水、一组放冷水）。控制方法：两组装置中葡萄糖的量和浓度相同、酵母菌的量相同、反应时间相同、容器大小相同，只有温度不同。通过比较气球膨胀程度来判断发酵速度（酵母菌发酵产生CO₂）。

**易错分析：** 对照实验中只能有一个变量（温度），其他条件都要保持相同。酵母菌发酵产生二氧化碳，二氧化碳越多气球越大。这是典型的控制变量实验。

![配图](figures/生命科学-生殖与发育/17-card.webp)

### 卡18

**提问：** 有性繁殖和无性繁殖的根本区别是什么？各有什么优点？

**答案：** 根本区别：有性繁殖经过两性生殖细胞（精子和卵细胞）的结合，无性繁殖不经过两性生殖细胞的结合。有性繁殖优点：后代具有更大的变异性和适应性。无性繁殖优点：繁殖速度快，能保持母体的优良性状。

**易错分析：** 区别在于是否"经过两性生殖细胞的结合"，不是看有没有种子。无性繁殖产生的后代遗传物质与母体完全相同（克隆）。

![配图](figures/生命科学-生殖与发育/18-card.webp)

### 卡19

**提问：** 植物开花后，传粉和受精的过程是怎样的？果实由花的哪部分发育而来？

**答案：** ①传粉：花粉从花药落到柱头上。②花粉萌发形成花粉管，穿过花柱进入子房到达胚珠。③精子通过花粉管与卵细胞结合形成受精卵。④受精后，子房发育成果实，胚珠发育成种子，受精卵发育成胚。

**易错分析：** 受精发生在胚珠内（不是子房内）。子房→果实，子房壁→果皮，胚珠→种子。

![配图](figures/生命科学-生殖与发育/19-card.webp)

### 卡20

**提问：** 完全变态发育和不完全变态发育有什么区别？各举两例。

**答案：** 完全变态：卵 → 幼虫 → 蛹 → 成虫（如家蚕、苍蝇、蝴蝶、蜜蜂、果蝇）。不完全变态：卵 → 若虫 → 成虫（如蝗虫、蟋蟀、蜻蜓、蝉）。区别在于是否有"蛹"期。

**易错分析：** 是否有蛹期是关键区别。苍蝇、家蚕是完全变态；蝗虫是不完全变态。青蛙是两栖动物不是昆虫。

![配图](figures/生命科学-生殖与发育/20-card.webp)

### 卡21

**提问：** 种子萌发需要哪些条件？

**答案：** 外界条件（环境条件）：一定的水分、充足的空气（氧气）、适宜的温度。自身条件：胚是完整的且有活力、不在休眠期、有足够的营养储备。

**易错分析：** 种子萌发不一定需要阳光和土壤（无土栽培也能萌发）。三个外界条件缺一不可。

![配图](figures/生命科学-生殖与发育/21-card.webp)

### 卡22

**提问：** 玉米种子和菜豆种子在结构上有什么区别？营养储存在哪里？

**答案：** 玉米（单子叶植物）种子：种皮+胚乳+胚（一片子叶），营养储存在胚乳中。菜豆（双子叶植物）种子：种皮+胚（两片子叶），无胚乳，营养储存在子叶中。共同点：都有种皮和胚（胚芽、胚轴、胚根、子叶）。

**易错分析：** 玉米的子叶不储存营养（起转运作用，营养在胚乳）。菜豆没有胚乳（营养在肥厚子叶中）。两种种子的营养来源不同。

![配图](figures/生命科学-生殖与发育/22-card.webp)
