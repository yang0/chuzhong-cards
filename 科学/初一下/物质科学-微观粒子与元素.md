## Anki 记忆卡

### 卡1

**提问：** 温度升高时，分子的运动速度如何变化？请用扩散现象举例说明。

**答案：** 温度升高时，分子的运动速度加快（温度越高，分子热运动越剧烈）。如：热水中的糖扩散比冷水中快；一滴红墨水滴入热水比滴入冷水中扩散得更快。这说明分子的运动速度与温度有关。

**易错分析：** 温度越高分子运动越快，两者是正相关关系。扩散现象是分子运动的结果，可以用来间接证明分子在运动。

### 卡2

**提问：** 浓氨水和酚酞溶液不接触，为什么酚酞溶液会变红？这说明了什么？

**答案：** 浓氨水挥发出的氨分子在空气中不断运动，遇到酚酞溶液使它变红。这说明了：①分子在不断地做无规则运动（扩散）；②氨水具有挥发性，挥发出的氨分子能通过空气扩散到其他地方。

**易错分析：** 浓氨水是混合物，氨气（NH₃）是它的溶质。氨分子从浓氨水中挥发出来，不是浓氨水自己在运动。这证明了分子在不断地运动。

![配图](figures/物质科学-微观粒子与元素/02-card.webp)

### 卡3

**提问：** "一个甲醛分子由一个碳原子、两个氢原子和一个氧原子构成"，这句话中甲醛由几种元素组成？一个甲醛分子中原子的总数是多少？

**答案：** 甲醛由三种元素组成：碳（C）、氢（H）、氧（O）。一个甲醛分子的化学式为 CH₂O，原子总数为 1+2+1 = 4。宏观上用"元素组成"，微观上用"分子/原子构成"。

**易错分析：** 分子由原子构成（微观），物质由元素组成（宏观）。分子中原子的个数 = 各元素下标之和。元素只讲种类不讲个数。

![配图](figures/物质科学-微观粒子与元素/03-card.webp)

### 卡4

**提问：** "只含有一种元素的物质一定是单质"这句话对吗？为什么？

**答案：** 不对。反例：氧气（O₂）和臭氧（O₃）混合在一起，虽然只含有氧元素，但这是两种不同的物质，属于混合物而不是单质。必须满足两个条件才是单质：①只由一种元素组成；②必须是纯净物（一种物质）。

**易错分析：** 单质需要同时满足"一种元素"+"纯净物"两个条件。由同种元素组成的物质不一定是纯净物（如O₂和O₃的混合物）。单质一定是纯净物。

![配图](figures/物质科学-微观粒子与元素/04-card.webp)

### 卡5

**提问：** 物质如何按组成进行分类？请用树状图说明纯净物、混合物、单质、化合物的关系。

**答案：** 第一步：按组成物质种类 → 纯净物（一种物质）或混合物（多种物质）。第二步：纯净物按组成元素种类 → 单质（一种元素）或化合物（多种元素）。化合物中由两种元素组成且其中一种是氧元素的叫氧化物（如CO₂、H₂O、Fe₃O₄）。

**易错分析：** 纯净物和混合物的根本区别是"是否由一种物质组成"（不是"是否由一种元素组成"）。由同种元素组成的纯净物才是单质。

![配图](figures/物质科学-微观粒子与元素/05-card.webp)

### 卡6

**提问：** 从微观和宏观两个角度，二氧化碳（CO₂）分别由什么构成/组成？

**答案：** 宏观：二氧化碳由碳元素和氧元素组成。微观：二氧化碳由二氧化碳分子构成；一个二氧化碳分子由一个碳原子和两个氧原子构成。

**易错分析：** 宏观用"组成+元素"（CO₂由碳元素和氧元素组成），微观用"构成+分子/原子"（CO₂由CO₂分子构成）。不能说"CO₂由碳原子和氧原子构成"——碳原子和氧原子在CO₂中已经结合成分子，不存在独立的碳原子和氧原子。

![配图](figures/物质科学-微观粒子与元素/06-card.webp)

### 卡7

**提问：** 单质和化合物是什么关系？它们是并列关系还是包含关系？

**答案：** 单质和化合物是并列关系，它们都包含在"纯净物"中。即：纯净物分为单质和化合物两大类，两者是兄弟关系（并列），不是父子关系。单质不是包含在化合物中，化合物也不是包含在单质中。

**易错分析：** 纯净物 ⊃ 单质；纯净物 ⊃ 化合物。单质和化合物是并列的，不能把单质归入化合物，也不能把化合物归入单质。

![配图](figures/物质科学-微观粒子与元素/07-card.webp)

### 卡8

**提问：** 请写出水、二氧化碳、氧气、氮气、氯化钠、甲烷的化学式。

**答案：** 水 H₂O、二氧化碳 CO₂、氧气 O₂、氮气 N₂、氯化钠 NaCl、甲烷 CH₄。

**易错分析：** 氧气和氮气是双原子分子，必须写 O₂ 和 N₂，不能写 O 和 N。氯化钠是 NaCl（离子化合物，1:1），甲烷是有机物最简单的烃。

![配图](figures/物质科学-微观粒子与元素/08-card.webp)

### 卡9

**提问：** 请写出钠离子、氯离子、钙离子、氧离子、镁离子、铝离子、硫酸根离子、氢氧根离子的符号。

**答案：** Na⁺、Cl⁻、Ca²⁺、O²⁻、Mg²⁺、Al³⁺、SO₄²⁻、OH⁻。

**易错分析：** 离子符号写法：数字在前正负号在后（Ca²⁺不是Ca⁺²），电荷数1省略不写。硫酸根和氢氧根是原子团离子（含多种原子的整体带电粒子）。

![配图](figures/物质科学-微观粒子与元素/09-card.webp)

### 卡10

**提问：** 从化学式 H₂O 中，你可以获得哪些信息？（宏观和微观各至少答两点）

**答案：** 宏观：①表示水这种物质；②水由氢元素和氧元素组成。微观：③表示一个水分子；④一个水分子由两个氢原子和一个氧原子构成；⑤水分子中H原子和O原子的个数比为2:1。

**易错分析：** 化学式既有宏观意义也有微观意义。H₂O中数字2的含义：表示一个水分子中含有2个氢原子（右下角的数字）。

![配图](figures/物质科学-微观粒子与元素/10-card.webp)

### 卡11

**提问：** 已知氢元素通常为+1价、氧元素通常为-2价，求 H₂SO₄ 中硫元素的化合价。

**答案：** 设 S 化合价为 x。（+1）×2 + x +（-2）×4 = 0 → 2 + x - 8 = 0 → x = +6。所以 H₂SO₄ 中硫元素为 +6 价。

**易错分析：** 计算化合价时记住：化合物中正负化合价代数和为0（化合物不显电性）。单质中元素化合价为0。氧在化合物中通常是-2价（过氧化物H₂O₂中为-1）。

![配图](figures/物质科学-微观粒子与元素/11-card.webp)

### 卡12

**提问：** 硫元素在 H₂S、S、SO₂、H₂SO₃、SO₃、H₂SO₄ 中分别是什么化合价？为什么硫有多种化合价？

**答案：** H₂S 中 S：-2价；S单质：0价；SO₂中 S：+4价；H₂SO₃中 S：+4价；SO₃中 S：+6价；H₂SO₄中 S：+6价。硫有多种化合价，因为硫原子最外层6个电子，可以得到电子显负价（-2），也可以失去不同数量的电子显正价（+4、+6）。

**易错分析：** 同种元素在不同化合物中可显示不同化合价（硫有-2、0、+4、+6价）。单质化合价为0。化合价不是固定不变的。

![配图](figures/物质科学-微观粒子与元素/12-card.webp)

### 卡13

**提问：** 铬（Cr）是某种不锈钢的重要成分。在 Cr₂O₃ 和 CrO₃ 中，铬分别是什么化合价？

**答案：** Cr₂O₃：2x + (-2)×3 = 0 → 2x = 6 → x = +3价。CrO₃：x + (-2)×3 = 0 → x = +6价。所以 Cr₂O₃ 中 Cr 为 +3 价，CrO₃ 中 Cr 为 +6 价。

**易错分析：** 铬是可变价态元素（常见+3和+6价）。计算时要将下标数字与化合价相乘。

![配图](figures/物质科学-微观粒子与元素/13-card.webp)

### 卡14

**提问：** 氮元素在 NH₃、N₂、NO、NO₂、HNO₃ 中分别是什么化合价？

**答案：** NH₃：x + (+1)×3 = 0 → x = -3价。N₂：0价（单质）。NO：x + (-2) = 0 → x = +2价。NO₂：x + (-2)×2 = 0 → x = +4价。HNO₃：(+1) + x + (-2)×3 = 0 → x = +5价。

**易错分析：** NH₃ 中氮是 -3 价（不是常见的+5价），这是常考易错点。氮元素从-3到+5有多种价态，是典型的可变价元素。

![配图](figures/物质科学-微观粒子与元素/14-card.webp)

### 卡15

**提问：** 化学符号中的数字"2"在不同位置上各代表什么含义？举例说明。

**答案：** ①右下角：表示一个分子中该原子的个数（如 H₂O 中的2表示一个水分子中有2个氢原子）。②右上角：表示离子带2个单位电荷（如 Ca²⁺）。③前面：表示2个粒子（如 2H 表示2个氢原子，2CO₂ 表示2个二氧化碳分子）。④正上方：表示化合价为+2价（如 Mg表示镁元素化合价为+2）。

**易错分析：** 四个位置的"2"含义都不同：右下角（原子个数）、右上角（电荷数）、前面（粒子个数）、正上方（化合价）。

![配图](figures/物质科学-微观粒子与元素/15-card.webp)

### 卡16

**提问：** 分子、原子和离子有什么区别？三者之间有什么关系？

**答案：** 分子：由原子构成，在化学变化中可再分。原子：化学变化中的最小粒子，可直接构成物质（如金属）。离子：带电的原子或原子团，由原子得失电子形成。关系：原子可结合成分子；原子得失电子变成离子。构成物质的三种基本粒子：分子、原子、离子。

**易错分析：** 三种粒子都能直接构成物质。铁（Fe）由原子构成、水（H₂O）由分子构成、氯化钠（NaCl）由离子构成。

![配图](figures/物质科学-微观粒子与元素/16-card.webp)

### 卡17

**提问：** 已知某原子的质子数为Z，中子数为N，则它的核外电子数是多少？质量数是多少？相对原子质量大约是多少？

**答案：** 核外电子数 = Z（原子呈电中性，质子数=电子数）。质量数 = Z + N。相对原子质量 ≈ 质量数 = Z + N（电子质量极小，不计入）。

**易错分析：** 质量数是整数（质子数+中子数），相对原子质量是近似值。核外电子数未必等于中子数。氢-1原子有1个质子、0个中子。

![配图](figures/物质科学-微观粒子与元素/17-card.webp)

### 卡18

**提问：** 离子是怎样形成的？以 NaCl 为例说明钠原子和氯原子如何变成离子。

**答案：** 钠原子（Na）最外层有1个电子，容易失去这个电子变成带1个正电荷的钠离子（Na⁺ — e⁻ → Na⁺）。氯原子（Cl）最外层有7个电子，容易得到1个电子变成带1个负电荷的氯离子（Cl + e⁻ → Cl⁻）。Na⁺和Cl⁻通过静电引力结合成NaCl。

**易错分析：** 金属原子容易失电子变阳离子，非金属原子容易得电子变阴离子。得失电子后，质子数不变（仍属同一元素），电子数改变（所以带电）。

![配图](figures/物质科学-微观粒子与元素/18-card.webp)

### 卡19

**提问：** 如何用"两步法"对常见物质进行分类？（纯净物/混合物 → 单质/化合物）

**答案：** 第一步：按组成物质是否单一 → 纯净物（一种物质）或混合物（多种物质混合）。第二步：纯净物按组成元素是否单一 → 单质（一种元素）或化合物（多种元素）。如：空气是混合物（N₂、O₂等混合），蒸馏水是纯净物中的化合物，铁粉是纯净物中的单质。

**易错分析：** 两步法要按顺序：先判断纯净/混合，再判断单质/化合物。不能跳过第一步直接判断单质/化合物。

![配图](figures/物质科学-微观粒子与元素/19-card.webp)

### 卡20

**提问：** 化学符号中各位置上的数字分别代表什么含义？（以 3SO₄²⁻ 和 Al³⁺ 为例）

**答案：** ①符号前面数字：表示粒子个数（3SO₄²⁻ 中3表示3个硫酸根离子）。②右上角数字：表示离子所带电荷数（Al³⁺ 的3+表示带3个单位正电荷，SO₄²⁻ 的2-表示带2个单位负电荷）。③右下角数字：表示一个粒子中某原子的个数（SO₄²⁻ 中4表示一个硫酸根中有4个氧原子）。④正上方数字：表示该元素的化合价。

**易错分析：** 区分右上角（离子电荷数）和正上方（化合价）的写法不同。Al³⁺右上角是离子符号，Al正上方是化合价标法。前面数字表示粒子个数。

![配图](figures/物质科学-微观粒子与元素/20-card.webp)
