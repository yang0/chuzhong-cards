## 单元纲要

人教版七下生物有大量人体生理内容（人的生殖、营养、呼吸、循环、排泄），这些在现有浙教版错题中未覆盖。本文件补充核心易错知识点。

## Anki 记忆卡

### 卡1

**提问：** 人体所需的六大营养物质是什么？哪些能提供能量？

**答案：** 六大营养物质：糖类、脂肪、蛋白质、水、无机盐、维生素。其中糖类、脂肪、蛋白质能提供能量（有机物）；水、无机盐、维生素不能提供能量。

**易错分析：** 水、无机盐、维生素不供能！它们是调节物质。

![配图](figures/人教版补充/01-card.webp)

### 卡2

**提问：** 消化系统由哪两部分组成？消化和吸收的主要场所是哪里？

**答案：** 消化系统由消化道（口、咽、食道、胃、小肠、大肠、肛门）和消化腺（唾液腺、胃腺、肝脏、胰腺、肠腺）组成。消化和吸收的主要场所是小肠。

**易错分析：** 肝脏是最大的消化腺，分泌胆汁（不含消化酶，但能乳化脂肪）。胃是初步消化蛋白质的场所。

![配图](figures/人教版补充/02-card.webp)

### 卡3

**提问：** 小肠有哪些结构特点有利于消化和吸收？

**答案：** ①小肠很长（5-6米）；②内表面有皱襞和小肠绒毛，大大增加表面积；③小肠绒毛内有丰富的毛细血管和毛细淋巴管；④小肠内有多种消化液（肠液、胰液、胆汁）。

**易错分析：** 皱襞和小肠绒毛都是为了增大表面积，但皱襞是宏观褶皱，绒毛是微观突起。

![配图](figures/人教版补充/03-card.webp)

### 卡4

**提问：** 三大有机物（糖类、蛋白质、脂肪）消化后的最终产物分别是什么？

**答案：** 糖类（淀粉）→葡萄糖；蛋白质→氨基酸；脂肪→甘油+脂肪酸。

**易错分析：** 淀粉在口腔开始消化（唾液淀粉酶），蛋白质在胃开始消化（胃蛋白酶），脂肪在小肠才开始消化（胆汁乳化+胰脂肪酶）。

![配图](figures/人教版补充/04-card.webp)

### 卡5

**提问：** 呼吸系统由哪些器官组成？气体交换的场所是哪里？

**答案：** 呼吸系统由呼吸道（鼻、咽、喉、气管、支气管）和肺组成。气体交换的场所是肺（具体在肺泡处）。

**易错分析：** 呼吸道是气体进出的通道，能使空气温暖、湿润、清洁，但气体交换只在肺泡进行。

![配图](figures/人教版补充/05-card.webp)

### 卡6

**提问：** 肺泡适合气体交换的结构特点有哪些？

**答案：** ①肺泡数量多；②肺泡壁和毛细血管壁都很薄，只由一层上皮细胞构成；③肺泡外面包绕着丰富的毛细血管。

**易错分析：** 肺泡壁和毛细血管壁都只有一层细胞——这是高效气体交换的关键。

![配图](figures/人教版补充/06-card.webp)

### 卡7

**提问：** 人体呼吸时，吸气和呼气分别是什么肌肉在作用？膈肌和肋间肌怎样变化？

**答案：** 吸气：膈肌收缩（膈顶下降）、肋间肌收缩（肋骨上提）→胸腔容积增大→肺内气压降低→气体进入。呼气：膈肌舒张（膈顶上升）、肋间肌舒张（肋骨下降）→胸腔容积缩小→肺内气压升高→气体排出。

**易错分析：** 吸气时膈肌是"收缩"不是"舒张"，膈顶是"下降"不是"上升"。

![配图](figures/人教版补充/07-card.webp)

### 卡8

**提问：** 血液由哪些成分组成？各自的功能是什么？

**答案：** 血液由血浆和血细胞组成。血浆：运输血细胞、营养物质和废物。红细胞：运输氧气（含血红蛋白）。白细胞：吞噬病菌，防御保护。血小板：止血和凝血。

**易错分析：** 红细胞没有细胞核！红细胞运输氧气靠血红蛋白，不是靠细胞本身。

![配图](figures/人教版补充/08-card.webp)

### 卡9

**提问：** 三种血管（动脉、静脉、毛细血管）如何区分？

**答案：** 动脉：管壁厚、弹性大、血流快，把血液从心脏送往全身。静脉：管壁薄、弹性小、血流慢，把血液从全身送回心脏（四肢静脉内有静脉瓣防止倒流）。毛细血管：连通最小动脉和静脉，管壁仅一层细胞，血流最慢，是物质交换场所。

**易错分析：** 动脉不一定都流动脉血（肺动脉流静脉血），静脉不一定都流静脉血（肺静脉流动脉血）。区分动脉和静脉看血流方向！

![配图](figures/人教版补充/09-card.webp)

### 卡10

**提问：** 体循环和肺循环的路径分别是什么？血液成分怎样变化？

**答案：** 体循环：左心室→主动脉→全身毛细血管→上下腔静脉→右心房。在全身毛细血管处，动脉血变静脉血（氧气给组织细胞）。肺循环：右心室→肺动脉→肺部毛细血管→肺静脉→左心房。在肺部毛细血管处，静脉血变动脉血（获得氧气）。

**易错分析：** 体循环和肺循环是同时进行的，在心脏处汇合。不是先体循环后肺循环！

![配图](figures/人教版补充/10-card.webp)

### 卡11

**提问：** 心脏四个腔的名称是什么？哪个腔壁最厚？为什么？

**答案：** 左心房、左心室、右心房、右心室。左心室壁最厚，因为左心室把血液泵向全身，需要的压力最大。

**易错分析：** 心脏左右不通（防止动脉血和静脉血混合），同侧心房和心室相通。

![配图](figures/人教版补充/11-card.webp)

### 卡12

**提问：** 泌尿系统由哪些器官组成？尿液形成的基本单位是什么？

**答案：** 泌尿系统由肾脏、输尿管、膀胱、尿道组成。肾脏是形成尿液的器官，其基本单位是肾单位（包括肾小球和肾小囊、肾小管）。

**易错分析：** 膀胱只储存尿液，不形成尿液！形成尿液在肾脏。

![配图](figures/人教版补充/12-card.webp)

### 卡13

**提问：** 尿液是怎样形成的？肾小球和肾小管各起什么作用？

**答案：** ①肾小球的过滤作用：血液流经肾小球时，除血细胞和大分子蛋白质外，其他物质（水、无机盐、葡萄糖、尿素等）被滤过到肾小囊形成原尿。②肾小管的重吸收作用：原尿流经肾小管时，全部葡萄糖、大部分水、部分无机盐被重新吸收回血液，剩下的形成尿液。

**易错分析：** 原尿中有葡萄糖，正常尿液中无葡萄糖（被全部重吸收了）。如果尿液中有葡萄糖，可能是糖尿病或肾小管病变。

![配图](figures/人教版补充/13-card.webp)

### 卡14

**提问：** 男性和女性的主要生殖器官分别是什么？各产生什么？

**答案：** 男性主要生殖器官是睾丸，产生精子和分泌雄性激素。女性主要生殖器官是卵巢，产生卵细胞和分泌雌性激素。

**易错分析：** 输卵管是受精场所（不是卵巢），子宫是胚胎发育场所（不是受精场所）。

![配图](figures/人教版补充/14-card.webp)

### 卡15

**提问：** 青春期的身体变化有哪些突出特征？

**答案：** ①身高突增（青春期发育的显著特点）；②性器官迅速发育和成熟；③出现第二性征（男：长胡须、喉结突出、声音变粗；女：骨盆宽大、乳房增大、声音变细）。

**易错分析：** 身高突增是进入青春期的"第一信号"，不是性发育。

![配图](figures/人教版补充/15-card.webp)

### 卡16

**提问：** 光合作用的原料、产物、条件和场所分别是什么？写出反应式。

**答案：** 原料：二氧化碳 + 水。产物：有机物（储存能量）+ 氧气。条件：光。场所：叶绿体。反应式：二氧化碳 + 水 →(光能,叶绿体) 有机物 + 氧气。

**易错分析：** 光合作用的场所是叶绿体，不是线粒体！叶绿体是"能量转换站"（光能→化学能）。

![配图](figures/人教版补充/16-card.webp)

### 卡17

**提问：** 呼吸作用的反应式是什么？光合作用和呼吸作用的主要区别是什么？

**答案：** 呼吸作用：有机物 + 氧气 → 二氧化碳 + 水 + 能量（场所：线粒体，所有活细胞时刻进行）。光合作用：储存能量、释放O₂、需要光、只在含叶绿体的细胞进行；呼吸作用：释放能量、释放CO₂、不需要光、在所有活细胞进行。

**易错分析：** 植物白天同时进行光合作用和呼吸作用！晚上只进行呼吸作用。

![配图](figures/人教版补充/17-card.webp)

### 卡18

**提问：** 蒸腾作用的主要器官是什么？气孔的作用是什么？

**答案：** 蒸腾作用的主要器官是叶。气孔是植物蒸腾失水的"门户"和气体交换的"窗口"，由两个保卫细胞围成。

**易错分析：** 气孔白天张开（进行蒸腾和气体交换），晚上闭合。保卫细胞通过控制气孔开闭来调节蒸腾作用。

![配图](figures/人教版补充/18-card.webp)
