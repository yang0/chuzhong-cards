# 错题补充 代数易错记忆

## 基本信息

- 科目：数学
- 年级：七年级下册
- 教材：错题补充
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析
- 来源：当前错题中的整式、幂运算、因式分解、分式与应用题易错点

## 单元纲要

本文件不是人教版七下主线章节，而是根据当前错题补充的“必须背熟”的代数工具。它们在错题中频繁出现，适合作为每日短卡复习。

## Anki 记忆卡

### 卡1

**提问：** 同底数幂相乘的公式是什么？

**答案：** $a^m\cdot a^n=a^{m+n}$。

**易错分析：** 底数不变，指数相加。

### 卡2

**提问：** 同底数幂相除的公式是什么？

**答案：** $a^m\div a^n=a^{m-n}$，其中 $a\ne0$。

**易错分析：** 底数不变，指数相减；要注意底数不能为 0。

### 卡3

**提问：** 幂的乘方公式是什么？

**答案：** $(a^m)^n=a^{mn}$。

**易错分析：** 指数相乘，不是相加。

### 卡4

**提问：** 积的乘方公式是什么？

**答案：** $(ab)^n=a^n b^n$。

**易错分析：** 错题中 $(xy)^n$ 要写成 $x^n y^n$。

### 卡5

**提问：** $(-a)^n$ 的符号怎样判断？

**答案：** $n$ 为偶数时 $(-a)^n=a^n$；$n$ 为奇数时 $(-a)^n=-a^n$。

**易错分析：** 先看指数奇偶，再定符号。

### 卡6

**提问：** 平方差公式是什么？

**答案：** $(a+b)(a-b)=a^2-b^2$。

**易错分析：** 常见形式是 $(a-1)(a+1)=a^2-1$。

### 卡7

**提问：** 完全平方公式有哪些？

**答案：** $(a+b)^2=a^2+2ab+b^2$；$(a-b)^2=a^2-2ab+b^2$。

**易错分析：** 中间项是 $\pm2ab$，不要漏掉 2。

### 卡8

**提问：** 多项式乘多项式怎样展开？

**答案：** 每一项都要分别相乘，例如 $(a+b)(c+d)=ac+ad+bc+bd$。

**易错分析：** 遇到减号时，减号要带着项一起运算。

### 卡9

**提问：** 什么是整体代入？

**答案：** 题目给出某个整体的值时，先把目标式化成这个整体，再代入求值。

**易错分析：** 如已知 $x^2-6x=20$，目标式应尽量化成 $x^2-6x$。

### 卡10

**提问：** 因式分解中，提公因式法的基本形式是什么？

**答案：** $ma+mb=m(a+b)$。

**易错分析：** 提公因式后，括号内每一项都要除以公因式。

### 卡11

**提问：** 因式分解中遇到 $y-x$ 和 $x-y$ 怎么办？

**答案：** 记住 $y-x=-(x-y)$。

**易错分析：** 偶次方时 $(y-x)^2=(x-y)^2$；奇次方时符号相反。

### 卡12

**提问：** 什么叫分式？

**答案：** 分母中含有字母的式子叫分式。

**易错分析：** 判断分式只看分母是否含字母，不是看分子。

### 卡13

**提问：** 分式有意义的条件是什么？

**答案：** 分母不等于 0。

**易错分析：** 做分式题先写限制条件。

### 卡14

**提问：** 分式值为 0 的条件是什么？

**答案：** 分子等于 0，且分母不等于 0。

**易错分析：** 不能只令分子为 0，还要检查分母。

### 卡15

**提问：** 分式除法中，除式需要满足什么额外条件？

**答案：** 除式不能为 0。

**易错分析：** 如 $\frac{x-1}{x-2}\div\frac{x-3}{x-4}$，除了 $x\ne2,4$，还要 $x\ne3$。

### 卡16

**提问：** 分式基本性质是什么？

**答案：** 分式的分子、分母同时乘以或除以同一个不为 0 的整式，分式的值不变。

**易错分析：** 必须是分子分母“同时”乘除，且乘除的式子不能为 0。

### 卡17

**提问：** 异分母分式加减的基本步骤是什么？

**答案：** 先通分，再把分子相加减，最后能约分就约分。

**易错分析：** 通分后分母保持不变，只处理分子。

### 卡18

**提问：** 为什么 $\frac1{2-x}=-\frac1{x-2}$？

**答案：** 因为 $2-x=-(x-2)$。

**易错分析：** 相反式变形时，负号不要丢。

### 卡19

**提问：** 两段路程相同、速度分别为 $a,b$，平均速度是多少？

**答案：** 平均速度是 $\frac{2ab}{a+b}$。

**易错分析：** 平均速度 = 总路程 ÷ 总时间，不是 $\frac{a+b}{2}$。

### 卡20

**提问：** 百分数增长 $m\%$ 后，数量变为原来的多少倍？

**答案：** 变为原来的 $1+\frac{m}{100}$ 倍。

**易错分析：** 连续增长两次要乘两次，例如 $a(1+\frac{m}{100})^2$。

### 卡21

**提问：** 三位数 $\overline{abc}$ 怎样用代数式表示？

**答案：** $\overline{abc}=100a+10b+c$。

**易错分析：** 数位不同，权重不同，不能写成 $a+b+c$。

### 卡22

**提问：** 一个整数除以 9 的余数与什么有关？

**答案：** 与它各位数字和除以 9 的余数相同。

**易错分析：** 理由是 $100a+10b+c=99a+9b+(a+b+c)$。

### 卡23

**提问：** $(x^n-1)\div(x-1)$ 的展开规律是什么？

**答案：** $(x^n-1)\div(x-1)=x^{n-1}+x^{n-2}+\cdots+x+1$。

**易错分析：** 这是等比求和的雏形，常用于 $1+2+2^2+\cdots+2^n$。

### 卡24

**提问：** 单项式相乘时，字母指数怎样处理？

**答案：** 系数相乘，同底数字母指数相加。

**易错分析：** 如 $3x^2y^m\cdot x^ny=3x^{n+2}y^{m+1}$。

## 补充易错卡片

### 卡25

**提问：** 相交线模型中，对顶角和邻补角有什么关系？

**答案：** 对顶角相等，邻补角互补。若一个角增大 $4^\circ$，它的对顶角也增大 $4^\circ$，相邻的两个角都减小 $4^\circ$。

**易错分析：** 不要只凭图形位置猜变化；先判断是对顶角还是邻补角。

### 卡26

**提问：** “同位角、内错角、同旁内角”分别常用于什么结论？

**答案：** 两直线平行时：同位角相等、内错角相等、同旁内角互补。反过来，若同位角相等、内错角相等或同旁内角互补，都可以判定两直线平行。

**易错分析：** 性质和判定方向不同：已知平行用“性质”求角，想证明平行用“判定”找角关系。

### 卡27

**提问：** 找同位角、内错角、同旁内角时，先看什么？

**答案：** 先找两条被截直线和同一条截线，再看两个角在截线两侧还是同侧、在两直线之间还是外侧。

**易错分析：** 不能只看角长得像不像；必须围绕“哪两条直线被哪一条线截”来判断。

### 卡28

**提问：** 若两条直线都垂直于同一条直线，可以推出什么？

**答案：** 在同一平面内，若 $a\perp c$，$b\perp c$，则 $a\parallel b$。

**易错分析：** “同垂直推出平行”常作为中间桥梁，用来把垂直条件转化成平行条件。

### 卡29

**提问：** 若两条直线分别平行于一个角的两边，它们夹成的角一定等于原角吗？

**答案：** 不一定。根据射线方向，夹角可能等于原角，也可能与原角互补。

**易错分析：** 这类作图题要看具体开口方向；不能机械写成“平行则对应角相等”。

### 卡30

**提问：** 过直线外一点作已知直线的平行线，有几条？

**答案：** 有且只有一条。

**易错分析：** 若两条直线都经过同一点，并且都平行于同一条已知直线，那么这两条直线其实是同一条直线，常用来证明三点共线。

### 卡31

**提问：** 有两组对边分别平行的四边形，角之间有什么常用关系？

**答案：** 它是平行四边形。对角相等，邻角互补。

**易错分析：** 若 $AD\parallel BC$ 且 $AB\parallel DC$，可得 $\angle A=\angle C$，$\angle B=\angle D$，并且相邻两个角和为 $180^\circ$。

### 卡32

**提问：** 计算 $x^{2n}=25$ 这类题时，为什么不能直接写 $x^n=5$？

**答案：** 因为 $x^{2n}=(x^n)^2=25$，所以 $x^n=\pm5$。

**易错分析：** 平方等于正数时有两个相反数解；漏掉负值会导致后续乘积结果少一种可能。

### 卡33

**提问：** $(-a)^2\div(-a)^5\cdot a^3$ 这类题最容易错在哪里？

**答案：** 先分别处理符号和指数：$(-a)^2=a^2$，$(-a)^5=-a^5$，所以原式 $=a^2\div(-a^5)\cdot a^3=-1$。

**易错分析：** 负号属于底数时要看指数奇偶；不要把 $(-a)^5$ 写成 $a^5$。

### 卡34

**提问：** 整式化简求值题为什么要先化简再代入？

**答案：** 先化简可以合并同类项、消去高次项，减少计算量和符号错误，再把给定数值代入。

**易错分析：** 若直接代入，展开乘法和负号容易乱；特别是括号前有负号时，括号内每一项都要变号。

### 卡35

**提问：** 已知 $x^2-6x=20$，求含 $x$ 的式子时，常用什么思路？

**答案：** 先把目标式化简，尽量整理出 $x^2-6x$ 这个整体，再整体代入 $20$。

**易错分析：** 看到已知条件是一个整体，不要急着求 $x$；能整体代入通常更快更稳。

### 卡36

**提问：** 已知 $a^2+a-1=0$，怎样处理含 $a^3$ 的式子？

**答案：** 由 $a^2+a-1=0$ 得 $a^2=1-a$，再乘以 $a$ 可得 $a^3=a-a^2$，把高次降为低次。

**易错分析：** 这类题重点是“降次”，不是解出 $a$ 的具体值。

### 卡37

**提问：** 因式分解时，遇到 $x-y$ 和 $y-x$ 应该怎样统一？

**答案：** 记住 $y-x=-(x-y)$。奇次方会变号，偶次方不变号：$(y-x)^2=(x-y)^2$，$(y-x)^3=-(x-y)^3$。

**易错分析：** 统一相反式时，最容易丢负号；先统一成同一个括号，再提公因式。

### 卡38

**提问：** 因式分解的一般检查顺序是什么？

**答案：** 先看能否提公因式，再看能否用平方差、完全平方等公式，最后检查是否分解到不能再分解。

**易错分析：** 因式分解不是展开；结果应是几个因式相乘的形式。

### 卡39

**提问：** 分式乘除混合运算前，为什么常先因式分解？

**答案：** 因式分解后可以看出相同因式，便于约分，并能减少后续计算量。

**易错分析：** 约分只能约“乘法因式”，不能在加减项之间随便约掉一部分。

### 卡40

**提问：** 分式有意义和分式值为 $0$ 的条件分别是什么？

**答案：** 分式有意义：分母不等于 $0$。分式值为 $0$：分子等于 $0$，并且分母不等于 $0$。

**易错分析：** 求分式为 $0$ 时，最后一定要把候选值代回分母检查。

### 卡41

**提问：** 分式除法中，为什么还要检查“除式不为 0”？

**答案：** 因为除以一个分式时，这个分式本身不能为 $0$，所以除了各分母不为 $0$，还要让除式的分子不为 $0$。

**易错分析：** 如 $\frac{x-1}{x-2}\div\frac{x-3}{x-4}$，限制条件是 $x\ne2,4,3$。

### 卡42

**提问：** 用方程组解应用题时，设未知数后第一步应该做什么？

**答案：** 找等量关系，再根据等量关系列方程。比如费用问题常用“每天费用 × 天数 = 总费用”，工程问题常用“工作效率 × 时间 = 工作量”。

**易错分析：** 不要看到两个未知量就乱列式；先把题目中的“总量相等、费用相等、工作量为1”写清楚。

### 卡43

**提问：** 已知方程组的解，怎样求方程中的参数？

**答案：** 把已知解代入原方程组，把 $x,y$ 换成具体数，就得到关于参数的新方程，再解参数。

**易错分析：** “解是 $x=3,y=-2$”表示这组数同时满足两个方程，不是只代入其中一个。

### 卡44

**提问：** 统计调查中，什么样的样本才适合估计总体？

**答案：** 样本要具有代表性，最好随机抽取，并尽量覆盖总体中的不同类型对象。

**易错分析：** 只调查某一个学校、某一个小区或某一类人群，往往样本来源单一，不能代表整个城市或总体。
