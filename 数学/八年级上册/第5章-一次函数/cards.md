# 第5章 一次函数

- 科目：数学
- 年级：八年级上册
- 教材：浙教版2026秋新版
- 卡片数：6



## 1. ZJM8A-05-001

- 章节：5.1 常量与变量
- 提问：常量和变量怎样区分？
- 答案：在一个变化过程中，取值发生变化的量叫变量，保持不变的量叫常量；同一个量在不同问题中身份可能不同。
- 易错：常量和变量要放在具体变化过程中判断。
- 记忆钩子：看过程中取值变不变。
- 考点：情境识别题
- 提示词：prompts/ZJM8A-05-001.md



## 2. ZJM8A-05-002

- 章节：5.2 认识函数
- 提问：函数关系的核心含义是什么？
- 答案：在某变化过程中，给定一个变量x的允许值，相应变量y都有唯一确定的值与之对应，则y是x的函数。
- 易错：一个x对应多个y时，y不是x的函数。
- 记忆钩子：给一个x，只能对应一个y。
- 考点：关系判断题
- 提示词：prompts/ZJM8A-05-002.md



## 3. ZJM8A-05-003

- 章节：5.3 一次函数的意义
- 提问：一次函数和正比例函数的形式是什么？
- 答案：一次函数一般为y=kx+b，其中k、b为常数且k≠0；当b=0时为正比例函数y=kx。
- 易错：一次函数要求自变量最高次数为1且k不能为0。
- 记忆钩子：y=kx+b，b为0变正比例。
- 考点：解析式识别题
- 提示词：prompts/ZJM8A-05-003.md



## 4. ZJM8A-05-004

- 章节：5.4 一次函数的图象与性质
- 提问：一次函数y=kx+b的图象与k、b有什么关系？
- 答案：图象是一条直线；k决定倾斜方向和程度，k>0时y随x增大而增大，k<0时减小；b是与y轴交点的纵坐标。
- 易错：增减性由k符号决定，不由b决定。
- 记忆钩子：k管倾斜和增减，b管上下位置。
- 考点：图象性质高频题
- 提示词：prompts/ZJM8A-05-004.md



## 5. ZJM8A-05-005

- 章节：5.4 一次函数的图象与性质
- 提问：怎样用两点法画一次函数图象？
- 答案：一次函数图象是直线，只需取两个不同的自变量值求对应点，描出两点并作过两点的直线。常取与坐标轴的交点。
- 易错：直线要向两端延伸，不能只连接成有限线段。
- 记忆钩子：两点确定一条直线。
- 考点：作图题
- 提示词：prompts/ZJM8A-05-005.md



## 6. ZJM8A-05-006

- 章节：5.5 一次函数的简单应用
- 提问：怎样用一次函数解决方案选择问题？
- 答案：把实际量设为变量，列出各方案的函数关系，在同一坐标系比较图象交点和不同区间的函数值，结合取值范围选择较优方案。
- 易错：交点只表示费用相等，交点两侧要分别判断哪个方案更优。
- 记忆钩子：建函数、找交点、分区间比较。
- 考点：方案决策综合题
- 提示词：prompts/ZJM8A-05-006.md
