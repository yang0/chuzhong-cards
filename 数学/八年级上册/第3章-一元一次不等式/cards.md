# 第3章 一元一次不等式

- 科目：数学
- 年级：八年级上册
- 教材：浙教版2026秋新版
- 卡片数：4



## 1. ZJM8A-03-001

- 章节：3.1 认识不等式
- 提问：不等式的解和解集有什么区别？
- 答案：使不等式成立的未知数值叫不等式的解；所有解组成的集合叫解集，通常用数轴表示。
- 易错：一个解只是解集中的一个值，不能代替整个解集。
- 记忆钩子：单个值是解，全部范围是解集。
- 考点：概念与数轴题
- 提示词：prompts/ZJM8A-03-001.md



## 2. ZJM8A-03-002

- 章节：3.2 不等式的基本性质
- 提问：不等式两边乘除负数时为什么要改变不等号方向？
- 答案：负数乘法会把数轴上的点关于原点翻转，原来较大的数变到左边，因此大小关系反向。
- 易错：只有乘或除以负数时变号，加减负数不需要变号。
- 记忆钩子：负数乘除像数轴翻面，不等号要转向。
- 考点：解不等式易错点
- 提示词：prompts/ZJM8A-03-002.md



## 3. ZJM8A-03-003

- 章节：3.3 一元一次不等式及其解法
- 提问：解一元一次不等式的一般步骤是什么？
- 答案：去分母、去括号、移项、合并同类项、系数化为1，并在乘除负数时改变不等号方向，最后用数轴表示解集。
- 易错：去分母时每一项都要乘公分母。
- 记忆钩子：去分母括号，移项合并，系数化一。
- 考点：计算必考题
- 提示词：prompts/ZJM8A-03-003.md



## 4. ZJM8A-03-004

- 章节：3.4 一元一次不等式组
- 提问：怎样求一元一次不等式组的解集？
- 答案：分别求出每个不等式的解集，再在同一数轴上取它们的公共部分；同大取大、同小取小、大小小大取中间、大大小小无解。
- 易错：不等式组取交集，不是把两个范围合并成并集。
- 记忆钩子：分别求解，数轴取公共部分。
- 考点：数轴与参数题
- 提示词：prompts/ZJM8A-03-004.md
