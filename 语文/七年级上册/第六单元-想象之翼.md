# 第六单元 想象之翼

## 基本信息

- 科目：语文
- 年级：七年级上册
- 教材：统编版 / 人教版《语文》（2024 秋版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"想象"为主题，选编中外富有想象力的文学作品。学习重点是快速阅读（每分钟400字以上）、联想与想象、理解童话和寓言的深层含义。

- 21 小圣施威降大圣 / 吴承恩（新增）
- 22 皇帝的新装 / 安徒生
- 23\* 女娲造人 / 袁珂
- 24 寓言四则：赫耳墨斯和雕像者 / 《伊索寓言》、蚊子和狮子 / 《伊索寓言》、穿井得一人 / 《吕氏春秋》、杞人忧天 / 《列子》
- 整本书阅读：《西游记》
- 课外古诗词诵读：秋词（其一）/ 刘禹锡、夜雨寄北 / 李商隐、十一月四日风雨大作（其二）/ 陆游、潼关 / 谭嗣同

## Anki 记忆卡

### 卡1

**配图：** ![](第六单元-想象之翼/01-xiaosheng-wucheng'en.png)

**提问：** 《小圣施威降大圣》出自哪部作品？作者是谁？（2024新增）

**答案：** 出自《西游记》，作者吴承恩，明代小说家。本文节选自《西游记》第六回，写二郎神（小圣）与孙悟空（大圣）斗法变化的精彩片段。

**易错分析：** "小圣"是二郎神（杨戬），"大圣"是孙悟空。注意不是"小圣"打败"大圣"，而是两人斗法，变化无穷。吴承恩是"明代"不是"清代"。

**记忆钩子：** 吴承恩西游记，二圣斗法72变。

### 卡2

**配图：** ![](第六单元-想象之翼/02-xiaosheng-content.png)

**提问：** 《小圣施威降大圣》中孙悟空和二郎神分别变了哪些东西？

**答案：** 孙悟空的变化：麻雀→大鹚老→鱼→水蛇→花鸨→土地庙→二郎神。二郎神的变化：饿鹰→大海鹤→鱼鹰→灰鹤→现原身→用弹弓打。最终二人回到花果山继续大战，不分胜负。

**易错分析：** 变化的顺序不能搞混。每一次变化都是"大圣变小圣追"，体现孙悟空的机智灵活和二郎神的紧追不舍。课文选择的精彩片段是《西游记》中最具想象力的段落之一。

**记忆钩子：** 雀鹚鱼蛇鸨庙神，鹰鹤鱼鹤弹弓追。

### 卡3

**配图：** ![](第六单元-想象之翼/03-xiaosheng-imagination.png)

**提问：** 《小圣施威降大圣》体现了《西游记》怎样的艺术特色？

**答案：** ①丰富的想象力——各种变化层出不穷，上天入地、水陆空全开；②幽默诙谐——孙悟空变土地庙时尾巴不好放，变成旗杆竖在庙后，十分滑稽；③人物对比——孙悟空的机智灵活与二郎神的冷静果决形成鲜明对照；④语言生动传神——打斗场面动作描写精彩。

**易错分析：** 斗法不仅展示变化的多，还体现人物性格——孙悟空顽皮机灵、二郎神认真执着。

**记忆钩子：** 想象丰富又幽默，二圣斗法显性格。

### 卡4

**配图：** ![](第六单元-想象之翼/04-emperor-new-clothes.png)

**提问：** 《皇帝的新装》的作者是谁？哪国人？体裁是什么？

**答案：** 安徒生，丹麦作家，世界著名童话作家。代表作《卖火柴的小女孩》《丑小鸭》《海的女儿》《拇指姑娘》等。体裁是童话。这篇童话通过一个愚蠢的皇帝被两个骗子愚弄的故事，讽刺了统治者的愚蠢和虚伪。

**易错分析：** 安徒生是"丹麦"人，不是"瑞典"或"挪威"。童话（fairy tale）不等同于"寓言"（fable）——童话篇幅更长、情节更丰富。

**记忆钩子：** 安徒生丹麦童话，皇帝新装讽虚伪。

### 卡5

**配图：** ![](第六单元-想象之翼/05-emperor-content.png)

**提问：** 《皇帝的新装》中谁说了真话？为什么只有孩子能说真话？

**答案：** 一个天真无邪的小孩说了真话："他什么衣服也没穿！"只有孩子能说真话，因为孩子天真无邪、没有私心杂念，不会像大人们那样因为害怕被认为"不称职"或"愚蠢"而撒谎。大人（皇帝、大臣、百姓）都被虚荣心和恐惧所束缚。

**易错分析：** 主题不只是"讽剌皇帝爱打扮"，更深层是批判成人世界的虚伪、自欺欺人和盲从心理。"新装"是看不见的——其实是"不存在的衣服"。

**记忆钩子：** 小孩说真话，大人皆虚伪——童真破谎言。

### 卡6

**配图：** ![](第六单元-想象之翼/06-emperor-new-clothes-theme.png)

**提问：** 《皇帝的新装》的写作特点有哪些？

**答案：** ①丰富的想象和大胆的夸张——骗子说"最笨的人看不见衣服"，居然人人都怕被看作笨蛋；②心理描写细腻——皇帝、大臣明明看不见衣服却不敢说，内心的恐慌和伪装写得入木三分；③对比手法——孩子的天真与大人们的虚伪形成强烈对比；④讽刺辛辣——用荒诞的故事揭露社会现实。

**易错分析：** 写作手法是"夸张"和"讽刺"，不是"比喻"或"拟人"。

**记忆钩子：** 夸张讽刺心理细，童话背后见人性。

### 卡7

**配图：** ![](第六单元-想象之翼/07-nvwa-yuanke.png)

**提问：** 《女娲造人》的作者是谁？体裁是什么？出自哪部作品？

**答案：** 袁珂，当代神话学家。体裁是神话。本文选自《神话故事新编》，是在中国古代神话基础上改编扩充而成的。讲述了女娲用黄土造人、用藤条甩泥浆造人的故事。

**易错分析：** 这是"神话"不是"童话"或"寓言"。神话与童话的区别：神话有神圣性、解释起源（人是哪里来的），童话注重道德教化和情节趣味。"女娲"的"娲"读 wā，不读 wō。

**记忆钩子：** 袁珂神话女娲造人，黄土造人是创世传说。

### 卡8

**配图：** ![](第六单元-想象之翼/08-nvwa-content.png)

**提问：** 《女娲造人》中女娲是一个怎样的形象？

**答案：** 女娲是一个充满智慧和创造力、勤劳善良、慈爱伟大的女神形象。她从无聊到灵机一动造人，看到新生命时充满喜悦；她不知疲倦地造人，还想出了用藤条甩出泥浆造人的办法，让人类遍布大地。她体现了先民对创造生命伟大力量的赞美。

**易错分析：** 女娲造人有"两个阶段"：先"手工捏"（造出的人是富贵者？）后"藤条甩"（造出的人是贫贱者？）——这个说法是后世附会，课文本身没有这种区分。不要画蛇添足。

**记忆钩子：** 女娲累了甩藤条，智慧慈爱创人类。

### 卡9

**配图：** ![](第六单元-想象之翼/09-yuyan-four.png)

**提问：** 第六单元寓言四则分别出自哪些作品？各讲了什么道理？

**答案：** 
①《赫耳墨斯和雕像者》（《伊索寓言》）——讽刺了妄自尊大、爱慕虚荣而不被重视的人（赫耳墨斯问自己雕像的价格，结果被说是"添头"白送）。
②《蚊子和狮子》（《伊索寓言》）——讽刺了打败大人物后得意忘形、最终栽在小人物手上的行为（蚊子打败狮子却被蜘蛛网粘住）。
③《穿井得一人》（《吕氏春秋》）——传闻不可轻信，要调查核实（丁家打井节省人力，被传言成"穿井得一人"——挖出一个人）。
④《杞人忧天》（《列子》）——讽刺不必要的忧虑和庸人自扰（杞国人担心天塌下来无处躲藏）。

**易错分析：** 《赫耳墨斯和雕像者》——问价三次（宙斯→赫拉→自己），层层推进，最后"添头"是讽刺高潮。"添头"指额外赠送的东西。《穿井得一人》是"得一个人的劳力"不是"挖出一个人"——这是典型"以讹传讹"的例子。

**记忆钩子：** 狂妄变添头，得意被蛛粘，以讹传讹得一人，杞人忧天自扰之。

### 卡10

**配图：** ![](第六单元-想象之翼/10-yuyan-writing.png)

**提问：** 寓言这种体裁有什么特点？

**答案：** ①篇幅短小；②用一个故事来说明一个道理（寓理于事）；③常用比喻、拟人、夸张等手法；④主人公可以是人也可以是动植物或其他事物；⑤结尾往往直接点明寓意（有的含蓄）。

**易错分析：** 寓言与童话的区别——寓言短小精悍、直接说理；童话篇幅较长、情节更丰富。

**记忆钩子：** 短小故事讲道理，主人公可人可物。

### 卡11

**配图：** ![](第六单元-想象之翼/11-xiyouji-wucheng'en.png)

**提问：** 《西游记》的作者是谁？这部作品的体裁和地位是什么？

**答案：** 吴承恩，明代小说家。《西游记》是中国古代四大名著之一，是中国古典文学中最杰出的神魔小说（浪漫主义长篇神魔小说）。全书共100回，以唐僧师徒四人西天取经为主线。

**易错分析：** 四大名著：《西游记》《三国演义》《水浒传》《红楼梦》。《西游记》是"神魔小说"不是"神话小说"或"志怪小说"。

**记忆钩子：** 吴承恩明代神魔小说，四大名著西天取经。

### 卡12

**配图：** ![](第六单元-想象之翼/12-xiyouji-characters.png)

**提问：** 《西游记》中四个主要人物各有什么性格特点？

**答案：** ①唐僧（玄奘）——慈悲善良、意志坚定，但有时善恶不分、软弱迂腐；②孙悟空——机智勇敢、神通广大、嫉恶如仇、桀骜不驯；③猪八戒——贪吃好色、贪生怕死，但憨厚可爱、能吃苦、关键时刻也出力；④沙和尚（沙僧）——忠心耿耿、任劳任怨、沉默寡言。

**易错分析：** 分析人物要全面——唐僧不能只答"善良"，也要说"软弱"；猪八戒不能只答"贪吃"，也要说"憨厚"。人物形象是立体的。

**记忆钩子：** 唐僧慈悲软弱，悟空勇敢桀骜，八戒贪吃憨厚，沙僧忠厚老实。

### 卡13

**配图：** ![](第六单元-想象之翼/13-xiyouji-theme.png)

**提问：** 《西游记》的主题思想是什么？

**答案：** ①通过唐僧师徒四人历经九九八十一难、最终取回真经的故事，表现了不畏艰险、锲而不舍的奋斗精神；②揭露和批判了天上人间封建统治者的腐朽（如妖怪往往是天上下凡的）；③歌颂了反抗压迫、追求自由的斗争精神（大闹天宫部分）；④体现了"修心"的哲理——取经的过程也是师徒四人不断完善自我、克服心魔的修行过程。

**易错分析：** 唐僧取经共"九九八十一难"（不是72或108）。主题不能只答"取经"——更深层是对人性修行的隐喻（每遇一难就是一次心性考验）。

**记忆钩子：** 八十一难取真经，战胜心魔终成佛。

### 卡14

**配图：** ![](第六单元-想象之翼/14-xiyouji-famous.png)

**提问：** 《西游记》中你印象最深的经典情节有哪些？（至少列举5个）

**答案：** ①大闹天宫（孙悟空反天庭）；②三打白骨精（孙悟空三次打死白骨精变的村姑、老妇、老翁）；③三借芭蕉扇（过火焰山）；④大闹盘丝洞（蜘蛛精）；⑤真假美猴王；⑥偷吃人参果；⑦女儿国；⑧车迟国斗法；⑨大战红孩儿；⑩火焰山。（任选5个即可）

**易错分析：** 这些情节的顺序和所在回合大致要知道。整本书阅读常考"你最喜欢哪个情节并说明理由"。

**记忆钩子：** 大闹天宫三打骨，芭蕉扇火焰山，真假猴王盘丝洞。

### 卡15

**配图：** ![](第六单元-想象之翼/15-autumn-lyrics-liuyuxi.png)

**提问：** "自古逢秋悲寂寥，我言秋日胜春朝"（《秋词·其一》）的作者和思想？

**答案：** 刘禹锡，唐代诗人。一反古代文人悲秋的传统，以豪迈的笔调赞美秋天"胜春朝"——秋天比春天更美。"晴空一鹤排云上，便引诗情到碧霄"，借白鹤凌云直上的形象，抒发了诗人昂扬向上、积极乐观、奋发有为的豪情壮志。

**易错分析：** 刘禹锡的诗风格豪迈，"自古逢秋悲寂寥"是破——打破传统，"我言秋日胜春朝"是立——建立自己的观点。"排云上"的"排"是"推/冲"的意思。

**记忆钩子：** 刘禹锡一反悲秋，豪情壮志上碧霄。

### 卡16

**配图：** ![](第六单元-想象之翼/16-night-rain-li-shangyin.png)

**提问：** "何当共剪西窗烛，却话巴山夜雨时"（《夜雨寄北》）表达了什么？

**答案：** 李商隐，唐代诗人。前两句写眼前之景——巴山夜雨涨满秋池，不知何时才能回去。后两句写未来之想——何时能和你一起在西窗下剪烛夜谈，回头再说起今天巴山夜雨时的思念之情。全诗以眼前景写未来情，以实写虚，表达了对远方亲友的深切思念和对重逢的期盼。

**易错分析：** "寄北"是寄给北方的人（妻子或友人）。"巴山"泛指巴蜀之地。"共剪西窗烛"是想象未来团聚的场景——虚实结合的手法。"却话"是"回头再说"。

**记忆钩子：** 李商隐巴山夜雨，望剪烛共话相思。

### 卡17

**配图：** ![](第六单元-想象之翼/17-nov-4-storm-luyou.png)

**提问：** "夜阑卧听风吹雨，铁马冰河入梦来"（《十一月四日风雨大作·其二》）表达了什么？

**答案：** 陆游，南宋爱国诗人。"僵卧孤村不自哀，尚思为国戍轮台"——虽年老体衰、被罢官闲居，但毫不自哀，仍然想着为国戍守边疆。"夜阑卧听风吹雨，铁马冰河入梦来"——深夜听到风雨声仿佛千军万马，梦境中金戈铁马、跨过冰河去杀敌！表达了诗人至死不渝的爱国情怀和收复失地的热切愿望。

**易错分析：** "僵卧"不是真的僵了，而是"年老体衰、闲居在家"。"轮台"借代边境（新疆轮台是古边防要地）。"风吹雨"既是实写也是比喻——比喻国家的危难局势。

**记忆钩子：** 陆游僵卧犹思战，风雨入梦铁马来。

### 卡18

**配图：** ![](第六单元-想象之翼/18-tongguan-tansitong.png)

**提问：** "河流大野犹嫌束，山入潼关不解平"（《潼关》）表达了什么思想？

**答案：** 谭嗣同，清末维新派政治家、诗人。"终古高云簇此城，秋风吹散马蹄声"——写景壮阔。"河流大野犹嫌束，山入潼关不解平"——黄河奔流在广阔的原野上还嫌束缚，进入潼关后的山峦重重叠叠不知平坦为何物。借山河的奔放不羁，抒发了诗人渴望冲破束缚、追求自由解放的豪情壮志。

**易错分析：** 谭嗣同是"清末"人，是维新变法（戊戌六君子之一）的代表人物。"犹嫌束"和"不解平"既是写山水的特点，也是诗人自我精神的写照——追求自由、不受束缚。

**记忆钩子：** 谭嗣同潼关咏山河，借景抒发自由志。

### 卡19

**配图：** ![](第六单元-想象之翼/19-unit6-outside-poems.png)

**提问：** 第六单元课外古诗词诵读四首分别表达了什么主题？

**答案：** 《秋词（其一）》——一反悲秋传统，表达豪迈乐观之情。《夜雨寄北》——思念亲友、期盼重逢。《十一月四日风雨大作》——老而弥坚的爱国情怀。《潼关》——渴望冲破束缚、追求自由。

**易错分析：** 四首诗主题鲜明不同。秋词是"豪情"，夜雨寄北是"思念"，十一月是"爱国"，潼关是"自由"。

**记忆钩子：** 秋词豪夜雨思，十一月爱国潼关自由。

### 卡20

**配图：** ![](第六单元-想象之翼/20-unit6-imagination.png)

**提问：** 本单元"想象"的主题在四篇课文中分别是如何体现的？

**答案：** 《小圣施威降大圣》——通过奇妙的变身斗法，展现了中国古典小说中天马行空的想象。《皇帝的新装》——通过"看不见的衣服"这个荒诞设定，展现了讽刺性想象。《女娲造人》——通过造人的神话解释人类的起源，展现了先民对人类由来的想象。《寓言四则》——通过虚构的小故事传达道理，展现了寓意性想象。

**易错分析：** 四类想象各不相同——变身想象、讽刺想象、创世想象、寓意想象。要区分开来。

**记忆钩子：** 斗法荒诞创世寓意，四种想象各有各妙。
