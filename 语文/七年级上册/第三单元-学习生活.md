# 第三单元 学习生活

## 基本信息

- 科目：语文
- 年级：七年级上册
- 教材：统编版 / 人教版《语文》（2024 秋版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"学习生活"为主题，带领学生回顾不同年代的求学经历，感受学习中的乐趣与成长。学习重点是默读、厘清文章思路、理解人物情感。

- 9 从百草园到三味书屋 / 鲁迅
- 10 往事依依 / 于漪（新增）
- 11\* 再塑生命的人 / 海伦·凯勒
- 12 《论语》十二章
- 整本书阅读：《朝花夕拾》
- 课外古诗词诵读：峨眉山月歌 / 李白、江南逢李龟年 / 杜甫、行军九日思长安故园 / 岑参、夜上受降城闻笛 / 李益

## Anki 记忆卡

### 卡1

**配图：** ![](第三单元-学习生活/01.webp)

**提问：** 《从百草园到三味书屋》的作者是谁？体裁是什么？选自哪部作品？

**答案：** 鲁迅，原名周树人，字豫才，浙江绍兴人。近代文学家、思想家、革命家。体裁是散文（回忆性散文），选自散文集《朝花夕拾》。

**易错分析：** 鲁迅原名"周树人"不是"周树"，"豫才"是字。"朝花夕拾"的"朝"读 zhāo（早晨）不读 cháo。《朝花夕拾》是散文集，不是小说集。

**记忆钩子：** 鲁迅周树人，朝花夕拾忆童年。

### 卡2

**配图：** ![](第三单元-学习生活/02.webp)

**提问：** 《从百草园到三味书屋》中百草园和三味书屋两部分各写了什么？用什么结构连接？

**答案：** 百草园部分写童年鲁迅在园中玩耍的乐趣——看景物、听故事、捕鸟等。三味书屋部分写读书生活——拜师、问"怪哉"、读书、画画等。两部分用"也许是因为拔何首乌毁了泥墙罢……"一个过渡段连接，用"承上启下"的写法。

**易错分析：** 文章结构是从"百草园（乐）"到"三味书屋（不那么自由）"的对比，不是平行并列。过渡段在文中第9段。

**记忆钩子：** 百草园乐三味书闷，过渡段承上启下。

### 卡3

**配图：** ![](第三单元-学习生活/03.webp)

**提问：** 《从百草园到三味书屋》中描写百草园景物（第二段）用了什么写作顺序？

**答案：** 由低到高（菜畦→石井栏→皂荚树→桑椹），再由高到低（鸣蝉→黄蜂→叫天子），再由低到高（油蛉→蟋蟀→蜈蚣→斑蝥→何首乌→木莲→覆盆子）。从视觉到听觉再到味觉，多感官描写。

**易错分析：** 顺序不是无序罗列，而是有层次的空间变化。从"不必说……也不必说……单是……"的句式来看，先略写整体再详写局部（泥墙根一带）。

**记忆钩子：** 高低交错多感官，不必说加单是。

### 卡4

**配图：** ![](第三单元-学习生活/04.webp)

**提问：** 《从百草园到三味书屋》中雪地捕鸟一段用了什么动词？有什么作用？

**答案：** "扫开、露出、支起、撒、系、牵、看、拉、罩"等一系列动词。准确生动地写出了捕鸟的全过程，表现了百草园给童年鲁迅带来的无限乐趣，同时也反映了闰土父亲捕鸟经验丰富。

**易错分析：** 动词要列全，不能只写三四个。这段是"动作描写"的经典范例，考试常考动词的连贯性。

**记忆钩子：** 扫露支撒系，牵看拉罩——捕鸟九连动。

### 卡5

**配图：** ![](第三单元-学习生活/05.webp)

**提问：** 《往事依依》的作者是谁？这是一篇什么体裁的文章？（2024新增）

**答案：** 于漪，当代著名语文教育家，"人民教育家"国家荣誉称号获得者。体裁是回忆性散文。文章回忆了作者少年时代的几件往事，表达了对读书和学习的热爱。

**易错分析：** "漪"读 yī（一），不读 qí。于漪是语文教育家，不是作家，这是2024版新加课文，之前是苏教版课文。

**记忆钩子：** 于漪人民教育家，往事依依忆读书。

### 卡6

**配图：** ![](第三单元-学习生活/06.webp)

**提问：** 《往事依依》中作者回忆了哪几件往事？

**答案：** ①小时候看山水画和《评注图像水浒传》插图，获得美的享受；②读《千家诗》感受四季风光，沉醉于诗歌意境；③听国文老师朗读《辛弃疾词》，激发爱国情怀；④听代课老师讲田忌赛马故事，懂得要智取。这些经历都促使作者爱上了读书。

**易错分析：** 这一课是2024新版新增，考试可能出现。四件事的次序不能混淆。每件事都对应一种"受益"——美的享受、诗的心情、爱国情怀、智慧启迪。

**记忆钩子：** 画水浒、千家诗、辛词、赛马——四事爱上读书。

### 卡7

**配图：** ![](第三单元-学习生活/07.webp)

**提问：** 《再塑生命的人》的作者是谁？"再塑生命"是什么意思？

**答案：** 海伦·凯勒，美国盲聋女作家、教育家、慈善家。代表作《假如给我三天光明》。"再塑生命"指莎莉文老师通过教育让海伦从黑暗无声的世界中走出来，获得了知识、光明和新生，是精神生命的重塑。

**易错分析：** 海伦·凯勒是"盲聋人"不是只盲不聋。"再塑"不是"改造"或"教育"，是给予新生般的改变。莎莉文老师是安妮·莎莉文。

**记忆钩子：** 海伦凯勒盲聋人，莎莉文重塑其生命。

### 卡8

**配图：** ![](第三单元-学习生活/08.webp)

**提问：** 《再塑生命的人》中"水"这个字对海伦意味着什么？

**答案：** 当莎莉文老师在海伦手上拼写"water"并让清凉的水流过她手心时，海伦突然明白了——万物都有自己的名字，语言不是手指游戏而是理解世界的钥匙。这是海伦思想的觉醒，是她"再塑生命"的转折点。

**易错分析：** 这段在文中很关键，"water"的领悟标志海伦从感知到认知的跨越。不是学会了单词，而是理解了"语言=意义"。

**记忆钩子：** 水流通手心，名字即世界——觉醒的转折点。

### 卡9

**配图：** ![](第三单元-学习生活/09.webp)

**提问：** 《论语》是一部怎样的书？作者是谁？

**答案：** 《论语》是儒家经典著作，是记录孔子及其弟子言行的书（语录体）。由孔子弟子及再传弟子编纂。孔子，名丘，字仲尼，春秋末期鲁国人，思想家、政治家、教育家，儒家学派创始人。

**易错分析：** 《论语》不是孔子写的，是"弟子编纂"！"论"读 lún，不读 lùn。"语"是言论、语录的意思。孔子是"春秋末期"不是"战国"。

**记忆钩子：** 论语非孔撰，弟子记言行。

### 卡10

**配图：** ![](第三单元-学习生活/10.webp)

**提问：** 《论语》十二章分别讲了哪些方面的内容？

**答案：** 
1. 学而时习之——学习方法与乐趣
2. 吾日三省吾身——自我反省（修身）
3. 吾十有五而志于学——人生阶段与为学境界
4. 温故而知新——温故知新
5. 学而不思则罔——学思结合
6. 贤哉回也——安贫乐道
7. 知之者不如好之者——兴趣是最好的老师
8. 饭疏食饮水——坚守道义（富贵浮云）
9. 三人行必有我师——虚心向他人学习
10. 子在川上曰——珍惜时间
11. 三军可夺帅也——匹夫不可夺志（坚定志向）
12. 博学而笃志——博学笃志、切问近思

**易错分析：** 孔子的核心思想：仁（仁爱）、礼（礼节）、义（道义）。《论语》章节中的关键词句默写是考试重点。

**记忆钩子：** 十二论语言学习修身立志惜时。

### 卡11

**配图：** ![](第三单元-学习生活/11.webp)

**提问：** 《论语》十二章中重点字词解释：时习、说、愠、三省、罔、殆、逾、立、惑、天命、耳顺、好、乐、笃。

**答案：**
- 时习：按时温习（时：按时）。
- 说：同"悦"，愉快。
- 愠（yùn）：生气，恼怒。
- 三省：多次自我反省（三：多次，非三次）。
- 罔（wǎng）：迷惑，感到迷茫。
- 殆（dài）：疑惑/有害。
- 逾（yú）：越过（不逾矩）。
- 立：能有所成就（三十而立）。
- 惑：迷惑、疑惑（四十而不惑）。
- 天命：上天的意旨/道的规律。
- 耳顺：能听得进不同意见。
- 好（hào）：喜欢、爱好（知之者不如好之者）。
- 乐（lè）：以……为快乐（好之者不如乐之者）。
- 笃（dǔ）：坚定（博学而笃志）。

**易错分析：** "说"通"悦"是通假字；"三省"是三"次"不是三"件"事；"罔"是迷惑，"殆"是疑惑，两字不同。"好"读 hào（爱好），"乐"读 lè（以……为乐）。

**记忆钩子：** 说通悦愠生气，罔迷惑殆疑惑。

### 卡12

**配图：** ![](第三单元-学习生活/12.webp)

**提问：** 《朝花夕拾》是一部什么书？收录了多少篇？主要写什么？

**答案：** 鲁迅的回忆性散文集。收录了10篇回忆性散文，原名《旧事重提》。主要记述鲁迅从童年到青年时期的生活经历和思想成长轨迹。

**记忆钩子：** 朝花夕拾十篇文，鲁迅旧事重提。

### 卡13

**配图：** ![](第三单元-学习生活/13.webp)

**提问：** 《朝花夕拾》10篇文章分别是什么？请一一列出。

**答案：** 《狗·猫·鼠》《阿长与〈山海经〉》《二十四孝图》《五猖会》《无常》《从百草园到三味书屋》《父亲的病》《琐记》《藤野先生》《范爱农》。

**记忆钩子：** 狗猫鼠长妈妈，二十四五猖无常，百草园父病琐记，藤野范爱农收场。

### 卡14

**配图：** ![](第三单元-学习生活/14.webp)

**提问：** 《朝花夕拾》中写了哪些主要人物？各有什么特点？

**答案：** ①阿长（长妈妈）——保姆，粗俗迷信但善良热心，给鲁迅买《山海经》；②藤野先生——日本老师，治学严谨、没有民族偏见；③范爱农——朋友，正直倔强、不满社会现实；④父亲——严厉但关爱儿子；⑤衍太太——自私虚伪、多嘴多舌的市井妇女。

**易错分析：** 阿长是"粗俗善良"不是"粗鲁坏心"；藤野没有民族偏见是核心品质；衍太太是反面形象，不要漏掉。

**记忆钩子：** 长妈妈买山海经，藤野严谨无偏见。

### 卡15

**配图：** ![](第三单元-学习生活/15.webp)

**提问：** 《朝花夕拾》的主题思想是什么？

**答案：** ①回忆童年趣事和青少年求学经历；②展现当时的社会风貌和人情世故；③批判封建礼教、旧教育制度和国民劣根性；④表达对师友的怀念和对黑暗社会现实的愤懑。

**易错分析：** 《朝花夕拾》不只是"回忆童年"，还有对社会的批判，有温情也有冷峻。分析时两面都要答到。

**记忆钩子：** 忆旧事批封建，有温情有批判。

### 卡16

**配图：** ![](第三单元-学习生活/16.webp)

**提问：** "峨眉山月半轮秋，影入平羌江水流"（《峨眉山月歌》）的作者和内容？

**答案：** 李白，唐代诗人。前两句写峨眉山秋夜月色——半轮秋月映在平羌江中随波流动。后两句"夜发清溪向三峡，思君不见下渝州"写连夜出发、思念友人。全诗连用五个地名（峨眉、平羌、清溪、三峡、渝州），意境空灵秀丽。

**易错分析：** "平羌"即青衣江。"半轮秋"是说秋月是半圆的上弦月或下弦月，不是满月。"思君"的"君"指友人。

**记忆钩子：** 李白五地名，峨眉秋月思友人。

### 卡17

**配图：** ![](第三单元-学习生活/17.webp)

**提问：** "正是江南好风景，落花时节又逢君"（《江南逢李龟年》）表达了什么情感？

**答案：** 杜甫，唐代诗人。前两句回忆当年在岐王、崔九家听李龟年唱歌的盛况。后两句写如今在江南落花时节重逢——表面写重逢的喜悦，实则寄寓了世运衰败、人生漂泊的无限感慨（以"落花"象征国运凋零）。全诗含蓄深沉，是"以乐景写哀情"。

**易错分析：** "落花时节"既指暮春，又象征国运衰落和国家的不幸。不是单纯的"重逢高兴"。"龟"读 guī，李龟年是著名歌唱家。

**记忆钩子：** 杜甫逢李龟年，落花时节忆盛衰。

### 卡18

**配图：** ![](第三单元-学习生活/18.webp)

**提问：** "遥怜故园菊，应傍战场开"（《行军九日思长安故园》）表达了什么？

**答案：** 岑参，唐代边塞诗人。时逢重阳节（九月九日），行军途中想到长安故园的菊花应该在战火中开放。"遥怜"二字写出对故乡的思念，"战场"二字暗含对安史之乱的忧愤和对百姓苦难的同情。以"菊"寄托乡思和忧国之情。

**易错分析：** "九日"指重阳节。岑参是边塞诗人，但这首不是写边塞而是写战乱思乡。"傍"读 bàng。

**记忆钩子：** 岑参重阳行军，战场菊花思故园。

### 卡19

**配图：** ![](第三单元-学习生活/19.webp)

**提问：** "不知何处吹芦管，一夜征人尽望乡"（《夜上受降城闻笛》）赏析。

**答案：** 李益，唐代诗人。前两句写边城夜色（回乐峰前沙似雪，受降城外月如霜），营造苍凉氛围。后两句写不知何处传来芦笛声，一夜之间所有戍边将士都遥望故乡——从视觉到听觉，借笛声抒发戍边将士的思乡之情。比喻（沙似雪、月如霜）写出边塞的寒冷荒凉。

**易错分析：** "受降城"是唐代边塞地名，不是"受降"的意思。"芦管"是一种乐器（笛类）。"尽"是"全都"的意思。

**记忆钩子：** 沙似雪月如霜，一夜笛声尽望乡。

### 卡20

**配图：** ![](第三单元-学习生活/20.webp)

**提问：** 第三单元课外古诗词诵读四首分别表达了什么主题？

**答案：** 《峨眉山月歌》——借月抒写对友人的思念。《江南逢李龟年》——感时伤世，怀念故人。《行军九日思长安故园》——重阳思乡+忧国忧民。《夜上受降城闻笛》——边塞将士的思乡之情。

**记忆钩子：** 李白思友杜甫伤时，岑参忧国李益思乡。

### 卡21

**配图：** ![](第三单元-学习生活/21.webp)

**提问：** 第三单元写作主题是什么？如何突出中心？

**答案：** 写作主题是"如何突出中心"。要做到：①围绕中心选材（跟中心无关的素材要舍得删）；②安排好主次详略（重点内容详写）；③运用恰当的写作手法（开门见山/卒章显志/前后呼应/对比衬托）。

**易错分析：** 2024版写作由"写人要抓住特点"改为"如何突出中心"，考试要注意是新版。

**记忆钩子：** 围绕中心选素材，详略得当前后呼应。
