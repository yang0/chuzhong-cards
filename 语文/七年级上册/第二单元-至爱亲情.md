# 第二单元 至爱亲情

## 基本信息

- 科目：语文
- 年级：七年级上册
- 教材：统编版 / 人教版《语文》（2024 秋版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"至爱亲情"为主题，选编四篇文章，从不同角度展现亲情的美好。学习重点是把握文章的感情基调，学习朗读技巧（语气、节奏），体会细节描写中的情感。

- 5 秋天的怀念 / 史铁生
- 6 散步 / 莫怀戚
- 7\* 散文诗二首：金色花 / 泰戈尔、荷叶·母亲 / 冰心
- 8 《世说新语》二则 / 刘义庆：咏雪、陈太丘与友期行

## Anki 记忆卡

### 卡1

**配图：** ![](第二单元-至爱亲情/01.webp)

**提问：** 《秋天的怀念》的作者是谁？请简述其人生经历特点。

**答案：** 史铁生，当代作家。双腿瘫痪后，在母亲鼓励下开始写作。代表作：《我与地坛》《务虚笔记》《病隙碎笔》。本文是一篇回忆性散文，表达对母亲的深切怀念。

**易错分析：** 史铁生是"瘫痪"不是"残疾"；他是"当代作家"不是"现代"（20世纪下半叶以后在世的）。"瘫痪"的"瘫"读 tān。

**记忆钩子：** 史铁生瘫后写作，我与地坛怀母亲。

### 卡2

**配图：** ![](第二单元-至爱亲情/02.webp)

**提问：** 《秋天的怀念》中"憔悴""烂漫""诀别"的意思和读音？

**答案：**
- 憔悴（qiáo cuì）：形容人瘦弱、面色不好看。
- 烂漫（làn màn）：颜色鲜明而美丽；也指坦率自然（天真烂漫）。
- 诀别（jué bié）：告别、离别（多指不易再见的离别）。

**易错分析：** "憔悴"不要写成"焦脆"或"瞧粹"；"诀别"的"诀"是言字旁，不是"决"。"烂漫"的"漫"读 màn，不是 mán。

**记忆钩子：** 憔悴面黄诀别离，烂漫花开忆母时。

### 卡3

**配图：** ![](第二单元-至爱亲情/03.webp)

**提问：** 《秋天的怀念》中母亲为什么对"看花"那么执着？

**答案：** 母亲坚持要推"我"去北海看菊花，是想让"我"从菊花傲霜绽放的生命力中得到鼓舞，重新燃起对生活的希望，不再沉浸在瘫痪的痛苦和绝望中。体现了母亲深沉无私的爱和良苦用心。

**易错分析：** "看花"不只是散心，菊花象征顽强生命力，母亲是想借此鼓励儿子。文中两次提到看花，第一次未去成（母亲病重），第二次去时母亲已去世。

**记忆钩子：** 菊花傲霜兴生命，母唤儿子莫绝望。

### 卡4

**配图：** ![](第二单元-至爱亲情/04.webp)

**提问：** 《秋天的怀念》中"母亲扑过来抓住我的手，忍住哭声说"一句，从哪些词能看出母爱的特点？

**答案：** "扑""抓"两个动词写出母亲看到儿子痛苦时的急切和心疼，"忍"字最能体现母亲的坚韧——她强忍住自己的悲伤，不让负面情绪影响儿子。这三个动词写出母爱既热烈深沉又克制隐忍。

**易错分析：** 只答"写了母爱"太笼统，要具体分析"扑"的急切、"抓"的紧张、"忍"的克制三层。"忍"是全文最打动人的字。

**记忆钩子：** 扑抓忍三字，母爱深且韧。

### 卡5

**配图：** ![](第二单元-至爱亲情/05.webp)

**提问：** 《散步》的作者是谁？文章写了什么内容？

**答案：** 莫怀戚，当代作家。文章通过一家四口在初春田野散步的小事——母亲要走大路，儿子要走小路，最终"我"决定走大路（尊老），母亲却改口走小路（爱幼），展现了中华民族尊老爱幼的传统美德和中年人的家庭责任感。

**易错分析：** 作者名"莫怀戚"——"戚"字容易写成"诚"或"戚"字写错。文章主题不只是"亲情"，还升华到"中年人的责任与担当"（背起整个世界）。

**记忆钩子：** 莫怀戚散步，尊老又爱幼。

### 卡6

**配图：** ![](第二单元-至爱亲情/06.webp)

**提问：** 《散步》中"我背上的是整个家，整个世界"如何理解？

**答案：** "我"背起母亲，妻子背起儿子，这象征着中年一代对家庭的责任与担当——上要赡养老人，下要抚养孩子，肩负着传承生命、延续家庭的重任。一个家庭就是一个小世界。

**易错分析：** "整个世界"是比喻（以小见大），不是说真的世界。"我"和妻子处在中间位置，承上启下，代表中年人。

**记忆钩子：** 背母背子是责任，一家就是小世界。

### 卡7

**配图：** ![](第二单元-至爱亲情/07.webp)

**提问：** 《散步》一文运用了什么写作手法？

**答案：** ①"以小见大"——从一家人散步的小事，表现尊老爱幼的传统美德和中年人的责任感。②大词小用——用"分歧""责任重大""整个世界"等大词写家庭小事，使文章有哲理性。③对称句——"母亲要走大路，儿子要走小路"等，富有节奏感。

**易错分析：** "大词小用"是本文最突出的语言特色，不要漏答。修辞方面有对仗/对称。

**记忆钩子：** 小事见大，大词小用，对称成趣。

### 卡8

**配图：** ![](第二单元-至爱亲情/08.webp)

**提问：** 《金色花》的作者是谁？哪个国家？体裁是什么？

**答案：** 泰戈尔，印度诗人、作家。1913年获诺贝尔文学奖（亚洲第一位）。代表作《吉檀迦利》《飞鸟集》《新月集》。本文出自《新月集》，体裁是散文诗。

**易错分析：** 泰戈尔是印度人，不是泰国或印尼。"金色花"是印度圣树。体裁是"散文诗"——兼有散文和诗的特点，不能只答"散文"或"诗"。泰戈尔不是英国人，是印度人但用孟加拉语写作。

**记忆钩子：** 泰戈尔印度诺贝尔，金色花出新月集。

### 卡9

**配图：** ![](第二单元-至爱亲情/09.webp)

**提问：** 《金色花》中孩子为什么想变成一朵金色花？

**答案：** 孩子想变成一朵金色花，悄悄地陪伴在母亲身边，和母亲玩捉迷藏——让母亲闻到花香、在书页上看到花瓣投影，却不告诉母亲自己就是那朵花。这表现了孩子对母亲纯真、俏皮、深沉的依恋和爱。

**易错分析：** 变成金色花是为了"悄悄地爱母亲"，不是炫耀或调皮。文章以孩子的口吻写，充满童趣、想象和温馨。

**记忆钩子：** 变花悄悄陪妈妈，纯真童稚爱无价。

### 卡10

**配图：** ![](第二单元-至爱亲情/10.webp)

**提问：** 《荷叶·母亲》的作者是谁？文中"荷叶"和"红莲"分别象征什么？

**答案：** 冰心，原名谢婉莹，现代作家、诗人。代表作《繁星》《春水》《寄小读者》。"荷叶"象征母亲（保护者），"红莲"象征子女（被保护者）。

**易错分析：** 冰心原名"谢婉莹"不是"谢婉"或"谢冰心"。文中"荷叶"是保护者（母亲），"红莲"是被保护的孩子——不要混淆。这是一首散文诗，不是普通散文。

**记忆钩子：** 冰心谢婉莹，荷叶护红莲如母亲护子。

### 卡11

**配图：** ![](第二单元-至爱亲情/11.webp)

**提问：** "母亲啊！你是荷叶，我是红莲。心中的雨点来了，除了你，谁是我在无遮拦天空下的荫蔽？"（《荷叶·母亲》）赏析这段文字。

**答案：** 运用了比喻和反问。把母亲比作荷叶，自己比作红莲，把人生风雨（挫折、困难）比作"心中的雨点"，生动形象地写出了母亲是孩子唯一的庇护和依靠。反问句加强了语气，表达了对母亲深深的感激和依恋。

**易错分析：** "心中的雨点"比喻人生挫折，而不是真的下雨。"荫蔽"提供庇护，和"遮蔽"稍有不同。三个比喻：荷叶→母，红莲→我，雨点→困难。

**记忆钩子：** 母是荷叶我红莲，雨点来时母是伞。

### 卡12

**配图：** ![](第二单元-至爱亲情/12.webp)

**提问：** 《世说新语》二则出自哪部作品？作者是谁？体裁是什么？

**答案：** 出自《世说新语》，编者刘义庆（南朝宋文学家）。《世说新语》是一部笔记小说/志人小说集，主要记载东汉末至东晋士大夫的言谈逸事。

**易错分析：** 刘义庆是"编者"（组织编纂），不是完全个人创作。"南朝宋"（刘宋）不是"宋朝"（赵宋），两者相差600多年。《世说新语》是志人小说，不是志怪小说（如《搜神记》）。

**记忆钩子：** 刘义庆南朝宋，世说新语记名士。

### 卡13

**配图：** ![](第二单元-至爱亲情/13.webp)

**提问：** 《咏雪》中"撒盐空中差可拟"和"未若柳絮因风起"哪个更好？为什么？

**答案：** 谢道韫的"柳絮因风起"更好。因为柳絮飘飞与雪花飘落的形态更相似（轻盈、飘逸），且柳絮给人以春天的联想，富有美感。而"撒盐"过于沉重，缺少雪花轻盈的质感。所以"柳絮"比喻更贴切、更有诗意。

**易错分析：** "差可拟"的"差"读 chā（差不多），不是 chà。"因风起"的"因"是"凭借、乘着"的意思。"柳絮因风起"表现出谢道韫（谢太傅侄女）的文学才华，后来称女子有文才为"咏絮之才"。

**记忆钩子：** 柳絮因风轻飘飞，咏絮之才谢道韫。

### 卡14

**配图：** ![](第二单元-至爱亲情/14.webp)

**提问：** 《陈太丘与友期行》中"日中不至，则是无信；对子骂父，则是无礼"是谁说的？说明了什么道理？

**答案：** 是陈元方（陈太丘之子）说的。说明做人要讲诚信（守时）、懂礼貌（尊重别人），这两点是立身处世的基本原则。元方时年七岁，但明理善辩，也说明家庭教育的重要。

**易错分析：** 说话者是"元方"（陈纪，字元方），不是陈太丘。太丘是官职名（太丘长），不是人名。"期日中"是"约定在正午"，"期"是约定。"相委而去"的"委"是"丢下、舍弃"。

**记忆钩子：** 元方七岁明信礼，不守信则失礼。

### 卡15

**配图：** ![](第二单元-至爱亲情/15.webp)

**提问：** 《世说新语》二则中重点字词解释：俄而、欣然、未若、期行、去后乃至、相委而去、下车引之、入门不顾。

**答案：**
- 俄而：不久，一会儿。
- 欣然：高兴的样子（欣：高兴；然：……的样子）。
- 未若：不如，比不上。
- 期行：约定同行（期：约定）。
- 去后乃至：离开后才到（去：离开；乃：才）。 
- 相委而去：丢下我走了（委：丢下；去：离开）。
- 下车引之：下车拉他（引：拉/牵）。
- 入门不顾：进门不回头看（顾：回头看）。

**易错分析：** "去"在古文中是"离开"，不是"到某地去"；"顾"是"回头看"，不是"照顾"；"相"这里指"我"（偏指一方）。文言文中的古今异义词要特别注意。

**记忆钩子：** 去是离开顾是看，委是丢下期是约。

### 卡16

**配图：** ![](第二单元-至爱亲情/16.webp)

**提问：** 第二单元写作主题是什么？记事作文要注意什么？

**答案：** 写作主题是"学会记事"。记事要注意：①写清楚事情的起因、经过、结果；②详写经过（高潮部分），略写起因和结果；③加入细节描写（动作、语言、神态、心理），让记事更生动；④在叙事中融入情感。

**易错分析：** 记事作文最容易犯的毛病是"流水账"，经过不详细。要答出"详略得当"，着重写经过。

**记忆钩子：** 起因经过结果，详经过略首尾加细节。

### 卡17

**配图：** ![](第二单元-至爱亲情/17.webp)

**提问：** 《金色花》和《荷叶·母亲》都写母爱，写法有什么不同？

**答案：** 《金色花》以孩子的视角，通过想象自己变成金色花和母亲嬉戏，表现孩子的天真和对母亲的依恋，活泼俏皮。《荷叶·母亲》以女儿的视角，借雨打红莲、荷叶庇护的眼前之景，引出对母爱的赞颂，含蓄深沉。

**易错分析：** 两文都是散文诗、都写母爱，但角度不同：金色花的"母爱"是从孩子的调皮陪伴来表现；荷叶的"母爱"是从景生情、直抒胸臆。

**记忆钩子：** 金色童趣荷叶深，两样母爱一样情。
