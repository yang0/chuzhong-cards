# 第一单元 四季美景

## 基本信息

- 科目：语文
- 年级：七年级上册
- 教材：统编版 / 人教版《语文》（2024 秋版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"四季美景"为主题，选编四篇写景抒情散文和四首古代诗歌。学习重点是品味语言、体会修辞（比喻、拟人）的表达效果，通过朗读感受景物之美。

- 1 春 / 朱自清
- 2 济南的冬天 / 老舍
- 3\* 雨的四季 / 刘湛秋
- 4 古代诗歌四首：观沧海 / 曹操、次北固山下 / 王湾、闻王昌龄左迁龙标遥有此寄 / 李白、天净沙·秋思 / 马致远

## Anki 记忆卡

### 卡1

**配图：** ![](第一单元-四季美景/01-zhuziqing-chun.png)

**提问：** 《春》的作者是谁？出自哪部作品集？作者是什么家？

**答案：** 朱自清，字佩弦，江苏扬州人，散文家、诗人、学者。代表作《背影》《荷塘月色》《匆匆》。本文出自《朱自清全集》。

**易错分析：** 朱自清是"散文家、诗人、学者"，不要漏写"学者"。他是江苏扬州人，不是北京人。

**记忆钩子：** 佩弦散文家，背影荷塘春。

### 卡2

**配图：** ![](第一单元-四季美景/02-chun-vocab.png)

**提问：** 《春》中"酝酿"怎么读？什么意思？"应和""黄晕""蓑笠"呢？

**答案：**
- 酝酿（yùn niàng）：造酒的发酵过程，比喻做准备工作；文中指各种气息在空气里像发酵一样越来越浓。
- 应和（yìng hè）：呼应、附和。
- 黄晕（huáng yùn）：昏黄不明的色彩。
- 蓑笠（suō lì）：蓑衣和斗笠，防雨用具。

**易错分析：** "酝酿"不读"wēn niàng"；"应和"的"和"读 hè，不是 hé；"黄晕"的"晕"读 yùn，不是 yūn。

**记忆钩子：** 酝（yùn）酿（niàng）别读温，应和（hè）别读合。

### 卡3

**配图：** ![](第一单元-四季美景/03-chun-writing.png)

**提问：** 《春》中作者描绘了哪几幅春日图景？分别从哪些角度写？

**答案：** 五幅图：春草图（草）、春花图（花）、春风图（风）、春雨图（雨）、迎春图（人）。从物的角度展现春天特征，最后落到人的活动上。

**易错分析：** 不能漏掉"迎春图"，五幅图由自然景物到人的活动，层层递进。顺序也不能搞反。

**记忆钩子：** 草花风雨人——五幅春图。

### 卡4

**配图：** ![](第一单元-四季美景/04-chun-rhetoric.png)

**提问：** 《春》中"像牛毛，像花针，像细丝"用的是什么修辞？有什么表达效果？

**答案：** 比喻和排比。把春雨比作牛毛（密）、花针（亮）、细丝（细），生动形象地写出了春雨细密、闪亮、柔长的特点；排比句式增强了语势和节奏感。

**易错分析：** 问"修辞"时要说全"比喻和排比"，不能只答一种。要分析出分别比作什么、各自的特点。

**记忆钩子：** 牛毛密花针亮细丝长，比喻加排比。

### 卡5

**配图：** ![](第一单元-四季美景/05-jinan-winter-lao-she.png)

**提问：** 《济南的冬天》的作者是谁？文中写出了济南冬天怎样的特点？

**答案：** 老舍，原名舒庆春，字舍予，北京人，现代作家。代表作《骆驼祥子》《四世同堂》《茶馆》。济南冬天的特点是"温晴"，通过对比北平、伦敦、热带突出济南冬天"温晴"的独特性。

**易错分析：** 老舍原名"舒庆春"，不是"舒春"；字"舍予"；他是满族人，北京人，不是济南人。特点答"温晴"不是"温暖"或"晴朗"，"温晴"是温暖晴朗、温和宜人的综合感觉。

**记忆钩子：** 老舍原名舒庆春，济南冬天温晴好。

### 卡6

**配图：** ![](第一单元-四季美景/06-jinan-winter-scene.png)

**提问：** 《济南的冬天》描写了哪些景物？按什么顺序写的？

**答案：** 描写了阳光朗照下的山、薄雪覆盖下的山、城外远山、济南的水四幅画面。按由暖阳到山色再到水色的空间顺序，从城内写到城外。

**易错分析：** 四幅画面缺一不可。"薄雪覆盖下的山"是重点段落，写的是小雪后的小山，不是大雪。

**记忆钩子：** 阳光山、薄雪山、远山、水，四个画面。

### 卡7

**配图：** ![](第一单元-四季美景/07-jinan-winter-comparison.png)

**提问：** 《济南的冬天》开头用了什么写作手法？

**答案：** 对比。将济南的冬天与北平（风大）、伦敦（雾多）、热带（日光毒）对比，突出济南冬天"温晴"的特点，表达对济南冬天的喜爱。

**易错分析：** 分析对比时要写出"和什么比"、"比出什么"，不能只答"用了对比"。

**记忆钩子：** 北风伦敦雾热带毒，对比出济南温晴。

### 卡8

**配图：** ![](第一单元-四季美景/08-rain-seasons.png)

**提问：** 《雨的四季》的作者是谁？文中分别描写了哪几个季节的雨？各有什么特点？

**答案：** 刘湛秋，当代诗人、翻译家。四个季节的雨：春雨（温柔、娇媚）、夏雨（热烈、粗犷、奔放）、秋雨（沉静、端庄、深情）、冬雨（自然、平静、纯洁）。

**易错分析：** 作者名容易写错别字："湛"是三点水+甚，不是"堪"或"甚"。四季的雨的形容词要对应准确，不能互换，如夏雨用"粗犷"不用"温柔"。

**记忆钩子：** 春雨柔夏雨犷秋雨静冬雨洁。

### 卡9

**配图：** ![](第一单元-四季美景/09-rain-seasons-rhetoric.png)

**提问：** "每一棵树仿佛都睁开特别明亮的眼睛"（《雨的四季》）用了什么修辞？

**答案：** 拟人。把树人格化，赋予树以人的动作（睁开明亮的眼睛），生动地写出了春雨过后树木焕然一新的生机。

**易错分析：** 不要只答"拟人"就完了，要写出"把什么比作什么"或"赋予了人怎样的动作"。为什么是"睁开的眼睛"？那是雨后树叶上的水珠在阳光下闪亮。

**记忆钩子：** 树睁眼睛，雨后生机。

### 卡10

**配图：** ![](第一单元-四季美景/10-guancanghai-caocao.png)

**提问：** 《观沧海》的作者是谁？哪个朝代？全诗哪里是写景的句子？

**答案：** 曹操，字孟德，东汉末年政治家、军事家、诗人。《观沧海》是《步出夏门行》中的一章。写景句："水何澹澹，山岛竦峙。树木丛生，百草丰茂。秋风萧瑟，洪波涌起。"

**易错分析：** 曹操是"东汉末年"不是"三国"或"汉代"，东吴蜀汉还没建立。"东临碣石"的"碣"读 jié，不是 jiē。"澹澹"读 dàn dàn。全诗最后两句"幸甚至哉，歌以咏志"是套语，不是内容主体。

**记忆钩子：** 孟德曹操东汉末，观海写景水山木草风浪。

### 卡11

**配图：** ![](第一单元-四季美景/11-cigu-beigu-caocao.png)

**提问：** 《观沧海》中"日月之行，若出其中；星汉灿烂，若出其里"用了什么手法？表达了什么？

**答案：** 虚写/想象/夸张的手法。诗人通过奇特的想象，写出日月星辰都好像在大海中运行，表现了大海吞吐日月、包蕴星汉的壮阔气势，抒发了诗人一统天下的雄心壮志和博大胸襟。

**易错分析：** 这句不是实写眼前之景，是虚写（想象）！不要答成"比喻"或"拟人"。"沧海"指大海，不是"苍茫的海"。

**记忆钩子：** 日月出海中，曹操壮志雄。

### 卡12

**配图：** ![](第一单元-四季美景/12-ci-beigu-wangwan.png)

**提问：** 《次北固山下》的作者是谁？哪个朝代？"次"是什么意思？

**答案：** 王湾，唐代诗人。"次"是停宿、停泊的意思。题目意思是停泊在北固山下。

**易错分析：** 作者是"王湾"不是"王维"（两人都是唐代诗人，容易混淆）。"次"表"停泊"不是"次序"或"次数"。

**记忆钩子：** 王湾不是王维，次是停泊。

### 卡13

**配图：** ![](第一单元-四季美景/13-ci-beigu-famous.png)

**提问：** "海日生残夜，江春入旧年"（《次北固山下》）好在哪里？

**答案：** ①写景：残夜未退红日已升，旧年未过江上已露春意。②哲理：旧事物中孕育着新事物，新事物必将取代旧事物，给人以积极向上的力量。③对仗工整，"海日"对"江春"，"生"对"入"，"残夜"对"旧年"。

**易错分析：** 赏析古诗不能只答"写景很美"，要答出深层含义——新旧交替的哲理。"生"和"入"用了拟人手法，使景物有生命力。

**记忆钩子：** 海日生于残夜，江春入于旧年——新旧交替。

### 卡14

**配图：** ![](第一单元-四季美景/14-wen-wangchangling.png)

**提问：** 《闻王昌龄左迁龙标遥有此寄》的作者朝代及题目关键词解释？

**答案：** 李白，唐代诗人。"左迁"是贬官。"龙标"有两个含义：①指王昌龄被贬的地方（龙标县）；②题目中代指王昌龄（用官职称呼人），诗中"随君直到夜郎西"的"君"才指王昌龄本人。

**易错分析：** "左迁"是"贬官降职"，不要答成"向左迁移"或"升官"。古代以右为尊，左为卑。"龙标"既指地名也借代指人，是双关。

**记忆钩子：** 左迁贬官李白送，龙标代指王昌龄。

### 卡15

**配图：** ![](第一单元-四季美景/15-wen-wangchangling-appreciation.png)

**提问：** "我寄愁心与明月，随君直到夜郎西"（《闻王昌龄左迁龙标遥有此寄》）赏析。

**答案：** 运用了拟人手法（赋予明月以人的情感），将"愁心"托付给明月，让明月代替自己陪伴友人，表达了对友人的同情、牵挂和思念之情。想象奇特，情感真挚。

**易错分析：** 这里把"愁心"寄给"明月"，是把抽象的情感具象化，也是化虚为实的手法。不要只答拟人。"夜郎"指唐代西南地区地名，不是贵州的夜郎国。

**记忆钩子：** 愁心寄明月，随君到夜郎。

### 卡16

**配图：** ![](第一单元-四季美景/16-tianjingsha-autumn.png)

**提问：** 《天净沙·秋思》的体裁是什么？作者是谁？哪个朝代？

**答案：** 体裁是元曲（小令），"天净沙"是曲牌名，"秋思"是题目。作者马致远，号东篱，元代戏曲作家、散曲家，被誉为"秋思之祖"。

**易错分析：** 这是"元曲"不是"宋词"。"天净沙"是曲牌名，不是题目。"秋思"的"思"读 sì（四声），不是 sī。

**记忆钩子：** 马致远元曲，秋思之祖天净沙。

### 卡17

**配图：** ![](第一单元-四季美景/17-tianjingsha-content.png)

**提问：** 《天净沙·秋思》中写了哪些意象？表达了什么情感？

**答案：** 意象：枯藤、老树、昏鸦、小桥、流水、人家、古道、西风、瘦马、夕阳、断肠人。表达了漂泊在外的游子思念家乡的孤寂悲苦之情。

**易错分析：** 意象不能只列几个，完整的11个意象构成了深秋黄昏的荒凉画面。"断肠人"指漂泊天涯的游子自己。全诗最后一句"夕阳西下，断肠人在天涯"是点名主旨的句子。"小桥流水人家"以温馨反衬游子的凄凉，是反衬手法。

**记忆钩子：** 枯藤老树昏鸦古道瘦马夕阳，断肠人在天涯思故乡。

### 卡18

**配图：** ![](第一单元-四季美景/18-tianjingsha-rhetoric.png)

**提问：** 《天净沙·秋思》最主要的艺术特色是什么？

**答案：** 借景抒情、情景交融。全诗前三句全是名词性意象（不用动词连接），直接罗列九种深秋景物，营造出萧瑟凄凉的氛围，最后一句点出断肠人，将景与情完美融合，被称为"秋思之祖"。

**易错分析：** 不用动词直接罗列名词意象——这是元曲的典型手法，叫"列锦"。不要答成比喻或拟人。

**记忆钩子：** 九景列锦不连词，景中自含思乡情。

### 卡19

**配图：** ![](第一单元-四季美景/19-unit1-writing.png)

**提问：** 第一单元写作主题是什么？写景作文要注意什么？

**答案：** 写作主题是"热爱写作，学会观察"。写景要注意：①仔细观察，抓住景物特征；②调动多种感官（视觉、听觉、嗅觉、触觉）；③运用比喻、拟人等修辞手法；④融情于景（景中含情）。

**易错分析：** 旧版是"热爱生活，热爱写作"，2024版改为"热爱写作，学会观察"，强调"观察"。答题要以新版为准。

**记忆钩子：** 仔细观察抓特征，多感官加修辞。

### 卡20

**配图：** ![](第一单元-四季美景/20-ancient-poetry-compare.png)

**提问：** 第一单元四首古代诗歌分别表达了什么主要情感？

**答案：** 《观沧海》：借大海壮阔抒统一天下的雄心壮志。《次北固山下》：写旅途思乡，同时蕴含新旧交替的哲理。《闻王昌龄左迁龙标遥有此寄》：表达对友人被贬的同情和牵挂。《天净沙·秋思》：游子漂泊在外的孤寂思乡之情。

**易错分析：** 四首诗歌情感不同，不可混淆。《次北固山下》兼有思乡和哲理两层；《观沧海》是壮志而非单纯的写景。

**记忆钩子：** 曹操壮志王湾思乡，李白寄愁致远断肠。
