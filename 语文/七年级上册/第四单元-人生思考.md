# 第四单元 人生思考

## 基本信息

- 科目：语文
- 年级：七年级上册
- 教材：统编版 / 人教版《语文》（2024 秋版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"人生思考"为主题，选编四篇文章，从不同角度阐释人生的价值与意义。学习重点是通过划分段落层次、抓关键词句来理清思路，理解作者的观点和情感。

- 13 纪念白求恩 / 毛泽东
- 14 回忆我的母亲 / 朱德（原八年级上册）
- 15\* 梅岭三章 / 陈毅（原九年级下册）
- 16 诫子书 / 诸葛亮

## Anki 记忆卡

### 卡1

**配图：** ![](第四单元-人生思考/01.webp)

**提问：** 《纪念白求恩》的作者是谁？白求恩是谁？

**答案：** 毛泽东。白求恩是加拿大共产党员、胸外科医生，受加拿大共产党和美国共产党派遣，率医疗队到中国抗日根据地支援，在抢救伤员时感染中毒性败血症而牺牲。

**易错分析：** 白求恩是"加拿大"人不是美国人。他是"胸外科医生"不是"全科医生"。"派遣"的"遣"读 qiǎn，不是 qián。

**记忆钩子：** 毛泽东书白求恩，加拿大医生援华牺牲。

### 卡2

**配图：** ![](第四单元-人生思考/02.webp)

**提问：** 《纪念白求恩》中毛泽东号召学习白求恩的哪些精神？

**答案：** ①国际主义精神（毫无利己专门利人的精神）；②毫不利己专门利人的精神（对工作极端负责、对人民极端热忱）；③对技术精益求精的精神。

**易错分析：** 原文的三个"精神"要点要答全。文章还运用了"对比"手法——将白求恩与"见异思迁""鄙薄技术工作"的人对比，突出白求恩的高尚。"拈轻怕重"的"拈"读 niān。

**记忆钩子：** 国际主义负责热忱，精益求精医术精。

### 卡3

**配图：** ![](第四单元-人生思考/03.webp)

**提问：** 《纪念白求恩》主要运用了什么写作手法？

**答案：** 对比。第二段将白求恩与"不少的人"对比——白求恩对工作极端负责、对人民极端热忱，而不少人"拈轻怕重""冷冷清清""漠不关心"；结尾将白求恩与"见异思迁""鄙薄技术工作"的人对比。通过对比，突出了白求恩的高尚品格。

**易错分析：** 本文是"议论文"（悼念性议论文），不是记叙文。对比是主要写法，但也用了叙议结合的手法。

**记忆钩子：** 对比突出白求恩，叙议结合显深刻。

### 卡4

**配图：** ![](第四单元-人生思考/04.webp)

**提问：** 《回忆我的母亲》的作者是谁？体裁是什么？（2024年从八上调至七上）

**答案：** 朱德，字玉阶，伟大的无产阶级革命家、军事家，中国人民解放军主要缔造者之一。体裁是回忆性散文。文章回忆了母亲勤劳朴实的一生，表达了对母亲的崇敬和怀念。

**易错分析：** 朱德是"革命家、军事家"不是"文学家"。"回忆性散文"不等同于一般的写人记叙文。这是朱德写的，不是别人写朱德。

**记忆钩子：** 朱德忆母勤劳一生，2024新调入七上。

### 卡5

**配图：** ![](第四单元-人生思考/05.webp)

**提问：** 《回忆我的母亲》中朱德的母亲具有怎样的品格？

**答案：** ①勤劳俭朴（天不亮就起床、煮饭、种田、养蚕、纺棉花）；②任劳任怨、宽厚仁慈（和蔼待人、周济穷人）；③坚强刚毅（支持朱德革命、不拖累儿子）；④深明大义（知道儿子做的是大事，不因自己耽误革命）。

**易错分析：** 不要只答"勤劳善良"，要具体到文中的事例。"任劳任怨"和"宽厚仁慈"的分点要清楚。母亲支持儿子革命是深明大义。

**记忆钩子：** 勤劳俭朴任劳怨，刚毅明义支持儿。

### 卡6

**配图：** ![](第四单元-人生思考/06.webp)

**提问：** 《回忆我的母亲》在语言上有什么特点？

**答案：** ①朴素真挚——语言朴实无华，没有华丽辞藻，但情感深沉；②于细微处见真情——通过日常琐事（纺线、挑水、做饭）展现母爱；③结尾升华——"尽忠于民族、尽忠于党"将母亲与民族命运联系起来，由个人到国家。

**易错分析：** 结尾"尽忠于民族和人民，尽忠于党"不是套话，而是朱德将个人情感升华为革命信念的表达。

**记忆钩子：** 朴实话语出真情，由家到国尽忠诚。

### 卡7

**配图：** ![](第四单元-人生思考/07.webp)

**提问：** 《梅岭三章》的作者是谁？体裁是什么？写于什么背景下？（2024年从九下调至七上）

**答案：** 陈毅，无产阶级革命家、军事家、外交家，中国人民解放军元帅。体裁是格律诗（七言绝句三首）。写于1936年冬，陈毅在梅岭被国民党军队围困，伤病伏丛莽间二十余日，虑不得脱，写了一组诗藏在衣底，表现了视死如归的革命气节和革命必胜的坚定信念。

**易错分析：** 1936年陈毅在江西梅岭被围困，此时是长征后南方游击战时期。"泉台""阎罗""烽烟""血雨腥风"等都是借代/比喻。"旌旗"借代军队，"烽烟"借代战争。三首诗是一个整体——第一首写决心牺牲，第二首写革命信念，第三首写革命必胜。

**记忆钩子：** 陈毅梅岭被围，绝笔诗明志见气节。

### 卡8

**配图：** ![](第四单元-人生思考/08.webp)

**提问：** "此去泉台招旧部，旌旗十万斩阎罗"（《梅岭三章》）如何理解？

**答案：** 运用了借代和想象的手法。"泉台"指黄泉、阴间，"阎罗"借指反动派/国民党统治者，"旌旗"借代军队。诗人想象即使牺牲到了阴间，也要召集旧部继续战斗，斩除阎罗。表达了视死如归的英雄气概和与敌人斗争到底的坚定信念。

**易错分析：** 这不是迷信，是用传统意象表达革命气节。"招旧部"和"斩阎罗"体现的是不投降、不屈服的精神。

**记忆钩子：** 死后也要召旧部，斩阎罗战到底。

### 卡9

**配图：** ![](第四单元-人生思考/09.webp)

**提问：** 《梅岭三章》中"南国烽烟正十年"和"投身革命即为家"分别有什么含义？

**答案：** "南国烽烟正十年"——陈毅从1926年参加革命到1936年写诗时正好十年，概括了南方革命战争的艰苦历程。"投身革命即为家"——以革命事业为家，意味着革命者四海为家、随时准备牺牲，表达了坚定的革命信念和为民奉献的精神。

**易错分析：** "十年"不是确指十年整，是"十年来"的概数。"血雨腥风"比喻残酷镇压。"取义成仁"借用传统文化中的"杀身成仁、舍生取义"。

**记忆钩子：** 南国烽烟十年战，革命为家志不移。

### 卡10

**配图：** ![](第四单元-人生思考/10.webp)

**提问：** 《诫子书》的作者是谁？体裁是什么？写给谁的？

**答案：** 诸葛亮，字孔明，号卧龙，三国时期蜀汉政治家、军事家。体裁是书信（家书）。是诸葛亮写给儿子诸葛瞻的告诫信，教导儿子修身养性、立志成才。

**易错分析：** 诸葛亮是"三国时期"不是"东汉"或"蜀国"。"诫子书"的"诫"是告诫、劝诫。"子"指儿子诸葛瞻，不是泛指子女。

**记忆钩子：** 诸葛亮诫子书，孔明教子修身篇。

### 卡11

**配图：** ![](第四单元-人生思考/11.webp)

**提问：** 《诫子书》中"静以修身，俭以养德"是什么意思？全文的主旨是什么？

**答案：** "静以修身"——以宁静（内心专注）来修养身心；"俭以养德"——以节俭来培养品德。全文主旨：告诫儿子要宁静淡泊、勤学立志，避免懈怠和浮躁，才能成才成器。

**易错分析：** "静"不是"安静"，而是"内心宁静专注、摒除杂念"。"俭"不仅是生活简朴，还有"节制、约束"的意思。注意"淫慢则不能励精"——"淫"是"放纵"，"慢"是"懈怠"。

**记忆钩子：** 静修身俭养德，淡泊明志宁静致远。

### 卡12

**配图：** ![](第四单元-人生思考/12.webp)

**提问：** 《诫子书》重点字词解释：夫、行、澹泊、无以、明志、致远、广才、淫慢、险躁、穷庐。

**答案：**
- 夫（fú）：句首发语词，无实义。
- 行：操守、品行。
- 澹泊（dàn bó）：内心恬淡，不慕名利。
- 无以：无法、没有用来……的。
- 明志：明确志向。
- 致远：实现远大目标（致：达到）。
- 广才：增长才干（广：增长）。
- 淫慢：放纵懈怠（淫：放纵；慢：懈怠）。
- 险躁：轻薄浮躁（险：轻浮）。
- 穷庐：穷困潦倒的居所（穷庐将复何及）。

**易错分析：** "夫"读 fú（阳平），不是 fū。"俭"不是仅仅"省钱"。"遂成枯落"的"遂"是"最终"。"将复何及"——"又怎么来得及！"表达惋惜之情。

**记忆钩子：** 夫是发语词，淫放纵慢懈怠。

### 卡13

**配图：** ![](第四单元-人生思考/13.webp)

**提问：** "非澹泊无以明志，非宁静无以致远"（《诫子书》）如何理解？

**答案：** 双重否定句式（非……无以……），强调"澹泊"和"宁静"的重要性。如果内心不恬淡寡欲，就无法明确自己的志向；如果内心不宁静专注，就无法达到远大的目标。这两句是《诫子书》的核心名句，常用来表达修身立志的处世哲学。

**易错分析：** 这句话一般写作"非淡泊无以明志，非宁静无以致远"（教材用"澹"通"淡"）。默写高频句，"致远"不要写成"至远"。

**记忆钩子：** 淡泊明志宁静致远——诫子书千古名句。

### 卡14

**配图：** ![](第四单元-人生思考/14.webp)

**提问：** 《诫子书》中"静"与"躁"对比说明了什么？

**答案：** "静以修身，俭以养德"——静带来正面结果（修身养德）；"淫慢则不能励精，险躁则不能治性"——躁（放纵、懈怠、轻薄、浮躁）导致负面结果（不能励精、不能治性）。通过对比强调了"静"对于成才的核心作用，"躁"是必须克服的。

**易错分析：** 对比分析要写出"静→好结果"和"躁→坏结果"两面的对照关系。

**记忆钩子：** 静修德躁败业，对比见高下。

### 卡15

**配图：** ![](第四单元-人生思考/15.webp)

**提问：** 第四单元写作主题是什么？如何做到思路清晰？

**答案：** 写作主题是"思路要清晰"。要求：①整体构思（确定中心、取舍材料）；②确定写作顺序（时间顺序、空间顺序、逻辑顺序）；③列提纲（先写什么后写什么）；④运用过渡句使段落衔接自然。

**易错分析：** "思路清晰"和"突出中心"不同——前者重结构衔接，后者重主题聚焦。

**记忆钩子：** 定中心选材料，列提纲用过渡。

### 卡16

**配图：** ![](第四单元-人生思考/16.webp)

**提问：** 本单元四篇文章分别从哪些角度讨论"人生的价值"？

**答案：** 《纪念白求恩》——从"对他人"的角度：毫不利己专门利人，为人民服务才是人生的价值。《回忆我的母亲》——从"对家庭到对国家"的角度：勤劳奉献、支持革命，平凡中见伟大。《梅岭三章》——从"对信仰"的角度：视死如归，为革命牺牲是最高价值。《诫子书》——从"对自我"的角度：宁静淡泊、勤学立志、修身养德。

**易错分析：** 四篇文章角度不同，考试可能要求对比分析。要理解每篇的核心价值取向。

**记忆钩子：** 白求恩利人，朱德爱国，陈毅献身，诸葛亮修身。
