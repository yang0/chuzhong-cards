# 第五单元 人与动物

## 基本信息

- 科目：语文
- 年级：七年级上册
- 教材：统编版 / 人教版《语文》（2024 秋版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元为"活动·探究"单元，以"人与动物"为主题。学习重点是默读、理清思路、概括中心思想，体会人与动物关系的深层思考。本单元包含阅读、实践活动和写作三个任务。

- 任务一：体会人与动物的关系
  - 17 猫 / 郑振铎
  - 18 我的白鸽 / 陈忠实（新增）
  - 19 大雁归来 / 利奥波德
  - 20 狼 / 蒲松龄
- 任务二：亲近动物，丰富生命体验
- 任务三：记叙与动物的相处

## Anki 记忆卡

### 卡1

**配图：** ![](第五单元-人与动物/01-cat-zhengzhenduo.png)

**提问：** 《猫》的作者是谁？体裁是什么？本文讲述了一个什么故事？

**答案：** 郑振铎，现代作家、文学史家。体裁是散文。文章写了家中三次养猫的经历——前两只猫活泼可爱但最终亡失，第三只猫因"我"的误解被赶出而死，作者对此深感愧疚和自责，表达了不能凭主观臆断对待生命的反思。

**易错分析：** "郑振铎"的"铎"读 duó，不读 yì。三只猫中第三只是重点——它丑、不活泼、有"凝望"的忧郁神情，被"我"冤枉咬死芙蓉鸟后挨打，"我"事后发现真相但追悔莫及。

**记忆钩子：** 郑振铎养三猫，第三只最愧疚。

### 卡2

**配图：** ![](第五单元-人与动物/02-cat-structure.png)

**提问：** 《猫》中三只猫的命运有什么不同？作者的感情有什么变化？

**答案：** 第一只猫：活泼可爱→病死（感到酸辛）。第二只猫：更有趣→被过路人捉走（怅然、愤恨）。第三只猫：不活泼、不受欢迎→被"我"冤枉赶走→死在邻居家（良心受伤、深深自责）。情感脉络：喜爱→失落→喜爱→失落→冤枉→愧疚→教训。

**易错分析：** 第三只猫不是"打死的"——原文是"被棒打后过了两个月死在了邻居家"。作者的"良心受伤"是因为"我永不养猫"——"我"把自己的主观臆断强加给了无法辩解的生命。

**记忆钩子：** 一病死二被偷三冤死，永不养猫是忏悔。

### 卡3

**配图：** ![](第五单元-人与动物/03-cat-lesson.png)

**提问：** 《猫》告诉了我们什么道理？

**答案：** ①不能凭个人好恶和主观臆断去判断事物、对待生命；②做人和处事要客观公正，避免冤枉无辜；③对待生命要有平等、尊重的态度；④一旦犯错要勇于反思和自责。

**易错分析：** 主题不只是"爱护动物"，更深层是"反思人类的主观臆断"和"公平对待生命"。不要答偏到"保护动物"。

**记忆钩子：** 别主观臆断，冤枉了无辜永难弥补。

### 卡4

**配图：** ![](第五单元-人与动物/04-white-dove-chenzhongshi.png)

**提问：** 《我的白鸽》的作者是谁？体裁是什么？（2024新增）

**答案：** 陈忠实，当代作家，代表作《白鹿原》（获茅盾文学奖）。体裁是散文。本文记叙了作者养白鸽的经历，描写了白鸽的美丽和灵性，以及人与鸽之间建立的情感联系。

**易错分析：** 陈忠实是陕西作家，以《白鹿原》闻名。"忠实"的"实"字不要写成"石"。这是2024版新增课文（从原苏教版教材移入），考试可能涉及。

**记忆钩子：** 陈忠实白鹿原作者，白鸽寄情意趣深。

### 卡5

**配图：** ![](第五单元-人与动物/05-white-dove-content.png)

**提问：** 《我的白鸽》中白鸽有什么特点？作者对白鸽的态度是怎样的？

**答案：** 白鸽的特点是：美丽（洁白羽毛、红玛瑙似的眼睛）、灵性（通人性、能与人和睦相处）、自由（在天地间飞翔）。作者对白鸽充满喜爱、欣赏和珍视，白鸽给作者带来了精神上的慰藉和生活的乐趣。

**易错分析：** 这篇文章不是单纯写动物，而是写人与动物之间的情感纽带和相互信任。白鸽象征着纯洁、和平与美好。

**记忆钩子：** 白鸽美丽有灵性，人鸽互信情感深。

### 卡6

**配图：** ![](第五单元-人与动物/06-geese-return-leopold.png)

**提问：** 《大雁归来》的作者是谁？体裁是什么？文中大雁归来有什么特点？

**答案：** 利奥波德，美国生态学家、环境保护主义者，代表作《沙乡年鉴》。体裁是观察笔记/科学散文。文中大雁归来的特点是：有季节性（每年三月归来）、成群结队（以"群"为单位飞行）、叫声嘹亮（春雁的叫声与秋雁不同）、是有秩序的社会性动物。

**易错分析：** 利奥波德不是作家而是生态学家，本文属于科学观察笔记。2024版从原八年级下册调入。"缄默"的"缄"读 jiān，不读 xián。

**记忆钩子：** 利奥波德沙乡年鉴，三月大雁归来有序。

### 卡7

**配图：** ![](第五单元-人与动物/07-geese-return-theme.png)

**提问：** 《大雁归来》表达了怎样的思想？

**答案：** ①表达了对大雁的喜爱和赞美——大雁是春天的使者，是自然界的灵性生物；②倡导人与自然和谐相处——人类应尊重、保护自然和野生动物；③批判了对自然环境的破坏——大雁的生存受到人类活动的威胁。

**易错分析：** 本文核心是"生态保护"和"人与自然和谐"，不是单纯写动物习性。"三月"在文中多次出现，是大雁北归的标志时间。

**记忆钩子：** 爱大雁护自然，人与动物本应和谐。

### 卡8

**配图：** ![](第五单元-人与动物/08-wolf-pusongling.png)

**提问：** 《狼》的作者是谁？哪部作品集？体裁是什么？

**答案：** 蒲松龄，字留仙，号柳泉居士，清代文学家。选自文言短篇小说集《聊斋志异》。《聊斋志异》是一部文言志怪小说集，以花妖狐魅的故事反映社会现实。"聊斋"是蒲松龄书斋名，"志异"即记录奇异之事。

**易错分析：** 蒲松龄是"清代"不是"明朝"或"宋朝"。"聊斋"是书斋名，"志"是"记录"不是"志向"。《聊斋志异》是"志怪小说"。

**记忆钩子：** 蒲松龄聊斋志异，清代文言写狐妖狼。

### 卡9

**配图：** ![](第五单元-人与动物/09-wolf-content.png)

**提问：** 《狼》讲述了一个什么故事？本文告诉我们什么道理？

**答案：** 一个屠夫晚归路上遇到两只狼，狼想方设法追赶、前后夹击屠夫，屠夫由退让到奋起反击，最终杀死两只狼。道理：①面对像狼一样的恶势力，不能心存幻想、一味退让，要敢于斗争、善于斗争；②狼再狡猾也斗不过人的智慧。

**易错分析：** 故事中屠夫一开始"投以骨"（退让）是错误示范，后来"奔倚其下、弛担持刀"才是正确应对。文章末尾的议论"禽兽之变诈几何哉？止增笑耳"是点睛之笔。

**记忆钩子：** 屠夫先退让后持刀杀狼，恶势力终败于智慧。

### 卡10

**配图：** ![](第五单元-人与动物/10-wolf-keywords.png)

**提问：** 《狼》重点字词解释：缀、从、驱、窘、敌、去、股、寐、黠、顷刻。

**答案：**
- 缀（zhuì）：连接/紧跟。
- 从：跟随（一狼仍从）。
- 驱：追赶。
- 窘（jiǒng）：处境困迫。
- 敌：攻击（前后受其敌）。
- 去：离开（一狼径去）。
- 股：大腿（断其股）。
- 寐（mèi）：睡觉（乃悟前狼假寐）。
- 黠（xiá）：狡猾。
- 顷刻：一会儿（顷刻两毙）。

**易错分析：** "去"是"离开"不是"到……去"。"股"在古文是"大腿"，现代汉语是"屁股"，古今异义。"盖以诱敌"——"盖"是"原来/大概是"。"犬坐于前"——"犬"是名词作状语，"像狗一样蹲坐"。

**记忆钩子：** 缀跟驱追赶，黠狡猾寐睡觉。

### 卡11

**配图：** ![](第五单元-人与动物/11-wolf-writing.png)

**提问：** 《狼》在写作上有什么特点？

**答案：** ①情节紧张曲折——遇狼→惧狼→御狼→杀狼→悟狼，环环相扣；②运用动作描写和神态描写——"屠大窘""狼不敢前"等；③结尾议论点明主旨——"禽兽之变诈几何哉？止增笑耳"；④语言简练传神——全文仅200余字就把故事讲得完整生动。

**易错分析：** 情节发展阶段要分清楚：遇狼（开端）→惧狼（发展）→御狼（高潮）→杀狼（结局）→评狼（议论）。

**记忆钩子：** 遇惧御杀评五步，200字讲完整故事。

### 卡12

**配图：** ![](第五单元-人与动物/12-wolf-wuqi.png)

**提问：** "一狼洞其中，意将隧入以攻其后也"（《狼》）中"洞"和"隧"的用法？

**答案：** "洞"是名词用作动词——打洞。"隧"也是名词用作动词——从通道中钻入/钻洞。两只狼分工配合：一只假寐诱敌，一只钻洞偷袭，但最终都被屠夫杀死，突出狼的狡猾和屠夫的机智。

**易错分析：** 名词活用为动词是文言文常见语法现象。两狼的分工是考试常见考点。

**记忆钩子：** 洞为打洞隧为钻，狼再狡猾也枉然。

### 卡13

**配图：** ![](第五单元-人与动物/13-unit5-writing.png)

**提问：** 第五单元写作任务是什么？记叙与动物相处要注意什么？

**答案：** 写作任务是"记叙与动物的相处"。要注意：①写清楚与动物相处的具体经过（相识、相处、情感变化）；②抓住动物的特征进行描写（外形、习性、动作）；③在叙事中融入感情和思考（不仅是写动物，还要写出自己的感悟）；④可以适当运用对比、衬托等手法。

**易错分析：** 本单元是"活动·探究"单元，写作不是独立于阅读的——要借鉴《猫》《我的白鸽》等文的写法。

**记忆钩子：** 记相处抓特征，入情感写感悟。

### 卡14

**配图：** ![](第五单元-人与动物/14-unit5-explore.png)

**提问：** 第五单元"活动·探究"中任务二"亲近动物，丰富生命体验"需要做什么？

**答案：** 任务二是实践活动：①观察身边的一种动物（宠物、小区动物、动物园动物等），做好观察记录；②了解动物的生活习性、行为特点；③与动物互动，体验与动物相处的感受；④通过观察和互动，深化对人与自然关系的思考。

**易错分析：** 这是"活动·探究"单元的特色——三个任务（阅读→实践→写作）是一个整体，不是割裂的。

**记忆钩子：** 观察动物做记录，互动体验深思考。

### 卡15

**配图：** ![](第五单元-人与动物/15-unit4-animals-compare.png)

**提问：** 《猫》《我的白鸽》《大雁归来》《狼》四篇文章对动物的态度有什么不同？

**答案：** 《猫》——悔恨与反思（主观臆断冤枉动物，深刻自责）。《我的白鸽》——喜爱与珍视（与白鸽建立美好情感）。《大雁归来》——欣赏与保护（从生态学角度尊重自然生命）。《狼》——警惕与斗争（将狼视为恶势力的象征）。

**易错分析：** 《狼》是寓言式小说，狼不是真实的狼而是"恶势力"的象征，与其他三篇不同。前三篇是散文/科学散文，写真实的人与动物关系。

**记忆钩子：** 猫反思鸽喜爱，雁保护狼斗争。
