# 第四单元 修身正己

## 基本信息

- 科目：语文
- 年级：七年级下册
- 教材：统编版 / 人教版《语文》（2024 修订版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"修身正己"为主题，探讨为人处世的基本道理。学习重点是略读、把握文章的基本观点、筛选关键信息。

- 14 叶圣陶先生二三事 / 张中行
- 15 驿路梨花 / 彭荆风
- 16 最苦与最乐 / 梁启超
- 17 短文两篇：陋室铭 / 刘禹锡、爱莲说 / 周敦颐

## Anki 记忆卡

### 卡1

**提问：** 《叶圣陶先生二三事》的作者是谁？叶圣陶是谁？

**答案：** 张中行，学者、散文家。叶圣陶（原名叶绍钧），现代作家、教育家、编辑家，代表作《稻草人》《倪焕之》。叶圣陶被誉为"优秀的语言艺术家"，是中国现代语文教育的奠基人之一。

**易错分析：** 叶圣陶原名"叶绍钧"。"钧"不要写错。他是教育家——主持编写了新中国第一套语文教材。"躬行君子"出自《论语》——"躬行君子，则吾未之有得"。

**记忆钩子：** 张中行忆叶圣陶，语文教育奠基人。

### 卡2

**提问：** 《叶圣陶先生二三事》中写了叶圣陶哪些品质？

**答案：** ①待人厚——吕叔湘的文章请叶老修改，叶老改得认真仔细；②律己严——作文和做人要求自己一丝不苟；③重视语文规范——主张写文章要"简洁"，批评"做"和"作"混用；④待人真诚——对晚辈关怀爱护、从不摆架子。

**易错分析：** 叶老最核心的品质是"宽厚待人，严于律己"。"写话"风格——叶老主张写文章要像说话一样自然明白。

**记忆钩子：** 待人宽厚律己严，语文规范躬行君子。

### 卡3

**提问：** 《驿路梨花》的作者是谁？"梨花"在文中有什么含义？

**答案：** 彭荆风，当代作家。"梨花"有多层含义：①自然界的梨花（美丽的梨花林）；②叫梨花的哈尼族小姑娘（纯洁善良）；③雷锋精神（驿路梨花处处开，雷锋精神代代传）。"梨花"象征了美好的品德和人与人之间的互助关爱。

**易错分析：** "驿路"指古代传递政府文书等的道路。"驿路梨花"双关：既指驿路上的梨花，也指梨花姑娘和雷锋精神。"梨花"三层含义要写全。文章以"梨花"贯穿全文——是线索。

**记忆钩子：** 彭荆风驿路梨花，三重含义代代传。

### 卡4

**提问：** 《驿路梨花》在结构上有什么特点？

**答案：** ①倒叙——从"我"和老余在山中借宿写起，引出小屋主人是谁的悬念；②层层设疑、悬念迭起——小屋主人是谁？先后猜测是瑶族老人、梨花姑娘、梨花妹妹，最后才揭晓是解放军建造；③插叙——穿插了小茅屋的来历。结尾引用陆游诗句"驿路梨花处处开"点明主题。

**易错分析：** 结构顺序：发现小屋（设悬念）→打听主人（层层解疑）→最后揭晓谜底。"悬念"是本文最突出的写作手法。

**记忆钩子：** 倒叙设悬念层层解，驿路梨花处处开。

### 卡5

**提问：** 《最苦与最乐》的作者是谁？体裁是什么？文中认为什么最苦什么最乐？

**答案：** 梁启超，字卓如，号任公，别号饮冰室主人，近代思想家、政治家、学者，戊戌变法领袖之一。体裁是议论文。最苦：未尽责任（背负未了的责任是人生最苦的事）。最乐：尽了责任（完成责任是人生最快乐的事）。人生当负责任，不能逃避。

**易错分析：** 梁启超号"饮冰室主人"——"饮冰"出自《庄子》"朝受命而夕饮冰"，形容内心忧虑。"任公"是号，不是字（字卓如）。本文是议论文（不是记叙文）。

**记忆钩子：** 梁启超饮冰室主人，尽责任最乐不尽最苦。

### 卡6

**提问：** 《陋室铭》的作者是谁？体裁是什么？"铭"是什么意思？

**答案：** 刘禹锡，唐代诗人。"铭"是古代刻在器物上用来警诫自己或称述功德的文字，后来成为一种文体（铭文），一般用韵。陋室铭就是为自己的简陋居室写的铭文，表达安贫乐道、高洁傲岸的情操。

**易错分析：** 刘禹锡是唐代诗人（之前学过《秋词》《酬乐天扬州初逢席上见赠》）。"铭"是一种文体，不是"名字"。《陋室铭》托物言志——借陋室表达志向。

**记忆钩子：** 刘禹锡陋室铭，托物言志安贫乐道。

### 卡7

**提问：** "斯是陋室，惟吾德馨"（《陋室铭》）怎么理解？

**答案：** 这是居室虽简陋，但因为主人品德高尚就不觉得简陋了。"德馨"是品德高尚（馨：芳香/美好）。全句表达了一种安贫乐道的精神——物质条件不重要，精神世界的高尚才是根本。

**易错分析：** "馨"读 xīn，香气的意思。"惟吾德馨"——"惟"是因为/只因为。"苔痕上阶绿"的"上"是动词（长到）。

**记忆钩子：** 斯是陋室惟德馨，物质不足精神补。

### 卡8

**提问：** 《陋室铭》中"南阳诸葛庐，西蜀子云亭"用了什么写法？有什么作用？

**答案：** 用典/类比。南阳诸葛庐（诸葛亮隐居南阳的草庐）和西蜀子云亭（扬雄在成都的亭子）也都是简陋的居所，但主人都是历史上德才兼备的名人。刘禹锡以他们自比，表明自己虽然身居陋室但志向高远、不慕荣华。

**易错分析：** "诸葛庐"和"子云亭"是用典，不是写实。刘禹锡以诸葛亮和扬雄自况——"我虽然住陋室，但和这些名人一样有高尚的品德"。

**记忆钩子：** 诸葛庐子云亭，以古贤自比显高洁。

### 卡9

**提问：** 《爱莲说》的作者是谁？"说"是什么文体？本文用了什么写法？

**答案：** 周敦颐，北宋哲学家，宋明理学的开创者。"说"是一种议论文体，可以说明事物、发表议论。本文借莲表达君子品格——"托物言志"（借莲花写君子）。以"菊"（隐逸者）和"牡丹"（富贵者）对比"莲"（君子者），突出莲的高洁。

**易错分析：** "说"读 shuō，是一种文体（如《马说》《师说》）。"周敦颐"的"敦颐"注意写法。"蕃"读 fán（多）。"濯清涟而不妖"——濯（zhuó）是洗涤。

**记忆钩子：** 周敦颐爱莲说，托物言志赞君子。

### 卡10

**提问：** "出淤泥而不染，濯清涟而不妖"（《爱莲说》）象征了君子什么品格？

**答案：** "出淤泥而不染"——象征君子生长在污浊的环境中却不受沾染（洁身自好）。"濯清涟而不妖"——象征君子被清水洗涤却不显妖媚（质朴庄重）。两句合起来刻画了君子高洁、正直、不媚俗的品格——这正是周敦颐心目中理想人格的写照。

**易错分析：** "淤泥"象征污浊的社会环境。"不染"是不被污染。"不妖"是不妖艳/不媚俗。全文核心——莲=君子。

**记忆钩子：** 出淤泥不染是洁，濯清涟不妖是正。

### 卡11

**提问：** "中通外直，不蔓不枝"（《爱莲说》）象征了什么？

**答案：** "中通"——莲茎中间贯通，象征君子通达正直。"外直"——外部挺直，象征君子行为方正。"不蔓不枝"——没有藤蔓和分枝，象征君子不攀附权贵、不结党营私。这是对君子行为品格的概括。

**易错分析：** "不蔓不枝"——"蔓"是名词用作动词（生藤蔓），"枝"也是名词用作动词（长分枝）——文言活用。

**记忆钩子：** 中通外直人通达，不蔓不枝不攀附。

### 卡12

**提问：** 第四单元写作主题是什么？怎样选材？

**答案：** 写作主题是"怎样选材"。要点：①围绕中心选材（跟中心无关的材料再好也要舍弃）；②选材要真实、新颖（避免老套素材）；③选材要典型（最能表现中心的一两个事例胜过泛泛的十个）；④注意详略安排（重点材料详写，次要材料略写）。

**易错分析：** "选材"和"突出中心"（七上）有关联但不相同——前者重"选什么"，后者重"怎么突出"。

**记忆钩子：** 围绕中心选真实，典型新颖详略当。
