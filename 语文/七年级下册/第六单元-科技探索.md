## Anki 记忆卡

### 卡1

**提问：** 《太空一日》的作者是谁？体裁是什么？

**答案：** 杨利伟，中国首位进入太空的航天员（神舟五号）。体裁是自传性纪实文章/回忆录。文章记述了作者在太空飞行一日中的经历、感受和见闻。

**易错分析：** 杨利伟是航天员不是作家。这篇文章选自他的自传《天地九重》。2003年10月15日神舟五号发射——这是中国首次载人航天飞行。

**记忆钩子：** 杨利伟太空一日，神五飞天第一人。

### 卡2

**提问：** 《太空一日》中杨利伟在太空中遇到了哪些意外情况？

**答案：** ①火箭上升时产生共振——身体极度难受，以为自己要牺牲了；②在太空中出现"本末倒置"的错觉——感觉天在下、地在上；③不时听到神秘"敲击声"——原因不明；④返回时舷窗出现裂纹——非常危险；⑤落地时巨大的撞击力——嘴巴被磕破。

**易错分析：** 最危险的"共振"发生在火箭上升阶段——杨利伟说"我以为我要牺牲了"。舷窗裂纹不是真裂，是隔热层烧裂。杨利伟用"归途如此惊心动魄"来形容返回过程。

**记忆钩子：** 共振错觉敲击声，舷窗裂纹返回险。

### 卡3

**提问：** 《"蛟龙"探海》的作者是谁？"蛟龙"指什么？（2024新增课文）

**答案：** 许晨，当代作家、记者。"蛟龙"指"蛟龙号"载人深潜器。文章记述了中国"蛟龙号"深潜器挑战深海、探索海底世界的故事，展现了中国载人深潜事业的伟大成就和科学家们的奋斗精神。

**易错分析：** 这是2024版新课文。与《太空一日》形成"上天入海"的对仗——太空对应深海。"蛟龙"既是中国神话中的龙，也是深潜器的名字（双关）。

**记忆钩子：** 许晨记蛟龙号，深海探索中国骄傲。

### 卡4

**提问：** 《带上她的眼睛》的作者是谁？体裁是什么？

**答案：** 刘慈欣，当代科幻作家，代表作《三体》（获雨果奖）。体裁是科幻小说。本文写一位地航员小姑娘被困在地心深处，"我"带上她的"眼睛"（传感眼镜）替她感受地面的美好，最后才知道她再也回不来的故事。

**易错分析：** 刘慈欣是"科幻作家"不是"科普作家"。"眼睛"不是真的眼睛，是"传感眼镜"——让使用者能远程感知视觉和触觉。"落日六号"是地航飞船——钻入地心。

**记忆钩子：** 刘慈欣三体作者，带上她的眼睛到地心。

### 卡5

**提问：** 《带上她的眼睛》中"她"是谁？"我"带上她的眼睛去了哪里？

**答案：** "她"是"落日六号"地航飞船的领航员，因事故永远被困在地心。她的"眼睛"（传感眼镜）被带出来，"我"戴着它去草原感受自然——看花、闻草、感受阳光。全文通过她的"眼睛"让我们重新审视身边的美好事物——我们习以为常的东西，对她来说是永远失去的天堂。

**易错分析：** 小说主题：①珍惜身边的美好；②对未知世界保持敬畏；③科技的双刃剑。"落日六号"沉到了地心再也回不来——她是孤独的、永远被困的。

**记忆钩子：** 她困地心我戴眼镜，平常景物是她的天堂。

### 卡6

**提问：** 《活板》的作者是谁？体裁是什么？"活板"指什么？

**答案：** 沈括，北宋科学家、政治家。选自《梦溪笔谈》（笔记体科学著作）。"活板"指活字印刷术（用胶泥刻字、烧硬后排版印刷）。本文详细记录了毕昇发明的活字印刷术的制作流程和特点。

**易错分析：** 沈括是北宋人，《梦溪笔谈》被誉为"中国科学史的里程碑"。活字印刷术的发明者是"毕昇"（北宋平民），不是沈括 ——沈括是记录者。"板"同"版"。"活"体现在"字是活的、排版是活的、印刷是活的"。

**记忆钩子：** 沈括梦溪笔谈，毕昇活字印刷术。

### 卡7

**提问：** 《活板》中活字印刷的工艺流程是怎样的？

**答案：** ①准备：用胶泥刻字→火烧令坚（制字）；②排版：设铁板→铺松脂蜡等→放铁范→排字→火上烤→压平；③印刷：一板印刷→另一板排版（交替使用，极为高效）；④拆版：用完后火烤铁板，字模自动脱落。核心：每字一印、随时排版、两板交替。

**易错分析：** 工艺顺序要记清。"印"有几个意思——①印刷（动词）；②字模（名词）；"止印三二本"的"止"同"只"。"极为神速"形容两板交替印刷的效率。

**记忆钩子：** 刻字→排版→印刷→拆版，两板交替效率高。

### 卡8

**提问：** 《活板》重点字词：板、印、范、就、燔、讫、拂、奇、殊。

**答案：**
- 板：同"版"（板印书籍）。
- 印：字模（每字为一印）。
- 范：框子（铁范）。
- 就：完成（更互用之，瞬息可就）。
- 燔（fán）：烧（不若燔土）。
- 讫（qì）：完毕（用讫再火令药镕）。
- 拂：擦（以手拂之）。
- 奇（jī）：零数/余数（长约八分有奇）。
- 殊：很/极（殊不沾污）。

**易错分析：** "有奇"的"奇"读 jī（零数），不是 qí。"持就火炀之"的"炀"读 yáng（烤）。"更互"是交替、互相。"字平如砥"——"砥"是磨刀石。

**记忆钩子：** 板同版印字模，燔烧讫完毕拂擦。

### 卡9

**提问：** 《海底两万里》的作者是谁？体裁是什么？

**答案：** 儒勒·凡尔纳，法国作家，被誉为"现代科幻小说之父"。体裁是科幻小说。主要讲述博物学家阿龙纳斯教授等人乘坐"诺第留斯号"（鹦鹉螺号）潜水艇在海底航行探险的故事。

**易错分析：** 凡尔纳是"法国"人，不是英国人。代表作"三部曲"：《格兰特船长的儿女》《海底两万里》《神秘岛》。"诺第留斯号"（Nautilus）船长是尼摩船长。

**记忆钩子：** 凡尔纳海底两万里，诺第留斯号尼摩船长。

### 卡10

**提问：** 《海底两万里》中尼摩船长是一个怎样的人？

**答案：** 尼摩船长是《海底两万里》的核心人物，他：①知识渊博——精通海洋学、工程学，设计和建造了"诺第留斯号"；②英勇果敢——在海底遇到过许多危险（章鱼袭击、冰山封路等）都能沉着应对；③富有正义感——帮助被压迫的民族；④神秘孤独——远离人类社会，内心有着复杂的故事和深沉的仇恨。

**易错分析：** 尼摩船长不是纯粹的好人或坏人——是一个复杂的人物。他的名字"尼摩"在拉丁语中意为"没有人"。他既是科学家、反抗者，也是复仇者。

**记忆钩子：** 尼摩船长知识渊博，孤独正义复仇者。

### 卡11

**提问：** "烟笼寒水月笼沙，夜泊秦淮近酒家"（《泊秦淮》）赏析。

**答案：** 杜牧，唐代诗人。"烟笼寒水月笼沙"——互文修辞，烟雾和月光笼罩着寒冷的江水和岸边的沙滩，渲染了朦胧冷寂的秦淮夜景。"夜泊秦淮近酒家"——夜晚停泊在秦淮河畔，靠近热闹的酒家。后两句"商女不知亡国恨，隔江犹唱后庭花"借商女唱《玉树后庭花》讽刺晚唐统治者的荒淫享乐、不顾国事。

**易错分析：** 两句"笼"是互文——不是烟笼水、月笼沙，而是烟和月共同笼罩了水和沙。《后庭花》即《玉树后庭花》——南朝陈后主所作、被认为是"亡国之音"。

**记忆钩子：** 杜牧秦淮夜泊，商女唱后庭花讽亡国。

### 卡12

**提问：** "可怜夜半虚前席，不问苍生问鬼神"（《贾生》）讽刺了什么？

**答案：** 李商隐，唐代诗人。汉文帝召见贾谊（贾生），并非询问治国安民之策，而是询问鬼神之事。"不问苍生问鬼神"——尖锐讽刺了统治者不关心百姓疾苦、不重视人才，只知道求仙问道的荒唐行为。李商隐借古讽今，讽刺晚唐皇帝的昏庸。

**易错分析：** 贾生是贾谊（西汉政论家），不是"贾岛"或"贾宝玉"。"虚前席"——"虚"是徒然，"前席"是古人坐席上听得入神时向前移动。这里的讽刺是"表面求贤实则不重人才"。

**记忆钩子：** 李商隐借贾谊讽今，问鬼不问苍生。

### 卡13

**提问：** "政入万山围子里，一山放出一山拦"（《过松源晨炊漆公店·其五》）的哲理？

**答案：** 杨万里，南宋诗人。行走在万山之中，翻过一座山，又被一座山拦住去路。哲理：人生就像翻山越岭——克服了一个困难，又会遇到新的困难。不要因为一时的成功而松懈，要有长期奋斗的准备。体现了诗人对人生困难连续性的深刻认识。

**易错分析：** "政"同"正"。"松源"是地名（今江西）。杨万里的诗风通俗自然——"诚斋体"。这句话常用来比喻困难接连不断、不能掉以轻心。

**记忆钩子：** 杨万里过松源，一山放出一山拦——困难连环。

### 卡14

**提问：** "有约不来过夜半，闲敲棋子落灯花"（《约客》）赏析。

**答案：** 赵师秀，南宋诗人（"永嘉四灵"之一）。前两句"黄梅时节家家雨，青草池塘处处蛙"写黄梅雨季的景色——雨声、蛙声烘托出夏夜的宁静。后两句写约好的客人过了半夜还没来，诗人无聊地敲着棋子、看灯花落下——一个"闲敲棋子"的动作，生动传达出等待中的寂寞、无聊和些许焦躁。

**易错分析：** "灯花"是灯芯燃烧后结成的花状物——古人以灯花为吉兆。"落灯花"说明等了很久。"有约不来"不是抱怨，而是一种文人闲适中的淡淡失落。

**记忆钩子：** 赵师秀约客不来，闲敲棋子灯花落。

### 卡15

**提问：** 第六单元是"活动·探究"单元，三个任务分别是什么？

**答案：** 任务一：阅读与理解——阅读《太空一日》《"蛟龙"探海》《带上她的眼睛》《活板》，把握科技探索类文章的阅读方法。任务二：科学与探索讨论——围绕"科技发展与人类未来"等话题进行讨论交流。任务三：写作实践——写一篇科技类文章（科幻故事、科普说明文或科技观察日记）。

**易错分析：** "活动·探究"单元的特点是"阅读→实践→写作"三位一体，不是割裂的。2024版本单元新增《"蛟龙"探海》，替换了原来的《伟大的悲剧》。

**记忆钩子：** 阅读实践加写作，三位一体探科技。
