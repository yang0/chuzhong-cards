# 第五单元 哲理之思

## 基本信息

- 科目：语文
- 年级：七年级下册
- 教材：统编版 / 人教版《语文》（2024 修订版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"哲理之思"为主题，通过自然景物和生活中的小事阐发人生哲理。学习重点是托物言志、借景抒情的写法，体会诗文中蕴含的哲理。

- 18 紫藤萝瀑布 / 宗璞
- 19\* 一棵小桃树 / 贾平凹
- 20\* 外国诗二首：假如生活欺骗了你 / 普希金、未选择的路 / 弗罗斯特
- 21 古代诗歌五首：登幽州台歌 / 陈子昂、望岳 / 杜甫、登飞来峰 / 王安石、游山西村 / 陆游、己亥杂诗（其五）/ 龚自珍

## Anki 记忆卡

### 卡1

**提问：** 《紫藤萝瀑布》的作者是谁？体裁是什么？"紫藤萝"是什么？

**答案：** 宗璞，当代女作家，原名冯钟璞，哲学家冯友兰之女。代表作《红豆》《弦上的梦》。体裁是散文。紫藤萝是一种藤本植物，开紫色花串。本文写作者见到一树盛开的紫藤萝花，由花悟理。

**易错分析：** 宗璞是冯友兰之女——常考题。"璞"读 pú。体裁是"散文"不是"小说"。"瀑布"是比喻——紫藤萝花像瀑布一样倾泻而下。

**记忆钩子：** 宗璞见紫藤萝，花悟人生哲理。

### 卡2

**提问：** 《紫藤萝瀑布》中作者从紫藤萝花感悟到了什么？

**答案：** ①花和人都会遇到各种各样的不幸（作者的弟弟身患绝症、紫藤萝曾遭摧残）；②但生命的长河是无止境的（紫藤萝又盛开了，繁花似锦）；③我们应该以积极乐观的态度面对生活中的挫折，珍惜生命、热爱生活。

**易错分析：** 主题核心是"生命的长河是无止境的"——个人不幸在浩瀚的生命长河中只是短暂的波折，生命本身生生不息。文中插入了一段关于"十年前紫藤萝被摧残"的回忆，与眼前繁花形成对比。

**记忆钩子：** 花谢花开人悲人喜，生命长河永不停息。

### 卡3

**提问：** 《紫藤萝瀑布》中"每一朵盛开的花就像是一个小小的张满了的帆"运用了什么修辞？

**答案：** 比喻。将紫藤萝花比作张满的帆，生动形象地写出了花朵饱满、充满生命力的形态，也暗示了生命之舟扬帆远航的寓意。"帆下带着尖底的舱，船舱鼓鼓的"——又把花比作船。这是连续比喻。

**易错分析：** 比喻+拟人（花像帆在"流动"）。全文的核心修辞是比喻——紫藤萝像瀑布、像帆、像船。不能只答"比喻"还要分析效果。

**记忆钩子：** 花如帆船满生机，生命流动不停息。

### 卡4

**提问：** 《一棵小桃树》的作者是谁？体裁是什么？

**答案：** 贾平凹，当代作家，代表作《废都》《秦腔》。体裁是散文。文章通过描写一棵小桃树的成长经历，托物言志，表达了自己在逆境中顽强奋斗的决心。

**易错分析：** 贾平凹的"凹"读 wā，是陕西方言音，不是 āo。他是陕西作家。"一棵小桃树"是"托物言志"——小桃树就是"我"，它的经历就是"我"的经历。

**记忆钩子：** 贾平凹wā一棵小桃树，托物言志不言弃。

### 卡5

**提问：** 《一棵小桃树》中小桃树的成长经历了哪些磨难？

**答案：** ①被猪拱过——差点被毁；②被人嫌弃——长在角落不被重视；③干旱——缺水但顽强生长；④风雨——被风吹雨打，花落满地。但即使如此它仍然在春天绽放，虽然花不多但代表了顽强的生命力。

**易错分析：** 小桃树的经历对应了作者的人生经历——作者也曾在人生道路上遭遇挫折但顽强拼搏。"蓄着"的"蓄"读 xù。文中的"我的小桃树"寄托了对奶奶的怀念。

**记忆钩子：** 小桃树屡遭磨难，顽强开花不言弃。

### 卡6

**提问：** 《假如生活欺骗了你》的作者是谁？哪国人？

**答案：** 普希金，俄国诗人，被誉为"俄国文学之父""俄国诗歌的太阳"。代表作《叶甫盖尼·奥涅金》《上尉的女儿》。这首诗写于诗人被流放期间，以劝说的口吻告诉人们要乐观面对挫折。

**易错分析：** 普希金是"俄国"诗人，不是"苏联"（他在1837年去世，苏联1917年才成立）。这首诗是"说理诗"——用平等对话的口吻，不是"哲理诗"（哲理诗更抽象）。

**记忆钩子：** 普希金俄国诗歌太阳，假如生活欺骗你别悲伤。

### 卡7

**提问：** "心儿永远向往着未来，现在却常是忧郁。一切都是瞬息，一切都将会过去"（《假如生活欺骗了你》）怎么理解？

**答案：** 诗人告诉我们要用发展的眼光看待生活——现在的痛苦是暂时的，未来是美好的。不要被眼前的困难打倒，要相信快乐的日子终会来临。这是普希金在流放中写下的诗，充满了乐观主义精神。

**易错分析：** "瞬息"极言时间短暂。"亲切的怀念"指过去的痛苦也会成为将来美好的回忆——逆境也是人生的宝贵经历。

**记忆钩子：** 一切都是瞬息，一切都将过去，快乐会来临。

### 卡8

**提问：** 《未选择的路》的作者是谁？"路"象征什么？

**答案：** 弗罗斯特，美国诗人。"路"象征人生道路/人生选择。诗人走在树林中遇到两条路分岔，他选择了更少人走的那条，从此决定了他一生的道路。表达了人生选择的重要性和选择的遗憾——选择了一条路就无法再走另一条。

**易错分析：** 弗罗斯特是"美国"诗人。这首诗不是真的写"路"，而是"人生选择"——象征。"一片树林里分出两条路——而我选择了人迹更少的一条，从此决定了我一生的道路"——这是全诗的核心。

**记忆钩子：** 弗罗斯特未选之路，人生岔路选择定命运。

### 卡9

**提问：** 登幽州台歌的作者是谁？"幽州台"是什么？

**答案：** 陈子昂，唐代诗人。幽州台（又称燕台、黄金台）是战国时期燕昭王为招纳贤士所筑。陈子昂登台远眺，感到天地悠悠、人生短暂、知音难觅——"前不见古人，后不见来者"，抒发了一种怀才不遇、孤独悲愤的情感。

**易错分析：** 幽州台和"黄金台"是同一个典故——燕昭王筑台招贤。陈子昂此时受到排挤，所以登台感慨。"怆然而涕下"的"怆"读 chuàng。

**记忆钩子：** 陈子昂独登幽州，念天地之悠悠独怆然。

### 卡10

**提问：** "会当凌绝顶，一览众山小"（《望岳》）表达了什么？

**答案：** 杜甫，唐代诗人。"会当"是一定要的意思。"凌绝顶"是登上最高峰。"一览众山小"是俯瞰群山、一切尽在眼底。这两句表达了诗人不怕困难、勇攀高峰的雄心壮志和积极进取的豪迈气概。全诗写的是东岳泰山。

**易错分析：** 杜甫写《望岳》时24岁，还没中进士，正值青年意气风发之时。这不是晚年杜甫的作品。"造化钟神秀"——"钟"是聚集。"荡胸生曾云"——"曾"同"层"。

**记忆钩子：** 杜甫年轻登泰山，会当凌顶众山小。

### 卡11

**提问：** "不畏浮云遮望眼，自缘身在最高层"（《登飞来峰》）的哲理是什么？

**答案：** 王安石，北宋政治家、文学家。不怕浮云遮住远望的视线，只因为我站在最高的地方。哲理：①站得高看得远——认识达到一定高度就不被假象迷惑；②掌握了正确的观点和方法，就能透过现象看本质。

**易错分析：** "浮云"比喻奸佞小人或困难阻碍。与王之涣"欲穷千里目，更上一层楼"和杜甫"会当凌绝顶，一览众山小"有相似之处，但王安石的更强调政治家的格局和胆识。

**记忆钩子：** 王安石登飞来峰，不畏浮云站得高。

### 卡12

**提问：** "山重水复疑无路，柳暗花明又一村"（《游山西村》）的哲理是什么？

**答案：** 陆游，南宋爱国诗人。原意写游山西村时的自然风光——山峦重叠、水流曲折，正担心无路可走时，忽然柳绿花艳、又出现一个村庄。哲理：在困境中不要放弃，坚持下去就会出现转机和希望。比喻困境中蕴含希望。

**易错分析：** 陆游是"南宋"诗人——不是北宋。"莫笑农家腊酒浑"——"浑"是浑浊（古代农家自酿酒）。"萧鼓追随春社近"——"春社"是古代立春后祭祀土地神的活动。

**记忆钩子：** 陆游山重水复疑无路，柳暗花明又一村——绝处逢生。

### 卡13

**提问：** "落红不是无情物，化作春泥更护花"（《己亥杂诗·其五》）的哲理是什么？

**答案：** 龚自珍，清代思想家、诗人。"落红"是落花。落花不是无情之物，它变成春泥滋养新花。哲理：①奉献精神——即使离开原来的位置，也要用自己的一切来培育新生力量；②新老交替——老一代培养新一代；③龚自珍辞官南归，但仍心系国家，愿意为国育才。

**易错分析：** "落红"不是"落红（红色）"，而是"落花"——诗人自喻。"更护花"的"花"比喻新事物/新生命。"己亥"是干支纪年——1839年。龚自珍是"清代"不是"近代"。

**记忆钩子：** 龚自珍落红护花，辞官归乡心系国。
