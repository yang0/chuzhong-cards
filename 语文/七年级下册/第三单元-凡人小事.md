# 第三单元 凡人小事

## 基本信息

- 科目：语文
- 年级：七年级下册
- 教材：统编版 / 人教版《语文》（2024 修订版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"凡人小事"为主题，关注普通人的故事，体会小人物身上的人性光辉。学习重点是熟读精思、通过细节描写把握人物特点。

- 10 阿长与《山海经》/ 鲁迅
- 11 老王 / 杨绛
- 12\* 台阶 / 李森祥
- 13 卖油翁 / 欧阳修
- 名著导读：《骆驼祥子》
- 课外古诗词诵读：竹里馆 / 王维、春夜洛城闻笛 / 李白、逢入京使 / 岑参、晚春 / 韩愈

## Anki 记忆卡

### 卡1

**配图：** ![](第三单元-凡人小事/01.webp)

**提问：** 《阿长与〈山海经〉》的作者是谁？出自哪部集子？阿长是谁？

**答案：** 鲁迅，选自散文集《朝花夕拾》。阿长（长妈妈）是鲁迅童年的保姆，一个粗俗迷信但善良淳朴的劳动妇女。

**易错分析：** 阿长是保姆，不是亲戚。这篇文章是《朝花夕拾》中的一篇——这部集子在七上第三单元也学过。

**记忆钩子：** 鲁迅忆保慈善良，阿长买山海经。

### 卡2

**配图：** ![](第三单元-凡人小事/02.webp)

**提问：** 《阿长与〈山海经〉》中鲁迅对阿长的情感是怎样变化的？

**答案：** 憎恶（踩死隐鼠、切切察察）→敬意（讲长毛故事）→憎恶（教繁琐规矩）→空前的敬意（买《山海经》）→怀念和祝福（文章结尾）。先抑后扬，情感是层层递进的。

**易错分析：** 情感变化中"敬意"有两次——第一次是讲长毛故事（觉得她有"伟大的神力"），第二次是买《山海经》（"别人不肯做/不能做的事她居然做到了"）。第二次敬意是真正的"空前的敬意"。

**记忆钩子：** 先抑后扬五层情，最敬山海经买到。

### 卡3

**配图：** ![](第三单元-凡人小事/03.webp)

**提问：** "哥儿，有画儿的'三哼经'，我给你买来了！"（《阿长与〈山海经〉》）这句话好在哪里？

**答案：** ①语言描写——"三哼经"说明她不识字，把《山海经》说成"三哼经"，表现她的粗朴可爱；②买书这件事别人不肯做、不能做，她居然做到了，表现她对鲁迅的爱是发自内心的、无私的；③这是全文情感转折的触发点——从此鲁迅对阿长产生了空前的敬意。

**易错分析：** "三哼经"说明阿长不识字但记住了书名的大致发音，非常传神——她用心记住了鲁迅想要的礼物。

**记忆钩子：** 三哼经可爱又暖心，不识字也记挂你。

### 卡4

**配图：** ![](第三单元-凡人小事/04.webp)

**提问：** 《老王》的作者是谁？体裁是什么？

**答案：** 杨绛，现代作家、翻译家、学者（钱锺书夫人）。代表作《干校六记》《我们仨》，译作《堂吉诃德》。体裁是散文。文章回忆了与人力车夫老王交往的几件事。

**易错分析：** 杨绛（jiàng），读绛不是"降"。"杨绛"是作家≠"杨降"。她是女作家。

**记忆钩子：** 杨绛写老王——幸运者对不幸者的愧怍。

### 卡5

**配图：** ![](第三单元-凡人小事/05.webp)

**提问：** 《老王》中作者与老王之间发生了哪些事？表现了什么？

**答案：** ①老王送冰（车费减半、冰大一倍）——老王的善良朴实；②老王送钱先生去医院不肯收钱——老王的知恩图报；③老王临终前送香油和鸡蛋——老王的赤诚感恩。最后老王去世，作者感到"愧怍"——因为自己没能更好地回报老王。

**易错分析：** 结尾"那是一个幸运者对不幸者的愧怍"是全文主旨——杨绛觉得自己是"幸运者"，老王是"不幸者"，自己给予老王的关心不够，表达了知识分子的自省精神。

**记忆钩子：** 送冰送医送香油，愧怍因为回报少。

### 卡6

**配图：** ![](第三单元-凡人小事/06.webp)

**提问：** 《老王》中老王"直着脚往里走"、"面如死灰"、"僵尸"般的描写有什么作用？

**答案：** 外貌和动作描写。写老王临终前的样子——身体虚弱到极点，但仍然坚持来送香油和鸡蛋。这些描写让读者感受到老王在生命最后一刻还在用最朴实的方式表达对杨绛一家的感激，令人心酸又感动。

**易错分析：** 这段外貌描写极其细致（面如死灰、眼上结了一层翳等），考试常问老王"那只好眼"——他是盲人（一只眼是瞎的），生活艰难。

**记忆钩子：** 僵尸般送鸡蛋，赤诚之心到最后一刻。

### 卡7

**配图：** ![](第三单元-凡人小事/07.webp)

**提问：** 《台阶》的作者是谁？"台阶"在文中有什么象征意义？

**答案：** 李森祥，当代作家。"台阶"象征：①地位和尊严（台阶高代表屋主人的地位高）；②父亲的理想和追求（父亲一生的目标就是造高台阶的新屋）；③中国农民的坚韧和勤劳（父亲用一生来实现这个朴素理想）。

**易错分析：** 本文是"小说"不是散文——李森祥是小说家。"台阶"的象征意义要答到"地位/尊严"这个层面。父亲的目标不是"住新屋"而是"造高台阶"。

**记忆钩子：** 台阶高地位高，父亲一生为一台。

### 卡8

**配图：** ![](第三单元-凡人小事/08.webp)

**提问：** 父亲为什么"总觉得屋前的台阶低"？（《台阶》）

**答案：** 因为老家有"台阶高地位高"的传统观念，台阶高低代表屋主人的社会地位。父亲觉得自己家的三级台阶太低，被人看不起，所以下定决心要造一栋有高台阶的新屋——这体现了父亲强烈的自尊心和对受人尊重的渴望。

**易错分析：** 父亲不是虚荣，而是中国农民朴素的自尊——他想要体面、受人尊敬。

**记忆钩子：** 台阶低被人瞧不起，父亲的夙愿是造高台。

### 卡9

**配图：** ![](第三单元-凡人小事/09.webp)

**提问：** 《卖油翁》的作者是谁？朝代？本文说明了什么道理？

**答案：** 欧阳修，字永叔，号醉翁，晚号六一居士，北宋政治家、文学家，"唐宋八大家"之一。本文选自《归田录》。说明熟能生巧的道理——"无他，但手熟尔"。也告诉我们即使有高超的本领也不要骄傲自满。

**易错分析：** 欧阳修是北宋人（不是唐或南宋），谥号"文忠"。"但微颔之"的"颔"读 hàn（点头）。"忿然"的"忿"读 fèn。本文文言文篇幅短小但寓意深刻。

**记忆钩子：** 欧阳修卖油翁，唯手熟尔不骄傲。

### 卡10

**配图：** ![](第三单元-凡人小事/10.webp)

**提问：** 《卖油翁》重点字词：矜、睨、颔、忿、尔、覆、徐、杓、遣。

**答案：**
- 矜（jīn）：自夸（公亦以此自矜）。
- 睨（nì）：斜眼看，不在意的样子。
- 颔（hàn）：点头（但微颔之）。
- 忿（fèn）：气愤（忿然）。
- 尔：同"耳"，罢了（但手熟尔）。
- 覆：盖（以钱覆其口）。
- 徐：慢慢地（徐以杓酌油沥之）。
- 杓（sháo）：同"勺"。
- 遣：打发（康肃笑而遣之）。

**易错分析：** "但微颔之"——"但"是"只/只是"（古义），不是"但是"。陈尧咨（zī）——不要读错。"自矜"是"自我夸耀"。

**记忆钩子：** 矜自夸睨斜看，但只是耳罢了。

### 卡11

**配图：** ![](第三单元-凡人小事/11.webp)

**提问：** 《骆驼祥子》的作者是谁？体裁是什么？祥子是谁？

**答案：** 老舍（舒庆春），现代作家，代表作《骆驼祥子》《四世同堂》《茶馆》。《骆驼祥子》是长篇小说，讲述北平人力车夫祥子三起三落的悲惨命运。

**易错分析：** 祥子来自农村，目标就是买一辆属于自己的车。他经历了"三起三落"——买车被抢→攒钱被敲诈→再买车因丧事卖掉，最终堕落。

**记忆钩子：** 老舍骆驼祥子，三起三落人力车夫。

### 卡12

**配图：** ![](第三单元-凡人小事/12.webp)

**提问：** 祥子的性格前后有什么变化？

**答案：** 前期：勤劳、淳朴、善良、有梦想、有上进心（像骆驼一样坚韧）。后期：堕落、麻木、自私、懒惰、行尸走肉（吃喝嫖赌、出卖朋友）。造成变化的原因是黑暗的社会现实——一个正直的人被旧社会逼成了"鬼"。

**易错分析：** 祥子的堕落不是他自己的错，是"黑暗的旧社会把人变成鬼"——这是小说的核心主题。

**记忆钩子：** 前期勤劳上进，后期堕落麻木，社会逼良为娼。

### 卡13

**配图：** ![](第三单元-凡人小事/13.webp)

**提问：** "深林人不知，明月来相照"（《竹里馆》）的作者和意境？

**答案：** 王维，唐代诗人。前两句"独坐幽篁里，弹琴复长啸"写独自在竹林中弹琴长啸。后两句写竹林深处无人知晓，只有明月相伴。表现了诗人宁静、淡泊、悠然自得的心境，诗中有画（王维"诗中有画"的特点）。

**易错分析：** "竹里馆"是辋川别墅中的一处景点，不是图书馆。"幽篁"指幽深的竹林。"长啸"是撮口发出长而清越的声音——不是"嘲笑"。"诗佛"王维擅长写山水田园诗。

**记忆钩子：** 王维独坐竹里馆，明月相伴自悠然。

### 卡14

**配图：** ![](第三单元-凡人小事/14.webp)

**提问：** "此夜曲中闻折柳，何人不起故园情"（《春夜洛城闻笛》）表达什么？

**答案：** 李白，唐代诗人。前两句"谁家玉笛暗飞声，散入春风满洛城"写春夜笛声随风飘满全城。后两句写听到《折杨柳》的曲子，谁能不思念自己的家乡呢？"折柳"是送别之曲，表达的是思乡之情。"满洛城"用夸张手法写笛声传播之广。

**易错分析：** "折柳"是乐府古曲《折杨柳》，不是真的折断柳枝。"满洛城"是夸张——洛阳城再大也不能全城听到笛声，这是诗人的艺术渲染。

**记忆钩子：** 李白闻折柳思故乡，夸张满洛城。

### 卡15

**配图：** ![](第三单元-凡人小事/15.webp)

**提问：** "马上相逢无纸笔，凭君传语报平安"（《逢入京使》）表达了什么？

**答案：** 岑参，唐代边塞诗人。前两句"故园东望路漫漫，双袖龙钟泪不干"写离开长安去边塞，回望故乡路途遥远、泪湿双袖。后两句写路上遇入京使者，想写信却没纸笔，只能托人带个口信——"我平安"——报平安。表达了对家乡和亲人的深切思念，以及边塞征人的豪迈与无奈。

**易错分析：** "龙钟"是沾湿淋漓的样子，不是老态龙钟。"传语"是口头带话，不是写信。

**记忆钩子：** 岑参逢使报平安，边塞征人思故乡。

### 卡16

**配图：** ![](第三单元-凡人小事/16.webp)

**提问：** "杨花榆荚无才思，惟解漫天作雪飞"（《晚春》）表达了什么？

**答案：** 韩愈，唐代诗人。前两句"草树知春不久归，百般红紫斗芳菲"写花草树木知道春天将去，纷纷争奇斗艳。后两句写杨花和榆荚没有鲜艳的颜色，只能像雪花一样漫天飞舞——它们也在用自己的方式留住春天。表达了要珍惜时光、抓住时机有所作为的寓意。

**易错分析：** "无才思"不是贬低杨花榆荚，而是说它们不像其他花那样艳丽。韩愈的本意是——不管有没有"才思"，都在努力展现自己的价值，人更应如此。

**记忆钩子：** 韩愈晚春杨花雪，珍惜时光各展才。
