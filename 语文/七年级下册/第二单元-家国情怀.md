# 第二单元 家国情怀

## 基本信息

- 科目：语文
- 年级：七年级下册
- 教材：统编版 / 人教版《语文》（2024 修订版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"家国情怀"为主题，选编了诗歌、散文和民歌，表现了不同历史时期中华儿女对祖国的热爱。学习重点是把握抒情方式、体会作品情境、学习做批注。

## Anki 记忆卡

### 卡1

**配图：** ![](第二单元-家国情怀/01.webp)

**提问：** 《黄河颂》的作者是谁？出自哪部作品？体裁是什么？

**答案：** 光未然（原名张光年），诗人、文学评论家。本文是《黄河大合唱》第二乐章的歌词。《黄河大合唱》由冼星海作曲，是一部大型合唱音乐作品。

**易错分析：** 光未然是"词作者"，冼星海是"曲作者"——两人合作。《黄河颂》是"歌词"不是"现代诗"（虽然形式上像诗）。"哺育"的"哺"读 bǔ，不是 pǔ。

**记忆钩子：** 光未然词冼星海曲，黄河大合唱颂民族魂。

### 卡2

**配图：** ![](第二单元-家国情怀/02.webp)

**提问：** 《黄河颂》中诗人从哪些方面赞美黄河？

**答案：** ①自然特点——黄河的气势磅礴（"惊涛澎湃""九曲连环"）；②地理特征——黄河是中华民族的摇篮（"五千年的古国文化，从你这发源"）；③精神象征——黄河代表我们民族的精神（"伟大坚强"）。诗人借黄河歌颂了中华民族的伟大坚强。

**易错分析：** 本文运用了"象征"手法——黄河象征中华民族。"望黄河滚滚"的"望"字统领全诗。歌词的特点是节奏鲜明、韵律铿锵。

**记忆钩子：** 一赞气势二赞摇篮三赞精神，黄河即民族。

### 卡3

**配图：** ![](第二单元-家国情怀/03.webp)

**提问：** 《老山界》的作者是谁？体裁是什么？老山界是什么地方？

**答案：** 陆定一，无产阶级革命家。体裁是回忆性散文。老山界是红军长征途中翻越的第一座难走的山（位于广西和湖南交界），课文记录了红军翻越老山界的经过。

**易错分析：** 陆定一是革命家不是作家。老山界是长征中的"第一座难走的大山"——红军不怕艰难险阻翻越过去。"横着心"不是"狠心"，而是"下定决心"。

**记忆钩子：** 陆定一记老山界，长征第一道难关。

### 卡4

**配图：** ![](第二单元-家国情怀/04.webp)

**提问：** 《老山界》中红军翻越老山界遇到了哪些困难？表现了红军什么精神？

**答案：** 困难：①山高路险（悬崖峭壁、路窄）；②天寒地冻（山上寒冷）；③粮食不足（饥饿）；④敌情紧张（有敌人追击）。表现了红军不畏艰险、乐观向上、团结互助的革命英雄主义和革命乐观主义精神。

**易错分析：** 困难要答四个方面，不只"山高路险"。"之"字路上的火把像"星星"——这段夜景描写非常出名，既写实又充满诗意。

**记忆钩子：** 山险天寒粮不足，红军乐观不怕苦。

### 卡5

**配图：** ![](第二单元-家国情怀/05.webp)

**提问：** 《谁是最可爱的人》的作者是谁？"最可爱的人"指谁？（2024恢复课文）

**答案：** 魏巍，当代作家。"最可爱的人"指中国人民志愿军。本文写于抗美援朝时期，通过三个典型事例：①松骨峰战斗（壮烈牺牲）；②马玉祥救朝鲜儿童（国际主义）；③防空洞里的对话（爱国奉献），表现了志愿军战士对祖国和人民的无比热爱。

**易错分析：** "最可爱的人"这个称呼就来自这篇文章。这是魏巍的代表作，2024版教材恢复入选。三个事例要记牢——松骨峰（牺牲精神）、救儿童（国际主义）、吃雪（奉献精神）。

**记忆钩子：** 魏巍写志愿军，松骨峰救儿童吃雪——最可爱的人。

### 卡6

**配图：** ![](第二单元-家国情怀/06.webp)

**提问：** 《谁是最可爱的人》中"他们的品质是那样的纯洁和高尚，他们的意志是那样的坚韧和刚强"用了什么修辞？

**答案：** 排比。连续用"他们的品质/意志/气质/胸怀"四个排比句，热情讴歌志愿军战士的崇高精神，节奏铿锵有力、感情强烈真挚，具有强大的感染力。

**易错分析：** 排比句在文章开头和结尾都有——首尾呼应。这是魏巍典型的抒情风格——叙述和议论相结合。

**记忆钩子：** 品质意志气质胸怀——排比颂英雄。

### 卡7

**配图：** ![](第二单元-家国情怀/07.webp)

**提问：** 《土地的誓言》的作者是谁？文中"土地"有什么象征意义？

**答案：** 端木蕻良，现代作家。"土地"象征祖国/故乡。本文写于九一八事变十周年，流亡在关内的东北人以火热的语言表达了对沦陷十年的故乡土地的深深眷恋和誓死收复失地的决心。

**易错分析：** "端木蕻良"是笔名，原名曹汉文。"蕻"读 hóng。他不是"端木"复姓，这是笔名。"土地的誓言"是"土地发出的誓言"还是"面对土地发出的誓言"？后者——是作者面对故土发出的誓言。

**记忆钩子：** 端木蕻良思东北，土地誓言语炽热。

### 卡8

**配图：** ![](第二单元-家国情怀/08.webp)

**提问：** 《木兰诗》的体裁是什么？选自哪里？讲述了什么故事？

**答案：** 体裁是乐府民歌（北朝民歌）。选自北宋郭茂倩编的《乐府诗集》。讲述了木兰女扮男装、代父从军、征战沙场、辞官还乡的故事，塑造了一位勤劳、孝顺、勇敢、不慕荣华的巾帼英雄形象。

**易错分析：** 《木兰诗》是"北朝民歌"不是"唐代诗歌"。"乐府"是汉代设立的音乐官署，后来把采集来的民歌也叫"乐府"。"帖花黄"的"帖"通"贴"。"著我旧时裳"的"著"读 zhuó（穿）。

**记忆钩子：** 北朝民歌木兰诗，女扮男装代父征。

### 卡9

**配图：** ![](第二单元-家国情怀/09.webp)

**提问：** 《木兰诗》中"东市买骏马，西市买鞍鞯，南市买辔头，北市买长鞭"用了什么修辞？

**答案：** 排比、互文。分别到东西南北市买齐了战马和装备，运用排比渲染了出征前紧张忙碌的气氛。实际上不是东市只买骏马、西市只买鞍鞯——这是"互文"修辞，意思是到各处市场把出征所需物品买齐。

**易错分析：** "互文"是古诗常见修辞——上下句互相补充、合在一起表达完整意思。这里的意思是"跑遍市场买齐装备"，不是特定东南西北各买一样。

**记忆钩子：** 东市西市南北市，互文排比忙备战。

### 卡10

**配图：** ![](第二单元-家国情怀/10.webp)

**提问：** "万里赴戎机，关山度若飞。朔气传金柝，寒光照铁衣"（《木兰诗》）赏析。

**答案：** 这四句写木兰奔赴战场和征战生活。"万里赴戎机"写行军之远，"关山度若飞"写速度之快——像飞一样翻越关隘和山峰。"朔气传金柝"写北方寒气中传来打更声，"寒光照铁衣"写月光照在铠甲上。用简洁概括的语言写出了征战的艰苦和木兰的英勇无畏。

**易错分析：** 这段在诗中属"略写"——征战十年只用六句概括，而"还乡"则详写。详略安排突出了木兰不慕荣华、思念家乡的主题。金柝（tuò）是古代军中守夜打更用的器具。

**记忆钩子：** 万里关山飞度，十年征战六句。

### 卡11

**配图：** ![](第二单元-家国情怀/11.webp)

**提问：** "雄兔脚扑朔，雌兔眼迷离；双兔傍地走，安能辨我是雄雌？"（《木兰诗》）是什么意思？

**答案：** 比喻，以双兔并跑难辨雄雌，比喻木兰女扮男装、代父从军多年却未被识破的奇迹。这是全诗的结尾，用生动活泼的比喻表现木兰的机智和自豪，也带有民间歌谣俏皮幽默的风格。

**易错分析：** "扑朔"是爬搔，"迷离"是眯着眼——但不考本意，考比喻义：男女在战场上难以分辨。"安能"的反问语气很重要——"怎么能分辨呢？"

**记忆钩子：** 双兔傍地难分雌雄，木兰从军不输儿郎。

### 卡12

**配图：** ![](第二单元-家国情怀/12.webp)

**提问：** 第二单元写作主题是什么？如何学习抒情？

**答案：** 写作主题是"学习抒情"。抒情方式有两种：①直接抒情（直抒胸臆）——直接表达喜怒哀乐，如"我热爱这片土地"；②间接抒情——借景抒情、托物言志、记事抒情，如通过描写景物或叙述事件来抒发感情。好文章往往两者结合。

**易错分析：** 直接抒情≠空洞口号，间接抒情≠没有感情。本单元《黄河颂》是直接抒情，《老山界》是间接抒情（在叙事中自然流露），要对比理解两种方式。

**记忆钩子：** 直接抒情直抒胸臆，间接抒情借景托物。
