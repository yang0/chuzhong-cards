# 第一单元 人物精神

## 基本信息

- 科目：语文
- 年级：七年级下册
- 教材：统编版 / 人教版《语文》（2024 修订版）
- 用途：Anki 背诵卡片
- 格式：提问 / 答案 / 易错分析 / 记忆钩子

## 单元纲要

本单元以"人物精神"为主题，学习通过典型事件和细节描写展现杰出人物的品格。学习重点是把握人物特征、理解人物精神。

- 1 邓稼先 / 杨振宁
- 2 说和做——记闻一多先生言行片段 / 臧克家
- 3\* 列夫·托尔斯泰 / 茨威格（2024新增，替换《回忆鲁迅先生》）
- 4 孙权劝学 / 《资治通鉴》

## Anki 记忆卡

### 卡1

**配图：** ![](第一单元-人物精神/01.webp)

**提问：** 《邓稼先》的作者是谁？邓稼先是什么人？

**答案：** 杨振宁，美籍华裔物理学家，诺贝尔物理学奖获得者。邓稼先是核物理学家、中国核武器研制工作的开拓者和奠基者之一，被誉为"两弹元勋"。

**易错分析：** 杨振宁是"美籍华裔"不是美籍华人——学术上称华裔。邓稼先是"两弹元勋"（原子弹、氢弹），不是"两弹一星元勋"（但通常也放一起说）。注意"稼"读 jià。

**记忆钩子：** 杨振宁写邓稼先，两弹元勋为国献身。

### 卡2

**配图：** ![](第一单元-人物精神/02.webp)

**提问：** 《邓稼先》一文写了邓稼先哪些事迹？表现了他怎样的精神？

**答案：** ①美国获得博士学位后立即回国（爱国奉献）；②领导研制原子弹和氢弹（无私奉献、艰苦奋斗）；③在戈壁滩上隐姓埋名28年（甘于奉献）；④自己冒着危险去处理核弹事故（舍生忘死、责任感强）。表现了他爱国奉献、艰苦奋斗、舍生忘死的崇高精神。

**易错分析：** 邓稼先"隐姓埋名28年"是常考点。他领导的不是"制造"而是"理论设计"。"两弹"指原子弹和氢弹，不是导弹。

**记忆钩子：** 回国隐名28年，戈壁造弹舍生死。

### 卡3

**配图：** ![](第一单元-人物精神/03.webp)

**提问：** 《邓稼先》中"邓稼先是中国几千年传统文化所孕育出来的有最高奉献精神的儿子"怎么理解？

**答案：** 这句话高度评价了邓稼先的奉献精神——中国传统文化强调家国情怀、忠孝节义、舍小家为大家，邓稼先身上集中体现了这些传统美德。同时他又是现代科学精神的践行者，是传统美德与现代科学的完美结合。

**易错分析：** 这句评价在文章末尾（第6节"永恒的骄傲"），是全文的点睛之笔。分析时要写出"传统+现代"两层。

**记忆钩子：** 传统文化育稼先，奉献精神古今合。

### 卡4

**配图：** ![](第一单元-人物精神/04.webp)

**提问：** 《说和做——记闻一多先生言行片段》的作者是谁？闻一多是谁？

**答案：** 臧克家，现代诗人，代表作《有的人》。闻一多是诗人、学者、民主战士。前期是"书斋里的学者"（研究古典文学），后期是"革命者"（拍案而起、投身民主运动）。

**易错分析：** 臧克家是诗人，代表作《有的人》（纪念鲁迅）。"臧"读 zāng，不读 cáng。闻一多前期"做了再说"（研究学问），后期"说了就做"（革命行动），前后变化是文章的关键线索。

**记忆钩子：** 臧克家写闻一多，学者变斗士言行合一。

### 卡5

**配图：** ![](第一单元-人物精神/05.webp)

**提问：** 臧克家《说和做》中闻一多作为学者和革命者分别有什么特点？

**答案：** 作为学者：做了再说、做了不说——潜心研究古典文学（《唐诗杂论》《楚辞校补》《古典新义》），十年艰辛。作为革命者：说了就做、言行一致——为争取民主拍案而起，发表演说、起草传单，最后献出生命。

**易错分析：** 前后对比不能混淆：学者是"做而不说"，革命者是"说而后做"。闻一多了不起的是两者集于一身。

**记忆钩子：** 学者做不说，革命者说了就做。

### 卡6

**配图：** ![](第一单元-人物精神/06.webp)

**提问：** 《说和做》中"仰之弥高，越高，攀得越起劲"表达了什么？

**答案：** 化用《论语》"仰之弥高，钻之弥坚"，形容闻一多钻研学问时对学术高峰的敬畏和追求。越高越要攀登，越难越要钻研——表现了他刻苦勤奋、勇攀学术高峰的精神。

**易错分析：** 这句是写他作为"学者"的阶段，不是革命者。原文化用《论语·子罕》——"仰之弥高，钻之弥坚"。

**记忆钩子：** 越难越钻，学者本色仰弥高。

### 卡7

**配图：** ![](第一单元-人物精神/07.webp)

**提问：** 2024版新课文《列夫·托尔斯泰》的作者是谁？主人公是谁？

**答案：** 茨威格，奥地利作家（1881-1942），代表作《人类群星闪耀时》《一个陌生女人的来信》。主人公是列夫·托尔斯泰，俄国作家，代表作《战争与和平》《安娜·卡列尼娜》《复活》。本文是一篇人物传记。

**易错分析：** 茨威格是"奥地利"人，不是"德国"或"瑞士"。列夫·托尔斯泰是"俄国"作家，不是苏联。高尔基说他"人类的一切都与我有关"——指托尔斯泰的大爱精神。

**记忆钩子：** 茨威格奥地利写托尔斯泰，俄国文豪列夫·托尔斯泰。

### 卡8

**配图：** ![](第一单元-人物精神/08.webp)

**提问：** 《列夫·托尔斯泰》中托尔斯泰的外貌有什么特点？为什么要这样写？

**答案：** 外貌特点：①平庸甚至丑陋——须发浓密、长相普通、相貌平平；②眼睛犀利——"一道黑豹似的目光""具有穿透力的目光"。先写外貌的平庸（抑），再写目光的犀利（扬），用"欲扬先抑"的手法突出托尔斯泰深邃的思想和伟大的精神世界。

**易错分析：** 本文最突出的写法是"欲扬先抑"——先极写其相貌平平（甚至丑陋），再写眼睛的惊人力量，"托尔斯泰的眼睛"能够"看到世界的本质"。

**记忆钩子：** 欲扬先抑写外貌，平庸外表藏犀利目光。

### 卡9

**配图：** ![](第一单元-人物精神/09.webp)

**提问：** "这道目光就像一把锃亮的钢刀刺了过来，又稳又准，击中要害"（《列夫·托尔斯泰》）用了什么修辞？

**答案：** 比喻。把托尔斯泰的目光比作锃亮的钢刀，生动形象地写出了他目光的犀利——看人看事一眼看透本质，具有穿透力和洞察力。同时也表现了他的严厉和直指人心的力量。

**易错分析：** "锃亮"的"锃"读 zèng，不读 chéng。这是茨威格刻画人物最精彩的一笔——用"钢刀"比喻目光的锋利。

**记忆钩子：** 目光如钢刀，一眼看透本质。

### 卡10

**配图：** ![](第一单元-人物精神/10.webp)

**提问：** 《孙权劝学》出自哪部书？体裁是什么？

**答案：** 出自《资治通鉴》，北宋司马光主持编纂。体裁是编年体通史。记载了从战国到五代的历史。本文写的是孙权劝吕蒙学习，吕蒙学有所成后令人刮目相看的故事。

**易错分析：** 《资治通鉴》是司马光"主持编纂"的，不是他一个人写的。"通鉴"的"鉴"是"镜子/借鉴"的意思——以史为鉴。"编年体"是按时间顺序编写的史书体例。

**记忆钩子：** 司马光资治通鉴，孙权劝学吕蒙成材。

### 卡11

**配图：** ![](第一单元-人物精神/11.webp)

**提问：** 《孙权劝学》中"卿今当涂掌事，不可不学"、"孤岂欲卿治经为博士邪"、"士别三日，即更刮目相待"三句话分别是谁说的？什么意思？

**答案：** 
①"卿今当涂掌事，不可不学"——孙权说：你现在当权管事了，不可不学习。
②"孤岂欲卿治经为博士邪"——孙权说：我难道是想让你研究经书做博士吗？（意为：只是让你广泛学习了解历史罢了）
③"士别三日，即更刮目相待"——吕蒙说（对鲁肃）：读书人分别几天，就要重新擦擦眼睛另眼相看。

**易错分析：** "博士"是当时掌管经学传授的官职，不是现代意义上的博士。"当涂"是"当权"的意思，不是"当路"。

**记忆钩子：** 孙权劝学三重意，当涂掌事治经博士刮目相待。

### 卡12

**配图：** ![](第一单元-人物精神/12.webp)

**提问：** 《孙权劝学》重点字词：辞、以、务、见、往事、但、涉猎、更、遂。

**答案：**
- 辞：推托（蒙辞以军中多务）。
- 以：用/拿（蒙辞以军中多务）。
- 务：事务（军中多务）。
- 见：了解/知道（见往事耳）。
- 往事：历史（见往事耳）。
- 但：只（但当涉猎）。
- 涉猎：粗略地阅读。
- 更（gèng）：重新（刮目相待）。
- 遂：于是/就（遂拜蒙母）。

**易错分析：** "但"在文中是"只"（古义），不是"但是"（今义）。"见往事耳"——"耳"是"罢了"的语气词。"孰若孤"——"孰"是"谁"，"若"是"比得上"。

**记忆钩子：** 但作只字看，往事是历史，涉猎非狩猎。

### 卡13

**配图：** ![](第一单元-人物精神/13.webp)

**提问：** 第一单元写作主题是什么？如何写出人物特点？

**答案：** 写作主题是"写出人物特点"。要点：①通过典型事例来表现人物（选最能体现人物特点的事）；②运用细节描写（动作、语言、神态、心理、外貌）；③可以运用正面描写和侧面烘托相结合；④适当运用对比、欲扬先抑等手法。

**易错分析：** 写人物≠流水账记生平。要选"典型事件"并用"细节描写"，这是两个关键。

**记忆钩子：** 典型事例加细节，正面侧面结合写。
