# 第3单元 古诗文

- 科目：语文
- 年级：八年级下册
- 教材：统编版2026春
- 卡片数：4



## 1. CN8B-03-001

- 章节：《桃花源记》
- 提问：桃花源寄托了作者怎样的社会理想？
- 答案：寄托对和平安宁、平等自足、没有剥削战乱生活的向往，也反映对现实黑暗社会的不满。
- 易错：桃花源是文学虚构的理想社会，不是历史上真实地点。
- 记忆钩子：以理想桃源反衬黑暗现实。
- 考点：主旨与虚实题
- 提示词：prompts/CN8B-03-001.md



## 2. CN8B-03-002

- 章节：《小石潭记》
- 提问：《小石潭记》中作者情感怎样变化？
- 答案：发现小潭时感到惊喜愉悦，观鱼时暂得欢欣，久坐后因环境幽深冷寂而触发被贬处境，转为凄清孤寂。
- 易错：景物描写与情感不是割裂的，环境冷清推动情绪转折。
- 记忆钩子：发现之乐—观鱼之乐—幽冷之悲。
- 考点：情景交融题
- 提示词：prompts/CN8B-03-002.md



## 3. CN8B-03-003

- 章节：《核舟记》
- 提问：《核舟记》怎样表现雕刻者技艺精湛？
- 答案：按空间顺序细致介绍核舟各部分，以极小材料上丰富人物、器物、文字和神态的精细程度突出因势象形、构思巧妙。
- 易错：说明重点不是核舟体积本身，而是小中见大、细节逼真。
- 记忆钩子：小小核舟，细节万千。
- 考点：空间顺序与细节题
- 提示词：prompts/CN8B-03-003.md



## 4. CN8B-03-004

- 章节：《诗经》二首
- 提问：《关雎》和《蒹葭》在表现爱情时有何共同特点？
- 答案：都运用重章叠句和兴的手法营造节奏与意境，表达真挚执着的追求；《蒹葭》更突出朦胧、可望难即。
- 易错：“兴”是先言他物以引起所咏之词，不等同于单纯比喻。
- 记忆钩子：重章复沓，借景起情。
- 考点：手法比较题
- 提示词：prompts/CN8B-03-004.md
