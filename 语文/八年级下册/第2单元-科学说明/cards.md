# 第2单元 科学说明

- 科目：语文
- 年级：八年级下册
- 教材：统编版2026春
- 卡片数：4



## 1. CN8B-02-001

- 章节：《大自然的语言》
- 提问：物候现象受哪些因素影响？
- 答案：主要受纬度、经度、高下差异和古今差异影响，分析时要结合温度、水分、地形与时间变化。
- 易错：因素排列有主次，纬度通常是最主要因素。
- 记忆钩子：纬度、经度、高下、古今。
- 考点：信息筛选与顺序题
- 提示词：prompts/CN8B-02-001.md



## 2. CN8B-02-002

- 章节：《阿西莫夫短文两篇》
- 提问：文章怎样用不同领域的证据说明科学问题？
- 答案：作者把恐龙化石分布与大陆漂移、撞击说与恐龙灭绝联系起来，通过证据链和跨学科推理说明不同科学领域相互关联。
- 易错：科普说明不是只给结论，还要追踪证据怎样支持结论。
- 记忆钩子：证据跨领域，推理成链条。
- 考点：逻辑关系与证据题
- 提示词：prompts/CN8B-02-002.md



## 3. CN8B-02-003

- 章节：《月亮从哪里来》
- 提问：科普文章怎样兼顾准确性与生动性？
- 答案：用准确术语、数据和证据保证科学性，同时借助设问、比喻、故事化叙述和清晰图景降低理解难度。
- 易错：语言生动不能牺牲科学事实，修辞必须服务于说明。
- 记忆钩子：术语保准确，表达助理解。
- 考点：语言特点题
- 提示词：prompts/CN8B-02-003.md



## 4. CN8B-02-004

- 章节：《时间的脚印》
- 提问：说明文常见说明顺序有哪些，怎样判断？
- 答案：有时间顺序、空间顺序和逻辑顺序；要看段落内容按过程先后、空间位置还是因果与主次关系组织。
- 易错：文章可能综合使用多种顺序，应按具体段落判断。
- 记忆钩子：看先后、看位置、看逻辑。
- 考点：说明顺序高频题
- 提示词：prompts/CN8B-02-004.md
